# [Luogu-P1972](https://www.luogu.com.cn/problem/P1972) HH的项链

## 1. 核心逻辑

- **问题本质**: 区间查询不同数字的数量。
    
- **破局转换**:
    
    1. **离线处理**：将询问按右端点 $R$ 从小到大排序。
        
    2. **扫描线思想**：当扫描指针移动到 $i$ 时，我们只希望 $a[i]$ 这一种颜色的**最右侧**那个位置产生贡献。
        
    3. **更新策略**：若 $a[i]$ 之前在位置 $lst$ 出现过，则在 BIT 中 `add(lst, -1)`，然后 `add(i, 1)`。
        
    4. **查询**：对于 $R=i$ 的所有询问，结果即为 `query(R) - query(L-1)`。
        

## 2. 代码实现

{% fold info @AcCode %}

```cpp
// P1972 HH的项链
// Key Logic: Offline processing with rightmost contribution maintenance

#include <bits/stdc++.h>
using namespace std;

const int maxn = 1e6 + 50;
int n, m;
int a[ maxn ], c[ maxn ], ans[ maxn ], lst[ maxn ];

struct Node
{
    int l, r, id;
    bool operator < ( const Node &other ) const
    {
        return r < other.r;
    }
} qs[ maxn ];

int lowbit( int x )
{
    return x & -x;
}

void add( int x, int v )
{
    for( int i = x; i <= n; i += lowbit( i ) )
    {
        c[ i ] += v;
    }
}

int query( int x )
{
    int res = 0;
    for( int i = x; i; i -= lowbit( i ) )
    {
        res += c[ i ];
    }
    return res;
}

int main( )
{
    ios::sync_with_stdio( 0 );
    cin.tie( 0 );

    cin >> n;
    for( int i = 1; i <= n; ++ i ) cin >> a[ i ];

    cin >> m;
    for( int i = 1; i <= m; ++ i )
    {
        cin >> qs[ i ].l >> qs[ i ].r;
        qs[ i ].id = i;
    }

    sort( qs + 1, qs + m + 1 );

    int cur = 1;
    for( int i = 1; i <= n; ++ i )
    {
        if( lst[ a[ i ] ] )
        {
            add( lst[ a[ i ] ], -1 );
        }
        add( i, 1 );
        lst[ a[ i ] ] = i;

        while( cur <= m && qs[ cur ].r == i )
        {
            ans[ qs[ cur ].id ] = query( qs[ cur ].r ) - query( qs[ cur ].l - 1 );
            ++ cur;
        }
    }

    for( int i = 1; i <= m; ++ i ) cout << ans[ i ] << "\n";

    return 0;
}
```

## 3. 归档备注

- 扫描线+离线的经典组合。
    
- 核心支点：区间查询不同数，只需保留最右侧的副本，因为它覆盖的区间范围最大。
