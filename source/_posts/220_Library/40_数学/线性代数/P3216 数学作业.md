# [P3216 [HNOI2011] 数学作业 - 洛谷](https://www.luogu.com.cn/problem/P3216)

## 1. 题面梗概

**中译中**：将 $1 \dots n$ 顺序拼接数，求该数对 $m$ 取模的结果。

> $1 \le n \le 10^{18}, 1 \le m \le 10^9$

## 2. 逻辑推导

我们可以简单的用 DP 思路写出转移方程。

设 $f_{i}$ 表示前 $i$ 个数拼接的结果，当前加入的数 $i$ 有 $d$ 位（ $d = \lfloor \lg i \rfloor + 1$ ），那么：

$$
f_{i} = f_{i-1} \cdot 10^{d} + i \pmod m
$$

但是问题在于 $n \leq 10^{18}$，这种线性的递推是 $O(n)$ 的，直接 TLE 一波送走。

遇到这种规模的数据，最容易能想到的复杂度是 $O( \log N )$，或许我们可以用矩阵快速幂。

### 2.1 答案矩阵

构造出答案矩阵：

$$
\begin{bmatrix}
dp[i] \\
i \\
1 \\
\end{bmatrix}
$$

#### 答案矩阵是怎么构造的呢？

答案矩阵一般都是单列的：

1. 先把 $dp[i]$ 写上，方便在最后直接输出。

$$
\begin{bmatrix}
dp[i] \\
 \\
 \\
\end{bmatrix}
$$

2. 回顾我们的递推方程，$dp[i]$ 需要通过 $dp[i-1]$ 和 $i$ 推出，这两项都需要体现在矩阵里，为了转移，我们还需要 $i$。

$$
\begin{bmatrix}
dp[i] \\
i \\
 \\
\end{bmatrix}
$$

3. 由 $i-1$ 是如何推出 $i$ 的呢？所以矩阵里还需要一个常数 $1$。

### 2.2 转移矩阵

构造出转移矩阵：

$$
\begin{bmatrix}
10^d & 1 & 0 \\
0 & 1 & 1 \\
0 & 0 & 1 \\
\end{bmatrix}
$$

#### 转移矩阵是怎么构造的呢？

其实可以理解为待定系数法：

1. $dp[i]$ 是如何推得的？$dp[i] = dp[i-1] \times 10^{d} + i$。

$$
\begin{bmatrix}
10^d & 1 & 0 \\
 \\
 \\
\end{bmatrix}
$$

2. 同理，要推出 $i + 1$，则需要在 $i$ 和 $1$ 的位置上点上 $1$。

$$
\begin{bmatrix}
10^d & 1 & 0 \\
0 & 1 & 1 \\
 \\
\end{bmatrix}
$$

3. 1 就不用多说了吧

$$
\begin{bmatrix}
10^d & 1 & 0 \\
0 & 1 & 1 \\
0 & 0 & 1 \\
\end{bmatrix}
$$

### 2.3 问题

我们发现 $d$ 会随着 $i$ 改变，这在矩阵快速幂中是不被允许的，因此不能一次矩阵快速幂解决，我们需要按照 $i$ 的位数，分别对每一段进行转移。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxsz = 5;
const int maxn = 100 + 5;

ll dp[maxn];
ll n, mod;

struct Matrix {
    ll m[maxsz][maxsz];
    int sz;
    Matrix( int s = 0 ) : sz( s ) { memset( m, 0, sizeof( m ) ); }
    void init( ) { for( int i = 1; i <= sz; ++ i ) m[i][i] = 1; }

    friend Matrix operator * ( const Matrix &a, const Matrix &b ) {
        Matrix res( a.sz );
        for( int i = 1; i <= res.sz; ++ i ) {
            for( int k = 1; k <= res.sz; ++ k ) {
                if( a.m[i][k] == 0 ) continue;
                for( int j = 1; j <= res.sz; ++ j )
                    res.m[i][j] = ( res.m[i][j] + a.m[i][k] * b.m[k][j] ) % mod;
            }
        }
        return res;
    }
};

Matrix mqpow( Matrix a, ll k ) 
{
    Matrix res( a.sz ); res.init( );
    for( ; k; k >>= 1, a = a * a )
        if( k & 1 ) res = res * a;
    return res;
}

ll qpow( ll a, ll k ) {
    ll res = 1;
    for( ; k; k >>= 1, a = a * a % mod )
        if( k & 1 ) res = res * a % mod;
    return res;
}

int main( )
{
    ios::sync_with_stdio( 0 ); cin.tie( 0 );
    
    cin >> n >> mod;

    ll ans[4] = { 0, 0, 0, 1 };
    
    for( ll d = 1, st = 1; st <= n; d ++, st *= 10 )
    {
        ll ed;
        if( st > n / 10 ) ed = n;
        else ed = st * 10 - 1;

        ll cnt = ed - st + 1;

        Matrix T( 3 );
        T.m[1][1] = qpow( 10, d );
        T.m[1][2] = 1; T.m[1][3] = 1;
        T.m[2][1] = 0; T.m[2][2] = 1; T.m[2][3] = 1;
        T.m[3][1] = 0; T.m[3][2] = 0; T.m[3][3] = 1;
        T = mqpow( T, cnt );
        
        ll nxt[4] = { 0, 0, 0, 0 };
        for( int i = 1; i <= 3; ++ i )
            for( int j = 1; j <= 3; ++ j )
                nxt[i] = ( nxt[i] + ans[j] * T.m[i][j] ) % mod;
        
        for( int i = 1; i <= 3; ++ i ) ans[i] = nxt[i];
        if( st > n / 10 ) break;
    }

    cout << ans[1] % mod << '\n';
    return 0;
}
```

## 4. 复盘

- **复杂度分析**：$O( \lg N \cdot 3^3 \cdot \log N )$。
    
- **矩阵乘法顺序**：本代码实现的是 $nxt[i] = \sum old[j] \cdot T[i][j]$，逻辑上等价于列向量左乘转移矩阵。
