# [P6583 回首过去 - 洛谷](https://www.luogu.com.cn/problem/P6583)

## 1. 题面梗概

**中译中**：在 $1 \le x, y \le n$ 范围内，求有多少对 $(x, y)$ 使得 $\frac{x}{y}$ 是有限小数。

> $n \le 10^{12}$

## 2. 逻辑推导

什么是有限小数？对于最简分数 $\frac{x}{y}$ 来说，当且仅当 $y = 2^a 5^b$ 的形式。

记 $v \in S = \{ 2^a \cdot 5^b \mid a, b \geq 0 \}$，那么答案就是：

$$
Ans = \sum_{i=1}^n \sum_{j=1}^n \left[ \frac{i}{\gcd(i, j)} = v \right]
$$

发现 $i$ 一定是 $v$ 的倍数，那么可以接着化简：

$$
Ans = \sum_{k=1}^{n / v} \sum_{j = 1}^{n} [gcd(vk, j) = v]
$$

然后把后面一个和式中的 $k$ 除掉，就有：

$$
Ans = \sum_{k=1}^{n / v} \sum_{j=1}^{n / k} [ gcd(v, j) = 1]
$$

由于只有 $2, 5$ 两个因子，不妨用容斥原理展开：

$$
Ans = \sum_{k=1}^{n / v} \left[ \lfloor n / k \rfloor  - [a>0]\frac{ \lfloor n / k \rfloor  }{2} - [b>0] \frac{ \lfloor n / k \rfloor  }{5} + [ab>0] \frac{ \lfloor n / k \rfloor  }{10} \right]
$$

由此可以得到 $80 pts$ 的做法，注意到

$$
\sum_{k=1}^{n / v} \left(  \left\lfloor  \frac{n}{k}  \right\rfloor  \right)
$$

这样的式子类似于 整除分块，于是我们可以把每一块都提前预处理，做个前缀和，然后对每一组 $a, b$ 以 $O( \log N )$ 的复杂度计算。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

vector< ll > s2, s5, s10;

ll calc( ll M )
{
    ll res = 0;
    if( M >= 1 ) res += M;
    ll c2 = upper_bound( s2.begin( ), s2.end( ), M ) - s2.begin( );
    res += c2 * ( M - M / 2 );
    ll c5 = upper_bound( s5.begin( ), s5.end( ), M ) - s5.begin( );
    res += c5 * ( M - M / 5 );
    ll c10 = upper_bound( s10.begin( ), s10.end( ), M ) - s10.begin( );
    res += c10 * ( M - M / 2 - M / 5 + M / 10 );
    
    return res;
}

int main( )
{
    ios::sync_with_stdio( 0 ); cin.tie( 0 );
    ll n; 
    cin >> n;
    
    for( ll p2 = 1, a = 0; p2 <= n; p2 *= 2, a ++ )
    {
        for( ll p5 = 1, b = 0; p2 * p5 <= n; p5 *= 5, b ++ )
        {
            ll v = p2 * p5;
            if( a > 0 && b == 0 ) s2.push_back( v );
            else if( a == 0 && b > 0 ) s5.push_back( v );
            else if( a > 0 && b > 0 ) s10.push_back( v );
        }
    }
    sort( s2.begin( ), s2.end( ) );
    sort( s5.begin( ), s5.end( ) );
    sort( s10.begin( ), s10.end( ) );

    ll ans = 0;

    for( ll l = 1, r; l <= n; l = r + 1 )
    {
        r = n / ( n / l );
        ans += ( r - l + 1 ) * calc( n / l );
    }

    cout << ans << endl;
    return 0;
}
```
