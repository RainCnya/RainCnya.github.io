# [P2421 [NOI2002] 荒岛野人 - 洛谷](https://www.luogu.com.cn/problem/P2421)

## 1. 题面梗概

**中译中**：有 $n$ 个野人在环形排列的 $M$ 个山洞中走。给定每个人的起点 $C_i$、速度 $P_i$ 和寿命 $L_i$。求最小的 $M$，使得任何两个野人在各自的寿命结束前，都不会在同一个洞穴相遇。

> $1 \le n \le 15, 1 \le C_i, P_i, L_i \le 100, 1 \le M \le 10^6$

## 2. 逻辑推导

这看起来是道数学问题，我们试试拆解一下这个问题。

因为 $n \leq 15$，所以我们可以大胆采用 $O( N^2 )$ 的暴力。

假设野人 $i$ 和 $j$ 相遇，意味着在第 $t$ 年，他们的位置模 $M$ 同余：

$$
C_{i} + P_{i} \cdot t \equiv C_{j} + P_{j} \cdot j \pmod M
$$

直接枚举 $M$ 也能接受 $O( M N^2 )$。

### 2.1 同余方程

那么我们接着整理上述同余式，可以得到关于 $t$ 的线性同余方程：

$$
(P_{i} - P_{j}) \cdot t \equiv C_{j} - C_{i} \pmod M
$$

令 $A = P_{i} - P_{j}$，$B = M$，$C = C_{j} - C_{i}$，对应不定方程：

$$
At \equiv C \pmod{B} \iff At + Bk = C
$$

然后 $exgcd$ 求解 $gcd(A, B)$ 即可，若 $gcd(A, B) \nmid C$ 则这对野人不会相遇。

若有解且 $t_{0} \leq \min( L_{i}, L_{j} )$，则说明二人会相遇。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxm = 1e6 + 5;
const int maxn = 15 + 5;

int c[maxn], p[maxn], l[maxn];
int n;

ll exgcd( ll a, ll b, ll &x, ll &y )
{
    if( b == 0 ) { x = 1, y = 0; return a; }
    ll d = exgcd( b, a % b, y, x );
    y -= a / b * x;
    return d;
}

int main( )
{
    ios::sync_with_stdio(0); cin.tie(0);
    
    cin >> n;
    int ma = -1;
    for( int i = 1; i <= n; ++ i )
    {
        cin >> c[i] >> p[i] >> l[i];
        ma = max( ma, c[i] );
    }

    for( int M = ma; M <= maxm; ++ M )
    {
        bool ok = 1;
        for( int i = 1; i <= n && ok; ++ i )
        {
            for( int j = i + 1; j <= n && ok; ++ j )
            {
                ll a = p[i] - p[j], b = M, cc = c[j] - c[i], x, y;
                ll d = exgcd( a, b, x, y );
                if( cc % d != 0 ) continue;
                b = abs( b / d ), cc = cc / d;
                x = ( x * cc % b + b ) % b;
                if( x <= l[i] && x <= l[j] ) ok = 0;
            }
        }
        if( ok )
        {
            cout << M << '\n';
            break;
        }
    }
    return 0;
}
```

## 4. 复盘

- **复杂度分析**：$O( M \cdot N^2 )$。
    
- **碎碎念**：把同余式转化为不定方程，再用 $Exgcd$ 求解即可。
