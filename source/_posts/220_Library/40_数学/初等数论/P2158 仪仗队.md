# [P2158 [SDOI2008] 仪仗队 - 洛谷](https://www.luogu.com.cn/problem/P2158)

## 1. 题面梗概

**中译中**：在一个 $N\times N$ 的点阵中，你站在 $(1, 1)$ 点，求能看到多少个不被遮挡的点。

> $1 \le N \le 40000$

## 2. 逻辑推导

什么情况下，点 $B(x_{2}, y_{2})$ 会挡住 $A(x_{1}, y_{1})$ ？很显然，当 $O, A, B$ 三点共线时，也就是斜率相同：$\frac{y_{2}}{x_{2}} = \frac{y_{1}}{x_{1}}$。

因此，一个点 $(x, y)$ 能被看到，当且仅当其斜率 $k = \frac{y}{x}$ 是最简分数，也就是说 $gcd( x, y) = 1$。

### 2.1 欧拉函数

由于点阵是 $N \times N$ 的，我们需要统计 $1 \leq x, y \leq N$ 上 $gcd(x, y) = 1$ 的对数。

除了 $(1, 1), (1, 2), (2, 1)$ 三个特殊点，其余点对称分布在 $y = x$ 的两侧。

对于 $y < x$ 的部分，给定一个 $x$，满足 $gcd(x, y) = 1$ 的 $y$ 的个数，恰好就是 **欧拉函数** $\phi(x)$ 的定义。

$$
Ans = 3 + 2 \cdot \sum_{i = 2}^{N - 1} \phi(i)
$$

当 $N = 1$ 时，点阵中只有一个点，答案为 $0$。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxn = 4e4 + 5;
int phi[maxn], p[maxn], cnt;
int n;

// 线性筛 - 欧拉函数预处理
void init_phi( ll n )
{
    phi[1] = 1;
    for( ll i = 2; i <= n; ++ i )
    {
        if( phi[i] == 0 ) p[++ cnt] = i, phi[i] = i - 1;
        for( ll j = 1; j <= cnt && i * p[j] <= n; ++ j )
        {
            if( i % p[j] == 0 )
            {
                phi[i * p[j]] = phi[i] * p[j];
                break;
            }
            else phi[i * p[j]] = phi[i] * ( p[j] - 1 );
        }
    }
}


int main( )
{
    ios::sync_with_stdio( 0 ); cin.tie( 0 );
    cin >> n;

    if( n == 1 ) { cout << 0 << '\n'; return 0; }

    init_phi( n );

    ll ans = 1;
    for( int i = 2; i <= n - 1; ++ i ) ans += phi[i];
    cout << ans * 2 + 1 << '\n';

    return 0;
}
```
