# [P2568 GCD - 洛谷](https://www.luogu.com.cn/problem/P2568)

## 1. 题面梗概

**中译中**：在 $1 \le x, y \le n$ 范围内，求满足 $\gcd(x, y)$ 为质数的数对 $(x, y)$ 总数。

> $1 \le n \le 10^7$

## 2. 逻辑推导

直接暴力枚举太蠢了，而且还要判断质数。

不妨倒过来思考，枚举每个可能的质数 $p \leq n$，统计满足 $gcd(x, y) = p$ 的对数，根据 $gcd$ 的性质：

$$
gcd(x, y) = p \iff gcd( \frac{x}{p}, \frac{y}{p} ) = 1
$$

其中 $1 \leq \frac{x}{p}, \frac{y}{p} \leq \frac{n}{p}$。

### 2.1 互质

那么问题就转化为了：对于给定的 $m = \lfloor n / p \rfloor$，求 $1 \leq a, b \leq m$ 且 $gcd(a, b) = 1$ 的数对 $(a, b)$ 的数量。

1. 若 $a = b$，只有 $a = b = 1$ 一组。

2. 若 $a < b$，对于确定的 $b$，满足条件的 $a$ 的个数即为 $\phi(b)$。

3. 根据对称性，$a > b$ 的情况与 $a < b$ 相等。

因此，范围内互质对的总数为：

$$
f(m) = 2 \cdot \sum_{i=2}^{m}(\phi(i) + 1) = 2 \sum_{i=2}^{m} \phi(i) - 1
$$

最终对所有质数 $p$ 的贡献求和即可：

$$
Ans = \sum_{p \leq n} \left( 2 \cdot \sum_{i=1}^{ \lfloor n / p \rfloor } \phi(i) - 1 \right)
$$

然后用 线性筛 $O(n)$ 预处理欧拉函数即可。发现复杂度还是有点高，我们用前缀和优化这个求和。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxn = 1e7 + 5;

int phi[maxn], p[maxn], cnt;
ll sumphi[maxn];
int n;

// 线性筛 - 预处理 phi 及其前缀和
void init_phi( int n )
{
    phi[1] = 1;
    for( int i = 2; i <= n; ++ i )
    {
        if( phi[i] == 0 ) p[++ cnt] = i, phi[i] = i - 1;

        for( int j = 1; j <= cnt && i * p[j] <= n; ++ j )
        {
            if( i % p[j] == 0 )
            {
                phi[i * p[j]] = phi[i] * p[j];
                break;
            }
            phi[i * p[j]] = phi[i] * ( p[j] - 1 );
        }
    }
    for( int i = 1; i <= n; ++ i ) sumphi[i] = sumphi[i - 1] + phi[i];
}

int main( )
{
    ios::sync_with_stdio(0); cin.tie(0);
    
    cin >> n;
    init_phi( n );

    ll ans = 0;
    for( int i = 1; i <= cnt; ++ i )
    {
        ans += sumphi[n / p[i]] * 2 - 1;
    }
    cout << ans << '\n';
    return 0;
}
```
