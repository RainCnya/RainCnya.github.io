# [P2261 [CQOI2007] 余数求和 - 洛谷](https://www.luogu.com.cn/problem/P2261)

## 1. 题面梗概

**中译中**：给定 $n$ 和 $k$，求 $\sum_{i=1}^n (k \bmod i)$ 的值。

> $1 \le n, k \le 10^9$

## 2. 逻辑推导

## 2. 逻辑推导

直接对余数求和没有线性规律，虽然打表发现似乎有些规律，但不完善。

根据模运算的定义 $a \bmod b = a - b \cdot \lfloor \frac{a}{b} \rfloor$，我们可以将原式子改成：

$$
\sum_{i=1}^{n}( k - i \cdot \lfloor \frac{k}{i} \rfloor ) = nk - \sum_{i=1}^{n}(i \cdot \lfloor \frac{k}{i} \rfloor )
$$

然后瓶颈在于如何高效求解 后一项式子 $\sum i \cdot \lfloor k / i \rfloor$。

下面列个表格找规律，这里设 $k = 5$。

|           $i$           |  1  |  2  |  3  |  4  |  5  |  6  |  7  |  8  |
| :---------------------: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
| $\lfloor k / i \rfloor$ |  5  |  2  |  1  |  1  |  1  |  0  |  0  |  0  |

我们发现 $\lfloor k / i \rfloor$ 在某一部分里是不变的，因此可以拆解求和。

设当前左端点为 $l$，商为 $\lfloor k / i \rfloor$，该商能维持的最右端点为 $r = \lfloor k / val \rfloor$ （ 注意不能超过 $n$ ）。

如果 $val = 0$，则后面的商都是 $0$，直接令 $r = n$。

这样，这个累积求和就通过分块解决了，这也是经典的整除分块模型。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main( )
{
    ios::sync_with_stdio(0); cin.tie(0);
    ll n, k;
    cin >> n >> k;

    ll ans = n * k;    
    for( ll l = 1, r; l <= n; l = r + 1 )
    {
        ll val = k / l;
        if( val == 0 ) r = n;
        else r = min( k / val, n );
        ans -= val * ( l + r ) * ( r - l + 1 ) / 2;
    }
    cout << ans << '\n';
    return 0;
}
```
{% endfold %}
