# [P1725 琪露诺 - 洛谷](https://www.luogu.com.cn/problem/P1725)

## 1. 题面梗概

**中译中**: 从 $0$ 点出发，每次跳跃距离在 $[L, R]$ 间，求跳到对岸（位置 $>N$）能拿到的最大点数和。

## 2. 逻辑推导

这一看是个路径规划决策问题，那么直接 **线性DP** 起手。

### 2.1 DP

**定义**：$f[i]$ 表示到第 $i$ 个格子的最大收益。

**转移**：由于跳跃距离为一个区间 $[L,R]$，那么前驱节点 $j \in [i-R,i-L]$。

$$
f[i] = a[i] + \max_{i-R \leq j \leq i - L}(f[j])
$$
但是很显然没法暴力枚举，我们可以对这个求 $\max$ 进行优化。

### 2.2 单调队列

随着 $i$ 的右移，显然这个 $j$ 的区间也随之右移，那这就是个滑动窗口求最值的问题。

不过这题的价值在于，单调队列的下标处理。

## 3. 代码实现

{% fold info @AcCode %}

```cpp
#include <bits/stdc++.h>
using namespace std;

const int maxn = 2e5 + 50;
const int inf = 0x3f3f3f3f;
int a[maxn], f[maxn];
int n, l, r;

int main( ) 
{
    ios::sync_with_stdio( 0 );
    cin.tie( 0 );

    cin >> n >> l >> r;
    for( int i = 0; i <= n; ++ i ) cin >> a[i];

    memset( f, 128, sizeof( f ) ); // 初始化为负无穷
    f[0] = 0;

    deque< int > q;
    int ans = -inf;

    for( int i = l; i <= n; ++ i ) 
    {
        // 入队候选点: i-l 刚刚进入窗口
        int candidate = i - l;
        while( ! q.empty( ) && f[ q.back( ) ] <= f[ candidate ] ) q.pop_back( );
        q.push_back( candidate );

        // 剔除过期点: < i-r
        while( ! q.empty( ) && q.front( ) < i - r ) q.pop_front( );

        if( f[ q.front( ) ] != (int)0x80808080 ) 
        {
            f[i] = f[ q.front( ) ] + a[i];
        }

        if( i + r > n ) ans = max( ans, f[i] );
    }

    cout << ans << "\n";
    return 0;
}
```

{% endfold %}

## 4. 复盘

- **复杂度分析**：$O(N)$。
    
- **碎碎念**：这是一道异步入队的 **单调队列优化** DP 模型。
