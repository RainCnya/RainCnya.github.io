# [P2015 二叉苹果树 - 洛谷](https://www.luogu.com.cn/problem/P2015)

## 1. 题面梗概

**中译中**：给出一棵 $n$ 个点的二叉树，每条边（树枝）都有一个苹果数量（边权）。现在要保留 $q$ 条边，且保留的边必须与根节点 $1$ 连通。求能保留的最大苹果总数。

> $1 \le n \le 100, 1 \le q < n$

## 2. 逻辑推导

这是一个典型的 树形DP 问题，我们可以定义 $f[u][j]$ 为以 $u$ 为根节点的子树，内有 $j$ 条边的最大苹果数。

由于左右子树互不干扰，所以是单独的子问题。

转移要考虑从子树合并上来：

$$
\begin{cases}
f[u][x + 1] \leftarrow f[l][x] \\
f[u][y + 1] \leftarrow f[r][y] \\
f[u][x + y + 2] \leftarrow f[l][x] + f[r][y]
\end{cases}
$$

这其实就是在树上做了一个背包。

### 2.1 边权与点权

上述做法基于边权，下面我们把难搞的边权化为点权处理，观察题目树形结构，我们发现每条边都可以唯一对应一个深的节点。

由此，保留 $Q$ 条边，其实就等于保留 $Q + 1$ 个点（根节点无边权）。

所以，这就变成了一个 “有依赖的背包问题”。

我们可以定义 $f[u][j]$ 表示以 $u$ 为根节点的子树，内有 $j$ 条点的最大苹果数。

转移：

$$
f[u][j] = \max_{u \to v} ( \max_{0 \leq k \leq j - 1}( f[l][k] + f[r][j-k-1] +val[w]))
$$

- $f[l][k]$ 表示 $u$ 的左儿子 $l$ 子树内选了 $k$ 个节点。

- $f[r][j - k - 1]$ 表示 $u$ 的左儿子 $r$ 子树内选了 $j - k - 1$ 个节点。

- 最后加上 $u$ 节点，总共就是 $j$ 个节点。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxn = 100 + 5;

struct Edge { int v, w; };
vector< Edge > adj[maxn];
int val[maxn];
ll f[maxn][maxn];
// f[u][i] 以 u 为根的子树中，选 i 个节点的最大价值
int n, q;

void dfs( int u, int p )
{
    int l = 0, r = 0;
    for( auto [v, w] : adj[u] )
    {
        if( v == p ) continue;
        val[v] = w;
        if( l == 0 ) l = v;
        else r = v;
        dfs( v, u );
    }
 
    if( l == 0 && r == 0 ) 
    {
        f[u][1] = val[u];
        return;
    }

    for( int j = 1; j <= n; ++ j )
    {   // 当前子树总共 j 个点
        for( int k = 0; k <= j - 1; ++ k )
        {   // 左边 k 个，右边 j-k-1 个
            f[u][j] = max( f[u][j], f[l][k] + f[r][j-k-1] + val[u] );
        }
    }
}

int main( )
{
    cin >> n >> q;
	
    for( int i = 1; i < n; ++ i )
    {
        int u, v, w; 
        cin >> u >> v >> w;
        adj[u].push_back({ v, w });
        adj[v].push_back({ u, w });
    }
	
    dfs( 1, 0 );
	
    cout << f[1][q + 1] << '\n';
    return 0;
}
```
{% endfold %}

## 4. 复盘

- **复杂度分析**：$O( N \cdot Q^2 )$。

- **碎碎念**：点权和边权的互相转化。
