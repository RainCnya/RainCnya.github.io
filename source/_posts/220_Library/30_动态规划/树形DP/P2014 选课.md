# [P2014 [CTSC1997] 选课 - 洛谷](https://www.luogu.com.cn/problem/P2014)

## 1. 题面梗概

**中译中**：有 $n$ 门课，每门课有学分 $s_{i}$ 和唯一的前置课程 $k_{i}$。现在要从中选 $m$ 门课，要求选任何一门课前必须先选其先修课。求最大学分和。

> $1 \le n, m \le 300, 1 \le s_i \le 20$

## 2. 逻辑推导

将 $k_{i} \to i$，得到的图由一棵根为 $0$ 的外向树和若干环构成。环是没有意义的，所以问题被转化为在树上选择若干个节点，最大化权值，此时可以考虑 树形DP。

考虑到子树作为子结构的时候，无论怎么选课，只有 “子树中一共选取了多少课程” 这一条件能够影响其到父亲节点的转移。

很容易写出的一个定义是：$DP[u][w]$ 表示根为 $u$ 的子树内，选取 $w$ 门课程的最大权值。

### 2.1 分组背包

在转移到父节点时，对于每个子树直接枚举其选择了多少课程。但是直接枚举复杂度是错误的，简单分析发现这是个分组背包模型。

> 分组背包：物品有若干类（子树），同一类里只能选取一个物品。

在 $u$ 为根的子树中，先假设 $u$ 不选。用 $g[v][w]$ 来辅助背包的转移，表示当前考虑到子树 $v$，一共选取了 $w$ 门课程，权值和最大是多少。在枚举到下一个子树 $v'$ 时有转移：

$$
g[v'][w + w'] = \max_{0 \leq w + w' \leq m} {g[v][w] + f[v'][w']}
$$

然后再考虑 $u$ 怎么选的情况。当子树中选了 $0$ 门课程时，$u$ 可选可不选。一旦子树中选了课程时，$u$ 就不得不选了。记 $v$ 为最后一个枚举到的子树。

$$
f[u][0] = g[v][0] \ f[u][w] = g[v][w-1] + s[u] \quad (w > 0)
$$

### 2.2 0/1背包

上面这个转移有点麻烦，我们换个角度分析。

对于节点 $u$ 的每一个子节点 $v$：我们要从分配给 $u$ 的总名额 $j$ 中，分出 $k$ 个名额给 子树 $v$。

由于选择子树内的任何课程都必须选 $u$ 本身，所以 $u$ 必须占据 $1$ 个名额。

这样的转移就是：

$$
f[u][j] = \max_{u \to v} \left( \max_{0 \leq k \leq j - 1} \{ f[u][j-k] + f[v][k] \} \right)
$$

- $f[u][j-k]$ 表示在处理子树 $v$ 之前，已经从 $u$ 及其处理的子树中选了 $j - k$ 个节点的最优解。

- $f[v][k]$ 表示从当前子树 $v$ 中选取 $k$ 个点的最优解。


## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxn = 300 + 5;

vector< int > adj[maxn];
int s[maxn];
ll dp[maxn][maxn];
// dp[u][i] 表示以 u 为根的子树中，选 i 个点的最大价值
int n, m;

void dfs( int u )
{
    dp[u][1] = s[u];

    for( int v : adj[u] )
    {
        dfs( v );
        
        for( int i = m + 1; i >= 1; -- i )
        {
            for( int j = 0; j < i; ++ j )
            {
                dp[u][i] = max( dp[u][i], dp[u][i - j] + dp[v][j] );
            }
        }
    }
}

int main( )
{
    ios::sync_with_stdio(0);
    cin.tie(0);
    
    cin >> n >> m;
    for( int i = 1; i <= n; ++ i ) 
    {
        int k;
        cin >> k >> s[i];
        adj[k].push_back( i );
    }

    dfs( 0 );
    cout << dp[0][m + 1] << '\n';
    return 0;
}
```
{% endfold %}

## 4. 复盘

- **复杂度分析**：时间 $O( N \cdot M )$。
    
- **碎碎念**：同样是一道树上的背包问题，从 分组背包 和 01背包 两个角度切入会得到不同的思路，但是复杂度和数学模型是一致的。
