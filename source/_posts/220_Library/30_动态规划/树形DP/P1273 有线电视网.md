# [P1273 有线电视网 - 洛谷](https://www.luogu.com.cn/problem/P1273)

## 1. 题面梗概

**中译中**：给出一棵以 1 为根的树，叶子节点是用户，每个用户有愿意支付的钱；边有传输成本。要求在不亏本（总收益 $\ge$ 总成本）的前提下，最多能让多少个用户看到转播。

> $2 \le N \le 3000, 1 \le M \le N-1$，$N$ 为总节点数，$M$ 为用户数。

## 2. 逻辑推导

这是一道 树形DP 问题，发现还存在容量限制，我们考虑背包模型。

题目中涉及点权和边权，为了方便 DP 转移，我们可以将边权下沉 —— 将连接 $u$ 与其子节点 $v$ 的边权 $w$ 视作进入子树 $v$ 必须支付的门槛。

### 2.1 树上背包

不同于常规背包求 “最大价值”，本题要求在 "不亏本" 的前提下，最大化用户数量，因此我们换个角度设 "用户数量" 为背包的容量。

定义 $dp[u][j]$ 为：在以 $u$ 为根的子树中，选择 $j$ 个用户所能获得的 *最大收益*（收益 = 支付总额 - 成本总额）。

**状态转移**： 对于节点 $u$ 的每一个子节点 $v$（边权为 $w$）：

$$dp[u][j] = \max_{k=1}^{j} \{ dp[u][j-k] + dp[v][k] - w \}$$

 $dp[u][j-k]$：在当前子树 $v$ 之前，已经从 $u$ 的其他分支选了 $j-k$ 个用户的最大收益。
    
$dp[v][k] - w$：在当前子树 $v$ 中选 $k$ 个用户，同时扣除连接 $u, v$ 的传输成本 $w$。

最后找出最大的不亏本的 $j$ 即可。

### 2.2 优化

由于本题 $N, M \leq 3000$，所以我们必须使用 $O( NM )$ 的树上背包优化：

- **子树规模限制**：在枚举 $j$ 和 $k$ 时，其上限应严格限制为当前已处理的子树中的 **用户规模**。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxn = 3000 + 5;
const int inf = 0x3f3f3f3f;

struct Edge { int v, w; };
vector< Edge > adj[maxn];
int dp[maxn][maxn];
// dp[u][j] 表示在 u 的子树中选 j 个点的最大收益
int sz[maxn];
// sz[u] 表示子树 u 中用户数量
int money[maxn];
int n, m;

void dfs( int u, int p )
{
    for( int j = 0; j <= m; ++ j ) dp[u][j] = -inf;
    dp[u][0] = 0;
	
	// 叶子结点处理
    if( u > n - m )
    {
        dp[u][1] = money[u];
        sz[u] = 1;
        return;
    }
	
    int tot = 0;
    for( auto [v, w] : adj[u] )
    {
        if( v == p ) continue;
        dfs( v, u );
        tot += sz[v];

        for( int j = tot; j >= 1; -- j )
        {
            for( int k = 1; k <= j && k <= sz[v]; ++ k )
            {
                if( dp[u][j - k] == -inf || dp[v][k] == -inf ) continue;
                dp[u][j] = max( dp[u][j], dp[u][j - k] + dp[v][k] - w );
            }
        }
        sz[u] = tot;
    }
}

int main( )
{
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> n >> m;
    for( int i = 1; i <= n - m; ++ i )
    {
        int k, v, w;
        cin >> k;
        for( int j = 1; j <= k; ++ j )
        {
            cin >> v >> w;
            adj[i].push_back({ v, w });
        }
    }

    for( int i = n - m + 1; i <= n; ++ i ) cin >> money[i];

    dfs( 1, 0 );

    for( int j = m; j >= 1; -- j )
    {
        if( dp[1][j] >= 0 )
        {
            cout << j << '\n';
            break;
        }
    }
    return 0;
}
```

## 4. 复盘

- **复杂度分析**：$O( NM )$。
    
- **碎碎念**：算是背包问题的变种，在满足 “不亏本” 的条件下，找到最多的 “用户数量”，需要换个角度切入。
