# [P1352 没有上司的舞会 - 洛谷](https://www.luogu.com.cn/problem/P1352)

## 1. 题面梗概

**中译中**：给出一棵 $n$ 个节点的树（职员从属关系），每个节点带权。需要选出一些点，使得它们的权值和最大，并且不存在两个点使得它们是父子关系。

> $1 \le n \le 6 \times 10^3, -128 \le r_i \le 127$

## 2. 逻辑推导

这是一个很经典的 树形DP 问题。

思考如何用子节点的状态推导出父节点的状态。

设 $f[u]$ 表示 $u$ 号点所在的子树内的最大权值和，$(u, v)$ 为父子关系，可以写出以下式子：

$$ 
f[u] = \sum_{u \to v} f[v]
$$
但是这样无法保证满足题目约束条件，所以需要设立更多的状态。

### 2.1 约束条件

题目中限制了父子关系，而上述转移是子节点向父节点转移。两者相一致，只需要记录自身相关信息即可，这样转移时，既有父节点的信息，也拥有子节点的信息。

设 $f[u][0]$ 表示 $u$ 节点没被选时，$u$ 所在子树的最大权值和，$f[u][1]$ 表示 $u$ 节点被选择时，那么：

$$
f[u][0] = \sum_{u \to v} \max(f[v][0], f[v][1]) \
$$

$$
f[u][1] = \sum_{u \to v} f[v][0]
$$
此处转移略去了 $f[v][1] \to f[u][1]$ 的状态，这组是不满足条件的。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxn = 6e3 + 5;

vector< int > adj[maxn];

ll f[maxn][2];
// f[u][0/1] 0不选 / 1选 u 的最大价值
int a[maxn];
int n;

void dfs( int u, int p )
{
    f[u][1] = a[u];
    for( int v : adj[u] )
    {
        if( v == p ) continue;
        dfs( v, u );
        f[u][0] += max( f[v][0], f[v][1] );
        f[u][1] += f[v][0];
    }
}

int main( )
{
    cin >> n;
    for( int i = 1; i <= n; ++ i ) cin >> a[i];
	   
    for( int i = 1; i < n; ++ i )
    {
        int u, v; cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
	
    dfs( 1, 0 );
	
    cout << max( f[1][0], f[1][1] ) << '\n';
    return 0;
}
```
