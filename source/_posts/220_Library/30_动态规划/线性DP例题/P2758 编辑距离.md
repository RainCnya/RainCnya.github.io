# [P2758 编辑距离 - 洛谷](https://www.luogu.com.cn/problem/P2758)

## 1. 题面梗概

**中译中**：给定两个字符串 $A$ 和 $B$。通过“插入”、“删除”、“替换”三种操作，求把 $A$ 改造为 $B$ 所需的最少步数。

> $1 \le |A|, |B| \le 2000$

## 2. 逻辑推导

因为此题的字符串长度似乎在操作中发生了变化，不好入手。但我们仍然可以大胆尝试 DP。

定义：$f[i][j]$ 表示将字符串 $A$ 的前 $i$ 个字符，转化为字符串 $B$ 的前 $j$ 个字符，的最小操作次数。

这样只需要考虑新的字符会对前继产生什么影响就行了。

### 2.1 转移

考虑 $f[i][j]$ 的来源：当前观察到 $a[i]$ 和 $b[j]$ 了。
- 可以视为 $f[i-1][j]$ 多加一个 $a[i]$，一次删除；
- 可以视为 $f[i][j-1]$ 多加一个 $b[i]$，一次插入；
- 也可以视为 $f[i-1][j-1]$，如果 $a[i] \neq b[j]$，一次替换。

那就很容易写出下方转移方程了：

$$
f[i][j] = \min( f[i-1][j] + 1, f[i][j-1] + 1, f[i-1][j-1] + [a[i] \neq a[j]] )
$$

> 其中 $[A_i \neq B_j]$ 是一个指示函数，相等为 0，不等为 1。

## 3. 代码实现

{% fold info @AcCode %}
```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;

const int maxn = 2e3 + 5;

int f[maxn][maxn];
// f[i][j] 表示 A 前 i 个字符 完全转化为 B 前 j 个字符 的最少操作次数
string A, B;

int main( )
{
    cin >> A >> B;
    int lenA = A.length( ), lenB = B.length( );
	
	// 注意边界情况
    for( int i = 1; i <= lenA; ++ i ) f[i][0] = i;
    for( int j = 1; j <= lenB; ++ j ) f[0][j] = j;
	
    for( int i = 1; i <= lenA; ++ i )
    {
        for( int j = 1; j <= lenB; ++ j )
        {
            f[i][j] = min( f[i - 1][j] , f[i][j - 1] ) + 1;
            f[i][j] = min( f[i][j], f[i - 1][j - 1] + ( A[i-1] != B[j-1] ) );
        }
    }
    
    cout << f[lenA][lenB] << '\n';
    return 0;
}
```
{% endfold %}

- **复杂度分析**：$O( |A| \cdot |B| )$。
