# 模意义下的指数问题

> 模乘法群描述可逆剩余类的乘法结构；
> 阶、原根、离散对数与二次剩余则研究幂在这一结构中的规律。

## 1 从模乘法到指数结构

模意义下的加法和乘法可以不断取模，但指数运算还会表现出周期、生成与求逆等更深的结构。

> 本章主线：剩余系 → 模乘法群 → 欧拉定理与指数降幂 → 元素的阶 → 原根 → 离散对数 → 二次剩余。

这些内容共同研究方程 $a^x\equiv b\pmod m$：有时需要计算一个巨大指数，有时需要寻找未知指数，有时则要求解模意义下的平方根。

---

## 2 剩余系与模乘法群

### 2.1 完全剩余系

模 $m$ 的每个剩余类中各取一个代表元，得到的集合称为模 $m$ 的完全剩余系。

最常用的完全剩余系是 $\{0,1,2,\ldots,m-1\}$。

若一个集合中任意两个元素模 $m$ 不同余，并且共有 $m$ 个元素，那么它就是模 $m$ 的完全剩余系。

### 2.2 既约剩余系

在完全剩余系中，只保留与 $m$ 互质的代表元，得到的集合称为模 $m$ 的既约剩余系。

例如，模 $10$ 的一个既约剩余系是 $\{1,3,7,9\}$。

既约剩余系中的元素个数为欧拉函数 $\varphi(m)$。

### 2.3 模乘法群

模 $m$ 的可逆剩余类在模乘法下构成阿贝尔群，记作

$$
(\mathbb Z/m\mathbb Z)^\times.
$$

其中：

- 元素是与 $m$ 互质的剩余类；
- 运算是模 $m$ 乘法；
- 单位元是 $[1]_m$；
- 每个元素都存在乘法逆元；
- 群的大小为 $\varphi(m)$。

> [!note] 与代数结构的联系
> 这里的“群的阶”指群中元素的个数；后文“元素的阶”则指一个元素经过多少次幂回到单位元。
>
> 参见：[[02_离散结构/02_代数运算结构#群]]

## 3 欧拉定理与指数降幂

### 3.1 欧拉函数回顾

若 $\displaystyle m=\prod_{i=1}^{k}p_i^{a_i}$，则

$$
\varphi(m)
=m\prod_{i=1}^{k}\left(1-\frac1{p_i}\right).
$$

欧拉函数的定义、质数幂公式与线性筛递推参见 [[03_数论函数与线性筛#4 欧拉函数]]。

### 3.2 欧拉定理

若 $\gcd(a,m)=1$，则

$$
a^{\varphi(m)}\equiv1\pmod m.
$$

原因是：把既约剩余系中的每个元素都乘以 $a$，取模后仍然得到同一个既约剩余系，只是顺序发生了变化。将所有元素相乘并约去原乘积，就得到欧拉定理。

因此，当 $a$ 与 $m$ 互质时，指数可以模 $\varphi(m)$：

$$
a^b\equiv a^{\,b\bmod\varphi(m)}\pmod m.
$$

### 3.3 费马小定理

若 $p$ 是质数且 $p\nmid a$，则 $\varphi(p)=p-1$，欧拉定理退化为

$$
a^{p-1}\equiv1\pmod p.
$$

这就是费马小定理，也是模质数快速求逆元的依据。

### 3.4 扩展欧拉定理

当 $a$ 与 $m$ 不互质时，不能直接把指数对 $\varphi(m)$ 取模。

更一般地，对任意非负整数 $b$ 都有

$$
a^b\equiv
\begin{cases}
a^b, & b<\varphi(m),\\
a^{\,b\bmod\varphi(m)+\varphi(m)}, & b\ge\varphi(m)
\end{cases}
\pmod m.
$$

因此，处理十进制大指数时，不仅需要计算 $b\bmod\varphi(m)$，还要判断原指数是否已经达到 $\varphi(m)$。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
using i128 = __int128_t;

// 扩展欧拉定理：十进制大指数降幂
namespace ExEuler {
/* ======================================== */
ll mul( ll a, ll b, ll mod ) {
    return (ll)( (i128)a * b % mod );
}

ll qpow( ll a, i128 k, ll mod ) {
    ll res = 1 % mod;
    a %= mod;
    for( ; k; k >>= 1, a = mul( a, a, mod ) ) {
        if( k & 1 ) res = mul( res, a, mod );
    }
    return res;
}

ll phi( ll n ) {
    ll res = n;
    for( ll p = 2; p <= n / p; ++ p ) {
        if( n % p != 0 ) continue;
        res = res / p * ( p - 1 );
        while( n % p == 0 ) n /= p;
    }
    if( n > 1 ) res = res / n * ( n - 1 );
    return res;
}

// 返回扩展欧拉定理中实际使用的指数
i128 reduce_exp( const string &s, ll ph ) {
    ll rem = 0;
    bool large = false;

    for( char ch : s ) {
        int digit = ch - '0';
        i128 cur = (i128)rem * 10 + digit;

        if( cur >= ph ) large = true;
        rem = (ll)( cur % ph );
    }

    return large ? rem + ph : rem;
}

// 求 a^b mod mod，其中 b 是非负十进制整数字符串
ll solve( ll a, const string &b, ll mod ) {
    if( mod == 1 ) return 0;
    ll ph = phi( mod );
    i128 k = reduce_exp( b, ph );
    return qpow( a, k, mod );
}
/* ======================================== */
}
```

## 4 阶与原根

### 4.1 元素的阶

设 $\gcd(a,m)=1$。满足 $a^k\equiv1\pmod m$ 的最小正整数 $k$，称为 $a$ 模 $m$ 的阶，记作 $\operatorname{ord}_m(a)$。

若 $a^n\equiv1\pmod m$，则 $\operatorname{ord}_m(a)\mid n$。

结合欧拉定理，可以得到

$$
\operatorname{ord}_m(a)\mid\varphi(m).
$$

因此，$a^0,a^1,\ldots,a^{\operatorname{ord}_m(a)-1}$ 模 $m$ 两两不同，而之后的幂按这一长度循环。

### 4.2 原根

若 $\operatorname{ord}_m(g)=\varphi(m)$，则称 $g$ 为模 $m$ 的原根。

此时

$$
1,g,g^2,\ldots,g^{\varphi(m)-1}
$$

恰好遍历整个模 $m$ 既约剩余系。

从群论角度看，原根就是模乘法群的生成元。模 $m$ 存在原根，等价于 $(\mathbb Z/m\mathbb Z)^\times$ 是循环群。

### 4.3 原根的存在与判定

对 $m\ge2$，模 $m$ 存在原根，当且仅当

$$
m=2,\ 4,\ p^k,\ 2p^k,
$$

其中 $p$ 是奇质数，$k$ 是正整数。

若模 $m$ 存在原根，则 $g$ 是原根，当且仅当对 $\varphi(m)$ 的每个不同质因数 $q$ 都有

$$
g^{\varphi(m)/q}\not\equiv1\pmod m.
$$

因此，只需要分解 $\varphi(m)$，从小到大枚举 $g$ 即可寻找最小原根。

若原根存在，其总数为 $\varphi(\varphi(m))$。

### 4.4 最小原根

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
using i128 = __int128_t;

// 原根：存在性判定与最小原根
namespace PrimitiveRoot {
/* ======================================== */
ll mul( ll a, ll b, ll mod ) {
    return (ll)( (i128)a * b % mod );
}

ll qpow( ll a, ll k, ll mod ) {
    ll res = 1 % mod;
    a %= mod;
    for( ; k; k >>= 1, a = mul( a, a, mod ) ) {
        if( k & 1 ) res = mul( res, a, mod );
    }
    return res;
}

ll phi( ll n ) {
    ll res = n;
    for( ll p = 2; p <= n / p; ++ p ) {
        if( n % p != 0 ) continue;
        res = res / p * ( p - 1 );
        while( n % p == 0 ) n /= p;
    }
    if( n > 1 ) res = res / n * ( n - 1 );
    return res;
}

vector<ll> distinct_factors( ll n ) {
    vector<ll> fac;
    for( ll p = 2; p <= n / p; ++ p ) {
        if( n % p != 0 ) continue;
        fac.push_back( p );
        while( n % p == 0 ) n /= p;
    }
    if( n > 1 ) fac.push_back( n );
    return fac;
}

// n = 2, 4, p^k, 2p^k，其中 p 为奇质数
bool exist( ll n ) {
    if( n == 2 || n == 4 ) return true;
    if( n < 2 ) return false;

    if( n % 2 == 0 ) n /= 2;
    if( n % 2 == 0 ) return false;

    ll p = 0;
    for( ll i = 3; i <= n / i; i += 2 ) {
        if( n % i == 0 ) {
            p = i;
            break;
        }
    }

    if( p == 0 ) return n > 1;

    while( n % p == 0 ) n /= p;
    return n == 1;
}

// 不存在原根时返回 -1
ll min_root( ll n ) {
    if( !exist( n ) ) return -1;
    if( n == 2 ) return 1;
    if( n == 4 ) return 3;

    ll ph = phi( n );
    vector<ll> fac = distinct_factors( ph );

    for( ll g = 2; g < n; ++ g ) {
        if( std::gcd( g, n ) != 1 ) continue;

        bool ok = true;
        for( ll q : fac ) {
            if( qpow( g, ph / q, n ) == 1 ) {
                ok = false;
                break;
            }
        }
        if( ok ) return g;
    }

    return -1;
}
/* ======================================== */
}
```

## 5 离散对数

### 5.1 问题形式

离散对数问题要求解

$$
a^x\equiv b\pmod m
$$

中的最小非负整数 $x$。

它与实数中的对数类似，但模意义下的幂具有周期，普通的取对数方法无法使用。

当 $g$ 是模 $m$ 的原根时，每个可逆剩余类都可以唯一写成 $g^x$，于是模乘法可以转化为指数上的模加法。

### 5.2 BSGS

当 $\gcd(a,m)=1$ 时，可以使用 Baby-Step Giant-Step 算法。

取 $t=\lceil\sqrt m\rceil$，将未知指数写成 $x=it-j$，其中 $0\le j<t$。

原方程转化为

$$
a^{it}\equiv ba^j\pmod m.
$$

先枚举小步 $j$，把 $ba^j$ 存入哈希表；再枚举大步 $i$，寻找 $(a^t)^i$ 是否出现过。

时间复杂度和空间复杂度均为 $O(\sqrt m)$。

### 5.3 ExBSGS

若 $\gcd(a,m)\ne1$，普通 BSGS 不能直接使用。

设 $d=\gcd(a,m)$。若 $d\nmid b$，方程无解；否则可以同时约去一个 $d$，并记录左侧被提取出来的系数。

不断重复这一过程，直到底数与新模数互质，再转化为普通 BSGS。

### 5.4 实现

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
using i128 = __int128_t;

// BSGS / ExBSGS：求 a^x = b (mod mod) 的最小非负整数解
namespace DiscreteLog {
/* ======================================== */
ll mul( ll a, ll b, ll mod ) {
    return (ll)( (i128)a * b % mod );
}

ll qpow( ll a, ll k, ll mod ) {
    ll res = 1 % mod;
    a %= mod;
    for( ; k; k >>= 1, a = mul( a, a, mod ) ) {
        if( k & 1 ) res = mul( res, a, mod );
    }
    return res;
}

ll exgcd( ll a, ll b, ll &x, ll &y ) {
    if( b == 0 ) {
        x = 1, y = 0;
        return a;
    }
    ll d = exgcd( b, a % b, y, x );
    y -= a / b * x;
    return d;
}

ll inv( ll a, ll mod ) {
    ll x, y;
    ll d = exgcd( a, mod, x, y );
    if( d != 1 ) return -1;
    return ( x % mod + mod ) % mod;
}

ll ceil_sqrt( ll n ) {
    ll t = sqrtl( (long double)n );
    while( (i128)t * t < n ) ++ t;
    while( t > 0 && (i128)( t - 1 ) * ( t - 1 ) >= n ) -- t;
    return t;
}

// 要求 gcd(a,mod)=1
ll bsgs( ll a, ll b, ll mod ) {
    if( mod == 1 ) return 0;

    a = ( a % mod + mod ) % mod;
    b = ( b % mod + mod ) % mod;

    if( b == 1 ) return 0;
    if( std::gcd( a, mod ) != 1 ) return -1;

    ll t = ceil_sqrt( mod );
    unordered_map<ll, ll> mp;

    ll cur = b;
    for( ll j = 0; j < t; ++ j ) {
        mp[cur] = j;
        cur = mul( cur, a, mod );
    }

    ll step = qpow( a, t, mod );
    cur = step;

    for( ll i = 1; i <= t + 1; ++ i ) {
        auto it = mp.find( cur );
        if( it != mp.end( ) ) {
            return i * t - it->second;
        }
        cur = mul( cur, step, mod );
    }

    return -1;
}

// 不要求 gcd(a,mod)=1
ll exbsgs( ll a, ll b, ll mod ) {
    if( mod == 1 ) return 0;

    a = ( a % mod + mod ) % mod;
    b = ( b % mod + mod ) % mod;

    if( b == 1 ) return 0;
    if( a == 0 ) return b == 0 ? 1 : -1;

    ll coeff = 1;
    ll add = 0;

    while( true ) {
        ll d = std::gcd( a, mod );
        if( d == 1 ) break;
        if( b % d != 0 ) return -1;

        b /= d;
        mod /= d;
        coeff = mul( coeff, a / d, mod );
        ++ add;

        if( coeff == b ) return add;
    }

    ll inv_coeff = inv( coeff, mod );
    ll target = mul( b, inv_coeff, mod );
    ll rest = bsgs( a, target, mod );

    return rest == -1 ? -1 : rest + add;
}
/* ======================================== */
}
```

## 6 二次剩余与 Cipolla

### 6.1 二次剩余

设 $p$ 为奇质数。若同余方程

$$
x^2\equiv n\pmod p
$$

存在解，则称 $n$ 是模 $p$ 的二次剩余；否则称为二次非剩余。

当 $n\not\equiv0\pmod p$ 时，欧拉判别准则给出

$$
n^{(p-1)/2}\equiv
\begin{cases}
1\pmod p, & n\text{ 是二次剩余},\\
-1\pmod p, & n\text{ 是二次非剩余}.
\end{cases}
$$

若 $x$ 是一个非零解，则另一个解是 $p-x$。

### 6.2 Cipolla 的核心构造

Cipolla 算法随机选择一个 $a$，使 $w=a^2-n$ 是模 $p$ 的二次非剩余。

随后在扩域中引入满足 $i^2=w$ 的元素，并计算

$$
(a+i)^{(p+1)/2}.
$$

结果的实部就是原方程的一个平方根。

随机选取的 $w$ 约有一半是二次非剩余，因此通常只需要常数次尝试；主要复杂度来自扩域快速幂，为 $O(\log p)$。

### 6.3 实现

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
using i128 = __int128_t;

// Cipolla：解 x^2 = n (mod p)，p 为质数
namespace Cipolla {
/* ======================================== */
ll P, w;
mt19937_64 rng( chrono::steady_clock::now( ).time_since_epoch( ).count( ) );

ll mul( ll a, ll b ) {
    return (ll)( (i128)a * b % P );
}

ll qpow( ll a, ll k ) {
    ll res = 1 % P;
    a %= P;
    for( ; k; k >>= 1, a = mul( a, a ) ) {
        if( k & 1 ) res = mul( res, a );
    }
    return res;
}

struct Comp {
    ll r, i;
};

Comp mul( Comp a, Comp b ) {
    ll real = ( mul( a.r, b.r ) + mul( mul( a.i, b.i ), w ) ) % P;
    ll imag = ( mul( a.r, b.i ) + mul( a.i, b.r ) ) % P;
    return { real, imag };
}

Comp qpow( Comp a, ll k ) {
    Comp res{ 1, 0 };
    for( ; k; k >>= 1, a = mul( a, a ) ) {
        if( k & 1 ) res = mul( res, a );
    }
    return res;
}

bool is_residue( ll n ) {
    if( n == 0 ) return true;
    return qpow( n, ( P - 1 ) / 2 ) == 1;
}

// 无解返回 {-1,-1}
pair<ll, ll> solve( ll n, ll p ) {
    P = p;
    n = ( n % P + P ) % P;

    if( P == 2 ) return { n, n };
    if( n == 0 ) return { 0, 0 };
    if( !is_residue( n ) ) return { -1, -1 };

    ll a;
    while( true ) {
        a = uniform_int_distribution<ll>( 0, P - 1 )( rng );
        w = ( mul( a, a ) - n + P ) % P;

        if( qpow( w, ( P - 1 ) / 2 ) == P - 1 ) break;
    }

    ll x = qpow( Comp{ a, 1 }, ( P + 1 ) / 2 ).r;
    ll y = ( P - x ) % P;

    if( x > y ) swap( x, y );
    return { x, y };
}
/* ======================================== */
}
```
