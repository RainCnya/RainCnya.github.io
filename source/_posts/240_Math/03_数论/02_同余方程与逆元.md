# 同余方程与逆元

> 同余把整数等式推广到模意义下；
> 裴蜀定理与扩展欧几里得算法则负责判断并构造其中的解。

## 1 从整数等式到模意义下的等式

普通整数等式要求两个整数完全相同，而同余只关心它们相差多少个模数。

> 本章主线：同余与剩余类 → 最大公因数与裴蜀定理 → 扩展欧几里得算法 → 一次不定方程 → 线性同余方程 → 乘法逆元 → 中国剩余定理 → 扩展中国剩余定理。

这些内容表面上分别处理方程、除法和方程组，核心始终是整数线性方程 $ax+by=c$。

---

## 2 同余与剩余类

### 2.1 同余的定义

设 $m$ 为正整数，$a,b\in\mathbb Z$。若 $m\mid(a-b)$，则称 $a$ 与 $b$ 模 $m$ 同余，记作 $a\equiv b\pmod m$

它等价于 $a,b$ 除以 $m$ 后具有相同余数。

例如：$17\equiv 5\pmod{12}$，因为 $12\mid(17-5)$。

### 2.2 剩余类

整数 $a$ 模 $m$ 的剩余类定义为 $[a]_m=\{a+km\mid k\in\mathbb Z\}$。

两个整数属于同一个剩余类，当且仅当它们模 $m$ 同余：$[a]_m=[b]_m \Longleftrightarrow a\equiv b\pmod m$。

同余关系将整数划分为 $m$ 个剩余类，通常选取 $0,1,2,\ldots,m-1$ 作为它们的代表元。

所有模 $m$ 剩余类构成集合 $\mathbb Z/m\mathbb Z$。

> [!note] 与关系理论的联系
> 同余是等价关系的典型例子，它按照余数把整数划分为互不相交的等价类。
>
> 参见：[[01_数学语言与证明/02_关系#等价关系与集合划分]]

### 2.3 同余运算

若 $a\equiv b\pmod m$ 且 $c\equiv d\pmod m$，则可以直接进行：

$$
\begin{align}
a \pm c &\equiv b \pm d \pmod m \\
ac &\equiv bd \pmod m \\
a^k &\equiv b^k \pmod{m} \\	
\end{align}
$$

因此，加法、减法、乘法和非负整数次幂都可以在过程中不断取模。

除法则需要额外条件。由 $ac\equiv bc\pmod m$，设 $d=\gcd(c,m)$，一般只能推出 $a\equiv b\pmod{m/d}$，

只有当 $\gcd(c,m)=1$ 时，才能在模 $m$ 意义下直接约去 $c$。

## 3 欧几里得算法与裴蜀定理

### 3.1 欧几里得算法

对正整数 $a,b$，设 $a = qb+r, \quad 0\le r<b$，一个整数同时整除 $a,b$，当且仅当它同时整除 $b,r$，

因此 $\gcd(a,b)=\gcd(b,a\bmod b)$ 不断递归，直到余数为 $0$，即可得到最大公因数。

{% fold info @欧几里得算法 %}
```cpp
ll gcd( ll a, ll b ) {
	if( b == 0 ) return a;
	else return gcd( b, a % b );
}
```
{% endfold %}

时间复杂度为 $O(\log\min(a,b))$。

### 3.2 裴蜀定理

**裴蜀定理**：对不全为 $0$ 的整数 $a,b$，存在整数 $x,y$，使得 $ax+by = \gcd(a,b)$。

更一般地，方程 $ax+by = c$ 有整数解，当且仅当 $\gcd(a,b)\mid c$。

裴蜀定理说明：所有形如 $ax+by$ 的整数线性组合，恰好构成 $\gcd(a,b)$ 的全部整数倍。

因此：若 $c$ 不是 $\gcd(a,b)$ 的倍数，方程一定无解，若是其倍数，则可以将等式整体放大得到一组解。

### 3.3 扩展欧几里得算法

扩展欧几里得算法在计算 $\gcd(a,b)$ 的同时，构造一组系数 $x,y$：$ax+by=\gcd(a,b)$

设递归已经求得 $bx'+(a\bmod b)y' = d$，

> 其中 $d=\gcd(b,a\bmod b)=\gcd(a,b)$。

因为 $a\bmod b=a-\left\lfloor a / b\right\rfloor b$，代入可得：$d=ay'+b\left(x'-\left\lfloor\frac ab\right\rfloor y'\right)$，

因此新的系数为 $x=y', \quad y=x'-\left\lfloor\frac ab\right\rfloor y'$。

> 递归边界为 $b=0$，此时 $a \cdot 1 + 0 \cdot 0 = a$。

{% fold info @扩展欧几里得算法 %}
```cpp
// 扩展欧几里得：求 ax + by = gcd(a,b) 的一组整数解
ll exgcd( ll a, ll b, ll &x, ll &y ) {
    if( b == 0 ) { x = 1, y = 0; return a; }
    ll d = exgcd( b, a % b, y, x );
    y -= a / b * x;
    return d;
}
```
{% endfold %}

> 该写法在递归调用中交换 `x, y` 的位置，使返回后的 `x` 已经对应推导中的 $y'$，再修正 `y` 即可。

若 $(x_0,y_0)$ 是 $ax+by=d$ 的一组解，则对任意 $t\in\mathbb Z$，$x=x_0+t\frac bd, y=y_0-t\frac ad$ 仍然是解。

扩展欧几里得只负责计算出其中的一组。

## 4 一次不定方程与线性同余方程

### 4.1 一次不定方程

考虑整数方程 $ax + by = c$，其中 $a,b,c\in\mathbb Z$，且 $a,b$ 不全为 $0$。

设 $d=\gcd(a,b)$。由裴蜀定理，方程有整数解当且仅当 $d\mid c$，若 $d\nmid c$，方程无整数解。

若有解，先用 Exgcd 求出一组特解 $ax^{*} + by^{*} = d$，再乘以 $c / d$，得到一组特解：

$$
x_0=x^*\frac cd, \qquad y_0=y^*\frac cd.
$$

若 $(x_0,y_0)$ 是一组特解，则全部整数解为

$$
\boxed{
\begin{aligned}
x&=x_0+t\frac bd,\\
y&=y_0-t\frac ad,
\end{aligned}
\qquad t\in\mathbb Z.
}
$$


### 4.2 线性同余方程

形如 $ax \equiv c \pmod m$ 的方程称为一元线性同余方程，其中 $m>0$。

由同余定义，$ax\equiv c\pmod m$ 等价于 $m \mid (ax-c)$，也就是存在整数 $y$，使得 $ax + my = c$。

所以线性同余方程本质上就是一次不定方程。

因此，设 $d = \gcd(a, m)$，方程有解当且仅当 $d \mid c$。

若有解，将方程同时除以 $d$：$\displaystyle \frac{a}{d}x \equiv \frac{c}{d} \pmod{ \frac{m}{d} }$。

此时 $a / d$ 与 $m / d$ 互质，可以乘上逆元，若 $x_{0}$ 是一组解，那么 $x\equiv x_0\pmod{m/d}$。

也就是说：解的周期是 $m/d$，在模 $m$ 的完整剩余系中，共有 $d$ 个不同解类。

$$
x_0,\ x_0+\frac md,\ x_0+2\frac md, \ldots ,\ x_0+(d-1)\frac md.
$$

特别地，当 $d=1$ 时，方程在模 $m$ 意义下有唯一解。

## 5 乘法逆元

### 5.1 定义与存在条件

若存在整数 $x$ 满足 $ax\equiv1\pmod m$，则称 $x$ 是 $a$ 模 $m$ 的乘法逆元，记作 $a^{-1}\pmod m$。

逆元是剩余类乘法中的逆元素，不是普通实数意义下的 $1/a$。

这是线性同余方程在 $c = 1$ 时的特殊情形，因此逆元存在当且仅当 $\gcd(a, m) = 1$。

当逆元存在时，它在模 $m$ 意义下唯一。

**模意义下的除法**：在 $b$ 可逆时，$\displaystyle \frac ab\pmod m$ 应理解为 $a\cdot b^{-1}\pmod m$。

### 5.2 扩展欧几里得求逆元

要求 $a^{-1}\pmod m$，等价于求 $ax + my = 1$

只要 $\gcd(a,m)=1$，扩展欧几里得就能给出一组系数 $x,y$，其中 $x$ 就是 $a$ 的一个逆元。

{% fold info @Exgcd 求逆元 %}
```cpp
// Exgcd 求逆元
ll exgcd( ll a, ll b, ll &x, ll &y ) {
    if( b == 0 ) { x = 1, y = 0; return a; }
    ll d = exgcd( b, a % b, y, x );
    y -= a / b * x;
    return d;
}

ll exgcd_inv( ll a, ll mod ) {
    ll x, y; 
    ll d = exgcd( a, mod, x, y );
    if( d != 1 ) return -1;
    return ( x % mod + mod ) % mod;
}
```
{% endfold %}

它只要求 $\gcd(a,m)=1$，模数不必是质数。

### 5.3 费马小定理求逆元

若 $p$ 是质数且 $p\nmid a$，费马小定理给出 $a^{p-1}\equiv1\pmod p$，两边乘以 $a^{-1}$ 得到 $a^{-1}\equiv a^{p-2}\pmod p$。

因此可以使用快速幂求逆元。

{% fold info @费马小定理求逆元 %}
```cpp
// 费马小定理求逆元
ll mul( ll a, ll b, ll mod ) { 
    return (ll)( (i128)a * b % mod ); 
}

ll qpow( ll a, ll k, ll mod ) {
    ll res = 1 % mod; a %= mod;
    for( ; k; k >>= 1, a = mul( a, a, mod ) )
        if( k & 1 ) res = mul( res, a, mod );
    return res;
}

ll fermat_inv( ll a, ll mod ) { 
    return qpow( a, mod - 2, mod ); 
}
```
{% endfold %}

这一方法要求模数为质数，并且 $a$ 不是模数的倍数。

### 5.4 线性递推求连续整数的逆元

当模数 $p$ 为质数且 $1\le n<p$ 时，可以在线性时间内求出 $1,2,\ldots,n$ 的全部逆元。

设 $\displaystyle p=\left\lfloor\frac pi\right\rfloor i+(p\bmod i).$

在模 $p$ 意义下整理，可以得到下列递推式：

$$
\operatorname{inv}[i] =\left(p-\left\lfloor\frac pi\right\rfloor\right) \operatorname{inv}[p\bmod i] \bmod p.
$$

{% fold info @线性递推求逆元 %}
```cpp
// 线性递推求逆元
ll inv[maxn];

void init_linear( int N, ll mod ) {
    inv[1] = 1;
    for( int i = 2; i <= N; ++ i ) inv[i] = ( mod - mod / i ) * inv[mod % i] % mod;
}
```
{% endfold %}

条件 $N<p$ 保证区间内每个整数都可逆。

### 5.5 一个数组的全部逆元

给定 $a_1,a_2,\ldots,a_n$，若每个 $a_i$ 在模 $m$ 下都可逆，可以只求一次总乘积的逆元，再在线性时间内得到每个元素的逆元。

定义前缀积 $\displaystyle P_i=\prod_{j=1}^{i}a_j$， 先求 $P_n^{-1}$，然后从右向左维护后缀乘积，即可得到 $a_i^{-1} = \left(\prod_{j \neq i}a_j\right) P_n^{-1}$。

{% fold info @数组整体求逆 %}
```cpp
// 数组整体求逆：默认 mod 为质数
ll inv[maxn], pre[maxn], suf[maxn];

ll mul( ll a, ll b, ll mod ) { 
    return (ll)( (i128)a * b % mod ); 
}

ll qpow( ll a, ll k, ll mod ) {
    ll res = 1 % mod; a %= mod;
    for( ; k; k >>= 1, a = mul( a, a, mod ) )
        if( k & 1 ) res = mul( res, a, mod );
    return res;
}

ll fermat_inv( ll a, ll mod ) { 
    return qpow( a, mod - 2, mod ); 
}

void init_array( int N, ll val[], ll mod ) {
    pre[0] = 1;
    for( int i = 1; i <= N; ++ i ) pre[i] = mul( pre[i - 1], val[i], mod );
    suf[N] = fermat_inv( pre[N], mod );
    for( int i = N; i >= 1; -- i ) {
        inv[i] = mul( pre[i - 1], suf[i], mod );
        if( i > 1 ) suf[i - 1] = mul( suf[i], val[i], mod );
    }
}
```
{% endfold %}

这一方法不要求模数为质数，只要求每个元素都与模数互质。

## 6 中国剩余定理

考虑同余方程组

$$
\begin{cases}
x\equiv a_1\pmod{m_1}, \\
x\equiv a_2\pmod{m_2}, \\
\qquad \vdots \\
x\equiv a_n\pmod{m_n}, \\
\end{cases}
$$

其中 $m_1,m_2,\ldots,m_n$ 两两互质。

令 $\displaystyle M = \prod_{i=1}^{n}m_i, \quad M_i=\frac M{m_i}$，

因为 $m_i$ 与其他模数两两互质，所以 $\gcd(M_i,m_i)=1$，

因此 $M_i$ 在模 $m_i$ 下存在逆元 $t_i$：$M_i t_i \equiv 1 \pmod{m_i}$；

定义 $e_i=M_i t_i$，则 $e_i\equiv1\pmod{m_i}$，而对 $j\ne i$，由于 $m_j\mid M_i$，又有 $e_i\equiv0\pmod{m_j}$，

因此 $x\equiv\sum_{i=1}^{n}a_iM_it_i\pmod M$ 同时满足所有方程。

该方程组的解在模 $M$ 意义下唯一。


{% fold info @中国剩余定理 %}
```cpp
// 中国剩余定理：x ≡ a[i] (mod m[i])，模数两两互质
ll a[maxn], m[maxn];

ll mul( ll a, ll b, ll mod ) { 
    return (ll)( (i128)a * b % mod ); 
}

ll exgcd( ll a, ll b, ll &x, ll &y ) {
    if( b == 0 ) { x = 1, y = 0; return a; }
    ll d = exgcd( b, a % b, y, x );
    y -= a / b * x;
    return d;
}

ll crt( int N ) {
    ll M = 1, res = 0;
    for( int i = 1; i <= N; ++ i ) M *= m[i];
    for( int i = 1; i <= N; ++ i ) {
        ll Mi = M / m[i], ti, y;
        exgcd( Mi, m[i], ti, y );
        ti = ( ti % m[i] + m[i] ) % m[i];
        ll term = mul( a[i], Mi, M );
        term = mul( term, ti, M );
        res = ( res + term ) % M;
    }
    return ( res + M ) % M;
}
```
{% endfold %}

时间主要消耗在 $n$ 次扩展欧几里得运算上。

## 7 扩展中国剩余定理

当模数不两两互质时，CRT 的直接构造失效，因为 $M_i$ 未必在模 $m_i$ 下存在逆元。

更一般的方法是不断合并两个同余方程。

假设当前已经得到 $x\equiv r\pmod M$，现在需要加入 $x\equiv a\pmod m$。

由第一个方程，可以写成 $x = r + Mk$，代入第二个方程：$Mk \equiv a - r \pmod m$

于是合并问题重新转化为一个线性同余方程。设 $d=\gcd(M,m)$：

- 若 $d\nmid(a-r)$，两个条件冲突，无解；
- 若有解，求出一组 $k_0$，新余数为 $r+Mk_0$；
- 新模数为 $\operatorname{lcm}(M,m)=Mm/d$。

因此，ExCRT 的本质是反复求解线性同余方程。

{% fold info @扩展中国剩余定理 %}
```cpp
// 扩展中国剩余定理：模数不要求互质
ll a[maxn], m[maxn];

ll mul( ll a, ll b, ll mod ) { 
    return (ll)( (i128)a * b % mod ); 
}

ll exgcd( ll a, ll b, ll &x, ll &y ) {
    if( b == 0 ) { x = 1, y = 0; return a; }
    ll d = exgcd( b, a % b, y, x );
    y -= a / b * x;
    return d;
}

ll excrt( int N ) {
    ll M = m[1], res = a[1];
    for( int i = 2; i <= N; ++ i ) {
        ll x, y, A = M, B = m[i], C = ( a[i] - res % B + B ) % B;
        ll d = exgcd( A, B, x, y );
        if( C % d != 0 ) return -1;
        x = mul( x, C / d, B / d );
        res += x * M, M *= B / d;
        res = ( res % M + M ) % M;
    }
    return res;
}
```
{% endfold %}
