## 3. 指数的同余规律

### 3.1 剩余系、乘法群与欧拉函数

#### 3.1.1 剩余系

**完全剩余系**：从模 $m$ 的每个剩余类中各取一个数构成的集合，通常取 $\{0,1,2,\dots,m-1\}$。

**既约剩余系**：在完全剩余系中，只保留与 $m$ 互质的数所构成的子集。

> 例如 $m = 10$ 的既约剩余系为 $\{ 1,3,7,9 \}$。

#### 3.1.2 模 $m$ 乘法群

这是 **抽象代数中群论** 的概念，既约剩余系中的元素在 “模 $m$ 乘法” 下是封闭的，且满足结合律、存在单位元 (1) 和逆元。

> 封闭表示任意两个元素相乘，结果仍然在这个集合中。

这就构成了一个乘法群，称为模 $m$ 乘法群，常记为 $Z_{m}^{*}$。这个群的**阶**即为欧拉函数 $\varphi(m)$，后续的指数规律本质上都是在研究这个群的结构。

#### 3.1.3 欧拉函数

**欧拉函数定义**：$\varphi(m)$ 表示在 $1 \sim m$ 中，与 $m$ 互质的整数的个数。也就是模 $m$ 乘法群的大小( 阶 )。

**欧拉函数性质**：

1. 若 $p$ 为素数，则 $\varphi(p) = p - 1$。
2. 若 $p$ 为素数，则 $\varphi(p^k) = p^k - p^{k-1} = p^{k-1}(p-1)$。
3. **欧拉反演公式**：$n = \sum_{d \mid n} \varphi(d)$，即 $n$ 的所有因子的欧拉函数之和等于 $n$ 本身。

**计算公式**：若 $n$ 的唯一分解式为 $n = \prod p_{i}^{a_{i}}$，则由性质 1 和性质 2 可推导：

$$
\phi(n) = \prod p_{i}^{a_{i}-1} (p_{i} - 1) = n \prod (\frac{p_{i}-1}{p_{i}})
$$

**积性证明**：

- 考虑使用中国剩余定理，对于任意 $x \in [1, nm]$，它与一个数对 $(x \bmod n, x \bmod m)$ 构成双射。
- 根据裴蜀定理 $gcd(x, nm) = 1 \iff gcd(x, n) = 1$ 且 $gcd(x, m) = 1$。
- 因此满足 $gcd(x, nm) = 1$ 的 $x$ 的个数，等于满足 $gcd(a, n) = 1$ 的 $a$ 的个数，乘以满足 $gcd(b, m) = 1$ 的 $b$ 的个数。
- 即 $\varphi(nm) = \varphi(n) \varphi(m)$，证毕。

### 3.2 欧拉定理与降幂公式

#### 3.2.1 欧拉定理

**欧拉定理定义**：若 $gcd(a, m) = 1$，则 $a^{\varphi(m)} \equiv 1 \pmod m$。

**证明**：

- 需要使用既约剩余系，设 $S = \{x_{1},x_{2},\dots,x_{\varphi(m)}\}$。
- 因为 $gcd(a, m) = 1$，所以集合 $aS = \{ax_{1}, ax_{2}, \dots \}$ 取模后仍等于 $S$ ( 封闭且无重复 )。
- 将两个集合的元素分别相乘，消去 $\prod x_{i}$ 即可得到 $a^{\varphi(m)} \equiv 1 \pmod m$，如下：

$$
\prod_{x \in S} x \equiv \prod_{x \in S} ax = a^{\varphi(m)} \prod_{x \in S}x \pmod m
$$

**费马小定理**：对于素数 $p$ ，有 $\varphi(p) = p-1$，因此费马小定理是欧拉定理的一个特例。

#### 3.2.2 扩展欧拉定理

在计算 $A^B \pmod m$ 时，底数 $A$ 可以直接取模缩小，但无法直接对指数 $B$ 取模，需要用到推广后的 **扩展欧拉定理** 对指数进行降幂。

**扩展欧拉定理**：当 $B \ge \varphi(M)$ 时，$A^B \equiv A^{(B \bmod \varphi(M)) + \varphi(M)} \pmod M$。

**降幂公式如下**：

$$
A^B = \begin{cases}
A^B & B < \varphi(m) \\
A^{(B \bmod \varphi(m)) + \varphi(m)} & B \geq \varphi(m)
\end{cases} \pmod m
$$

> 注：此公式不需要 $A, m$ 互质，是处理大指数取模的通用手段。

{% fold info @扩展欧拉定理 %}
```cpp
ll get_phi( ll n ) {
    ll res = n;
    for( ll i = 2; i * i <= n; ++ i ) {
        if( n % i == 0 ) {
            res = res / i * (i - 1);
            while( n % i == 0 ) n /= i;
        }
    }
    if( n > 1 ) res = res / n * (n - 1);
    return res;
}

// 读入字符串指数的同时，利用 phi(m) 降幂。
ll depow( int phi ) {
    ll b = 0;
    bool flag = 0;
    for( char ch : sb ) {
        b = b * 10 + (ch - '0');
        if( b >= phi ) flag = 1, b %= phi;
    }
    if( flag ) b += phi;
    return b;
}

ll inv_euler( string sa, string sb ) {
	ll a = 0;
	for( char c : sa ) a = ( a * 10 + ( c - '0' ) ) % mod;
	ll phi = get_phi( mod );
	ll b = depow( phi, sb );
	return qpow( a, b ) % mod;
}
```

### 3.3 阶与原根

#### 3.3.1 阶

对于 $n \in Z$，幂次 $a^n \bmod m$ 呈现一种循环结构，这个循环节的最小长度，就是 $a$ 模 $m$ 的阶。 

**阶的定义**：幂 $a^n \bmod m$ 第一次回到起点 $a^{0} \bmod m = 1$ 时的指数。

**数学定义**：若 $a \perp m$，使得 $a^{x} \equiv 1 \pmod{m}$ 成立的最小正整数 $x$，就称为 $a$ 模 $m$ 的阶，记作 $\delta_{m}(a)$。

**阶的性质**：$a \perp m$ 前置条件不再赘述。

1. **周期性**：幂次 $a^0, a^1, \cdots, a^{\delta_{m}(a)-1}$ 模 $m$ 两两不同余。

2. **整除性**：若 $a^n \equiv 1 \pmod{m}$，这个 $n$ 一定是 $\delta_{m}(a)$ 的倍数，即 $\delta_{m}(a) \mid n$。

3. 结合欧拉定理和性质 2，有 $\delta_{m}(a) \mid \varphi(m)$。

> 欧拉定理回顾 $a^{\varphi(m)} \equiv 1 \pmod{m}$

#### 3.3.2 原根

既然每个数都有自己的阶，并且阶最大不超过 $\varphi(m)$，那么存不存在一个数 $g$，它的阶刚好等于 $\varphi(m)$ 呢？

**原根的定义**：若 $g \perp m$ 且 $\delta_{m}(g) = \varphi(m)$，则称 $g$ 为模 $m$ 的原根。

**群论**：原根 $g$ 本质上是模 $m$ 乘法群的**生成元**。

- 若 $g$ 是模 $m$ 的原根，意味着它乘方 $\varphi(m)$ 次才回到起点，刚好对应整个既约剩余系。这也就是 **生成元** 的含义。一旦有了原根，所有的 *模乘法*，都可以被转化为原根指数上的 *模加法*。( NTT 前置 )

#### 3.3.3 原根的判定与求解

**原根存在定理**：模 $m$ 有原根，当且仅当 $m \in \{1, 2, 4, p^k, 2p^k\}$，其中 $p$ 为奇素数。

**原根判定定理**：对于整数 $m \geq 3$ 和 $g \perp m$，那么 $g$ 是模 $m$ 的原根，当且仅当对于 $\varphi(m)$ 的每个素因数 $p$ 都有 $g^{\frac{\varphi(m)}{p}} \not\equiv 1 \pmod{m}$。

**原根个数**：若模 $m$ 存在原根，当且仅当 $d \mid \varphi(m)$ 时，模 $m$ 的 $d$ 阶元素存在，且恰好有 $\varphi(m)$ 个，特别的，模 $m$ 的原根的个数为 $\varphi(\varphi(m))$ 个。

**求解最小原根算法**：利用阶整除 $\varphi(m)$ 的性质 ( $\delta_m(g) \mid \varphi(m)$ )。

1. 若 $g$ 不是原根，它的阶一定小于 $\varphi(m)$，并且是 $\varphi(m)$ 的某个约数。
2. 我们只需要对 $\varphi(m)$ 进行质因数分解：$\varphi(m) = p_1^{a_1} p_2^{a_2} \dots p_k^{a_k}$。
3. 从小到大枚举 $g \in [2, m-1]$。如果对于所有的质因子 $p_i$，都满足 $g^{\frac{\varphi(m)}{p_i}} \not\equiv 1 \pmod m$，那么 $g$ 就是原根。

**最小原根生成所有原根**：如果 $g_{0}$ 的周期是 $\varphi(m)$，那么 $(g_{0}^{k})$ 的周期就是 $\frac{\varphi(m)}{\gcd(k, \varphi(m))}$。

- 因此只需要枚举与 $\varphi(m)$ 互质的指数 $k$，计算 $g_{0}^{k} \pmod m$ 就能找到所有的原根。

{% fold info @原根综合求解 %}
```cpp
// 原根存在性判定
bool has_root( ll n ) {
    if( n == 2 || n == 4 ) return 1;
    if( n % 2 == 0 ) n /= 2;
    if( n % 2 == 0 || n <= 1 ) return 0;
    for( int j = 1; j <= pcnt && primes[j] <= n; ++ j ) {
        ll p = primes[j];
        if( n % p == 0 ) {
            while( n % p == 0 ) n /= p;
            return n == 1;
        }
    }
    return n > 1;
}
// 找到最小的原根
ll get_min_root( ll n ) {
	if( !has_root( n ) ) return -1;
	if( n == 2 || n == 4 ) return n - 1;
	ll phim = Factor::get_phi( n );
	ll tmp = phim;
	vector<ll> facs = divide( phim );
	for( ll g = 2; g < n; ++ g ) {
		if( gcd( g, n ) != 1 ) continue;
		bool ok = 1;
		for( ll pi : facs ) {
			if( qpow( g, phim / pi, n ) == 1 ) { ok = 0; break; }
		}
		if( ok ) return g;
	}
	return -1;
}
// 获取所有原根
vector<ll> get_all_roots( ll n ) {
	ll g0 = get_min_root( n );
	if( g0 == -1 ) return { };
	ll phim = Factor::get_phi( n );
	vector<ll> roots;
	ll cur = 1;
	for( int k = 1; k <= phim; ++ k ) {
		cur = cur * g0 % n;
		if( gcd( k, phim ) == 1 ) roots.push_back( cur );
	}
	sort( roots.begin( ), roots.end( ) );
	return roots;
}
```

### 3.4 离散对数 BSGS

#### 3.4.1 离散对数

在实数域中，已知底数 $a$ 和结果 $b$，求未知指数 $x$ 的运算称为对数：$a^x = b \implies x = \log_{a}{b}$，而在模运算中，类似的方程就变成了 $A^{x} \equiv B \pmod{M}$。

在 3.3.2 原根 中提到，如果 $g$ 是模 $M$ 的原根，那么对于任意与 $M$ 互质的数 $B$，必定存在唯一的整数 $x \in [0, \varphi(M)-1]$，使得：$g^x \equiv B \pmod{M}$。

这个唯一的指数 $x$，就被称为以原根 $g$ 为底，$B$ 的 **离散对数**，记作 $ind_{g}(B)$。

类似实数对数，离散对数也有类似的性质：

$$
ind_{g}(A \times B) \equiv ind_{g}(A) + ind_{g}(B) \pmod{\varphi(M)}
$$

这也就是前面所提到的，离散对数将模意义下的乘法系统，映射到了关于模 $\varphi(M)$ 的加法系统。

#### 3.4.2 BSGS

面对任意给定的方程 $A^{x} \equiv B \pmod{M}$，如何求解 $x$？

**工具**：Baby-Step Giant-Step ( 大步小步算法 / 北上广深算法 )。

**核心**：折半查找思想。

设步长 $t = \lceil \sqrt{M} \rceil$，那么 $x = i \cdot t - j$，其中 $i \in [1,t]$ 大步，$j \in [0, t-1]$ 小步。

将拆分后的 $x$ 代入原方程：

$$
\begin{align}
A^{i \cdot t - j} \equiv B \pmod{M} \\
A^{i \cdot t} \equiv B \cdot A^{j} \pmod{M} \\
\end{align}
$$

> 其中 $A \perp M$，因为最后一步两边除以 $A^{j}$ 得到 $A^{i \cdot t - j} \equiv B \pmod{M}$，所以要求 $A \perp M$。

**算法流程**：

1. **小步建表**：枚举 $j \in [0, t-1]$，计算出所有的右半边，将结果作为 Key，$j$ 作为 Value 存入哈希表，如果 Key 相同，保留最大的 $j$，为保证结果尽可能小。

2. **大步查表**：枚举 $i \in [1,t]$，计算出所有的左半边，去查哈希表，如果找到了匹配的值，那么当前的 $x = i \cdot t - j$ 就是要求的最小非负整数解。

{% fold info @BSGS %}
```cpp
// 求解 A^x ≡ B (mod M)
ll BSGS( ll A, ll B, ll M ) {
    A %= M; B %= M;
    if( B == 1 ) return 0;
    ll t = ceil( sqrt( M ) );
    unordered_map<ll, ll> mp; // 用哈希表存 (值, j)
    // 小步建表 - 右半边
    ll cur = B;
    for( ll j = 0; j < t; ++ j ) {
        mp[cur] = j; // 若有重复，会自动覆盖，保留最大的 j
        cur = cur * A % M;
    }
    // 大步查表 - 左半边
    ll at = qpow( A, t, M );
    cur = at;
    for( ll i = 1; i <= t; ++ i ) {
        if( mp.count( cur ) ) {
            return i * t - mp[cur]; // 找到一组解
        }
        cur = cur * at % M;
    }
    return -1; // 查无此解
}
```

#### 3.4.3 ExBSGS 

若 $a \not\perp p$，那么在解决 $a^{x} \equiv b \pmod{p}$ 过程中，设 $d = \gcd(a, p)$。

1. 判定：如果 $d > 1$，但 $b \not\mid d$，则方程无解。
2. 反之如果能被整除，则将方程两边同时除以 $d$，得到下式：

$$
\frac{a}{d} \cdot a^{x-1} \equiv \frac{b}{d} \pmod{ \frac{p}{d} }
$$

3. 循环剔除：重复上述操作，记录一共除以了 $k$ 次公因数，并且把前面提取出来的系数累乘记作 $A$，直到 $\gcd(a, p) = 1$。

$$
A \cdot a^{x-k} \equiv B \pmod{M} 
$$

> 其中 $M$ 和 $B$ 均为最终化简后的值。

4. BSGS：此时 $a$ 和 $M$ 互质，可以直接求解 $A \cdot a^{x'} \equiv B \pmod{M}$，最终答案为 $x'+ k$。 

{% fold info @ExBSGS %}
```cpp
// A^x ≡ B (mod M)
ll exBSGS( ll A, ll B, ll M ) {
    A %= M, B %= M;
    if( B == 1 || M == 1 ) return 0;
    if( A == 0 ) return B == 0 ? 1 : -1;
    ll C = 1, k = 0, d;
    // 循环消除因子直到 A 和 M 互质
    while( ( d = gcd( A, M ) ) > 1 ) {
        if( B % d != 0 ) return -1; // 无法整除 一定无解
        k ++, B /= d, M /= d;
        C = C * ( A / d ) % M;
        if( C == B ) return k; // 提前匹配到了答案
    }
	// 后续和 BSGS 一样，只是左边多了个常数 C
    ll t = ceil( sqrt( M ) );
    unordered_map<ll, ll> mp;
    ll cur = B;
    for( ll j = 0; j < t; ++ j ) {
        mp[cur] = j;
        cur = cur * A % M;
    }
    ll at = qpow( A, t, M );
    cur = C * at % M;
    for( ll i = 1; i <= t; ++ i ) {
        if( mp.count( cur ) ) return i * t - mp[cur] + k;
        cur = cur * at % M;
    }
    return -1;
}
```

### 3.5 二次剩余与 Cipolla 算法

**问题**：求解 $x^{2} \equiv N \pmod{p}$，( $p$ 为奇素数 )。若存在解，则称 $N$ 是模 $p$ 的二次剩余。

#### 3.5.1 欧拉判别准则与勒让德符号

利用勒让德符号 $\left(\frac{N}{p}\right)$ 判定解是否存在，根据欧拉判别准则：

$$
\left( \frac{N}{p} \right) \equiv N^{\frac{p-1}{2}} \pmod{p}
$$

- $N^{\frac{p-1}{2}} \equiv 1 \pmod{p}$ 表示 $N$ 为二次剩余。
- $N^{\frac{p-1}{2}}\equiv -1 \pmod{p}$ 表示 $N$ 非二次剩余（无解）。

#### 3.5.2 Cipolla 算法

在判定有解之后，通过构造负数域求根。

1. 随机枚举一个 $a$，使得 $a^{2} - N$ 为 **非二次剩余**，期望两次找到。
2. 定义虚数单位 $i = \sqrt{a^{2} - N}$，建立复数域结构 $X + Y \cdot i$。
3. 通过代数推导（ 二项式定理，费马小定理 展开 ），方程的根为：

$$
x \equiv (a + i)^{\frac{p+1}{2}} \pmod{p}
$$

4. 另一个根为 $p - x$。

{% fold info @Cipolla %}
```cpp
ll w; // 虚数单位 i 的平方值: a^2 - n
struct Complex { ll r, i; };
Complex operator * ( const Complex &a, const Complex &b ) {
	ll rr = ( a.r * b.r % mod + a.i * b.i % mod * w % mod ) % mod;
	ll ii = ( a.r * b.i % mod + a.i * b.r % mod ) % mod;
	return Complex( rr, ii );
}
Complex qpow_C( Complex a, ll k, ll mod ) {
	Complex res( 1, 0 );
	for( ; k; k >>= 1, a = a * a % mod )
		if( k & 1 ) res = res * a % mod;
	return res;
}
bool check( ll n, ll mod ) {
	if( n == 0 ) return 1;
	return qpow( n, (mod - 1) / 2, mod ) == 1;
}
// 返回解的一对 pair。若无解返回 {-1,-1}
pair<ll, ll> solve( ll n, ll mod ) {
	n %= mod;
	if( n == 0 ) return {0, 0};
	if( !check( n, mod ) ) return {-1, -1};
	mt19937_64 rng( time(0) );
	ll a;
	while( 1 ) {
		a = rng( ) % mod;
		w = ( a * a % mod - n + mod ) % mod;
		if ( qpow(w, (mod - 1) / 2, mod ) == mod - 1 ) break;
	}
	Complex res = qpow_C( Complex( a, 1 ), (mod + 1) / 2, mod );
	ll x1 = res.r, x2 = mod - res.r;
	if( x1 > x2 ) swap( x1, x2 );
	return {x1, x2};
}
void solve( ) {
    ll n;
    cin >> n >> mod;
    auto [x1, x2] = cipolla( n );
    if( x1 == -1 ) cout << "No Solution" << '\n';
    else if( x1 == x2 ) cout << x1 << '\n';
    else cout << x1 << " " << x2 << '\n';
}
```
