## 2. 模意义下的运算

### 2.1 裴蜀定理

这是一切线性同余方程的起点。

**裴蜀定理**：对于任意非零整数 $a, b$，关于 $x, y$ 的线性不定方程 $ax + by = c$ 有整数解的 **充要条件** 是：$c$ 是 $\gcd(a, b)$ 的倍数。

这意味着，无论 $x, y$ 怎么取，组合出来的数一定能被 $\gcd(a, b)$ 整除。

**推广**：多个数也是同理，$a_1 x_1 + a_2 x_2 + \dots + a_n x_n = c$ 有解当且仅当 $c$ 是 $\gcd(a_1, a_2, \dots, a_n)$ 的倍数。

既然 $ax + by = gcd(a, b)$ 必然有解，这个解应该怎么算出来呢？

### 2.2 扩展欧几里得 Exgcd

普通的欧几里得算法利用 $\gcd(a, b) = \gcd(b, a \bmod b)$ 递归求出最大公约数。

**Exgcd** 则是在递归返回的过程中，同时把 $ax + by = \gcd(a, b)$ 的一组特解 $(x, y)$ 给求出来。

**简要推导**：

- 设 $ax + by = gcd(a, b), bx'+(a \bmod b)y' = gcd(b, a \bmod b)$，

- 因为 $gcd(a, b) = gcd(b, a \bmod b)$，所以 $ax + by = bx' + (a \bmod b)y'$。

- 整理得 $ax + by = ay' + b(x' - \lfloor a / b \rfloor y')$，

- 因为 $a = a, b = b$，所以 $x = y', y = x' - \lfloor a / b \rfloor y'$。

{% fold info @exgcd %}
```cpp
ll exgcd( ll a, ll b, ll &x, ll &y ) {
    if( b == 0 ) { x = 1, y = 0; return a; }
    // 注：这里传参 x, y 位置互换了，使得递归出来的就是新 x 和旧 y
    ll d = exgcd( b, a % b, y, x );
    y -= a / b * x; // 根据推导公式修正 y
    return d; // 返回最大公约数
}
```
{% endfold %}

### 2.2.1 一次不定方程

对于方程 $a_{1}x_{1} + a_{2}x_{2} = b$ 来说，裴蜀定理指出该方程有解，当且仅当 $d = gcd( a_{1}, a_{2}) \mid b$。

假设这一条件成立，利用 Exgcd 求出 $a_{1}x_{1} + a_{2}x_{2} = d$ 的一组整数解 $(x_{1}^{*},x_{2}^{*})$。

由此可以得到原方程的一组特解 $x_{1}^{o}, x_{2}^{o} = \left( \frac{b}{d}x_{1}^{*}, \frac{b}{d} x_{2}^{*} \right)$。

要得到全部解，可以考虑将原方程与恒等式 $a_{1}x_{1}^{o} + a_{2}x_{2}^{o} = b$ 相减，就有：

$$
a_{1}(x_{1}-x_{1}^{o}) + a_{2}(x_{2}-x_{2}^{o}) = 0
$$

这是一个关于 $(x_{1}-x_{1}^{o}, x_{2}-x_{2}^{o})$ 的其次一次不定方程，它有通解：

$$
(x_{1} - x_{1}^o, x_{2} - x_{2}^o) = (t \frac{a_{2}}{d}, -t \frac{a_{1}}{d} ), (t \in \mathbb{Z})
$$

因此原方程的通解就是：

$$
\begin{align}
(x_{1}, x_{2}) = \left(  x_{1}^{o} + t \frac{a_{2}}{d}, x_{2}^{o} - t \frac{a_{1}}{d} \right), (t \in \mathbb{Z}) \\
(x_{1}, x_{2}) = \left( \frac{b}{d} x_{1}^{*} + t \frac{a_{2}}{d}, \frac{b}{d}x_{2}^{*} - t \frac{a_{1}}{d} \right), (t \in \mathbb{Z})
\end{align}
$$

这其实是直线 $a_{1}x_{1} + a_{2}x_{2} = b$ 上一系列等间隔分布的整点。

### 2.3 乘法逆元

在模 $p$ 意义下，$+, -, \times$ 都可以直接取模，只有 $\div$ 不行，即 

$$
(A / B) \bmod p \neq (A \bmod p) / (B \bmod p)
$$

为了在模意义下处理 **除法运算**，我们需要找到 $B$ 的乘法逆元 $inv(B)$，使得 $B \times inv(B) \equiv 1 \pmod p$，满足这个条件后，除以 $B$ 就等于 乘以 $inv(B)$。

#### 2.3.1 费马小定理

**费马小定理**：若 $p$ 是质数，且 $a$ 不是 $p$ 的倍数，有 $a^{p-1} \equiv 1 \pmod p$。

> 条件：$p$ 必须为质数

左右两边同时乘以一个 $a^{-1}$ 可以得到 $a^{-1} \equiv a^{p-2} \pmod p$，由于 $p$ 比较大，所以需要采用 $O(\log p)$ 的快速幂求解。

{% fold info @逆元 - 费马小定理 %}
```cpp
ll qpow( ll a, ll k, ll mod ) {
	ll res = 1;
	for( ; k; k >>= 1, a = (a * a) % mod ) 
		if( k & 1 ) res = (res * a) % mod;
	return res;
}

ll inv_fermat( ll a, ll mod ) {
	return qpow( a, mod - 2, mod );
}
```
{% endfold %}

#### 2.3.2 Exgcd 求逆元

要求 $ax \equiv 1 \pmod p$，其实就等价于求不定方程 $ax + py = 1$，根据裴蜀定理，只要 $gcd(a, p) = 1$，就必定有解，即存在逆元。

> 条件： $a \perp p$，即 $a$ 与 $p$ 互质

{% fold info @逆元 - Exgcd %}
```cpp
ll inv_exgcd( ll a, ll b ) {
	ll x, y;
	ll d = exgcd( a, b, x, y );
	if( d != 1 ) return -1; // 不互质，无逆元
	return ( x % b + b ) % b; // 防止负数，转正处理
}
```
{% endfold %}

#### 2.3.3 线性递推求逆元

如果需要用到巨量组合数，我们需要求解 $1 \sim n$ 所有数字的逆元，单次 $O(\log p)$ 会超时，我们需要采用线性递推的方式求解，下面简要推导：

> 要求：$p$ 必须为质数

设 $p = k \times i + r$，$k \times i + r \equiv 0 \pmod p \implies r \equiv -ki \pmod p$。
两边同乘 $i^{-1} \cdot r^{-1}$ 得到：$i^{-1} \equiv -k \cdot r^{-1} \pmod p$。

> $k = \lfloor p / i \rfloor, r = p \bmod i$。

**结论**：$inv[i] = (p - p/i) \cdot inv[p \bmod i] \pmod p$。

{% fold info @逆元 - 线性递推 %}
```cpp
void linear_inv( int n, ll mod ) {
    inv[1] = 1;
    for( int i = 2; i <= n; ++ i ) {
        inv[i] = ( mod - mod / i ) * inv[mod % i] % mod;
    }
}
```
{% endfold %}

#### 2.3.4 前缀积求逆元

如果给定一个无规律的数组 $a[1 \dots n]$，要求每个数的逆元。

**思路**：先求出前缀积 `pre[i]`，对总乘积 `pre[n]` 求一次逆元（记作 `ipre[n]`）。然后逆推：`ipre[i-1] = ipre[i] * a[i]`，那么 $a[i]$ 的逆元就等于 `pre[i-1] * ipre[i]`。其实同理 2.2.3 也可以用同样的方法逆推算出来。

{% fold info @逆元 - 前缀积逆推 %}
```cpp
void array_inv( int n, ll mod ) {
    pre[0] = 1;
    for( int i = 1; i <= n; ++ i ) pre[i] = pre[i-1] * a[i] % mod;
    ipre[n] = qpow( pre[n], mod - 2 ); // 仅做一次快速幂
    // 后缀推前缀逆元
    for( int i = n - 1; i >= 1; -- i ) ipre[i] = ipre[i+1] * a[i+1] % mod; 
	// 此时 a[i] 的逆元即为 pre[i-1] * ipre[i] % mod
}
```
{% endfold %}

### 2.4 线性同余方程

形如 $ax \equiv c \pmod p$ 的方程，被称为一元线性同余方程，可以直接转化为二元一次不定方程 $ax + py = c$。

一般思路：

1. 先求出 $ax_{0} + py_{0} = gcd(a, p)$ 的特解 $x_{0}$。
2. 若 $c$ 不能被 $gcd(a, p)$ 整除，则无解。
3. 否则，两边同乘 $\frac{c}{gcd(a,p)}$，方程特解为 $X = x_{0} \times \frac{c}{gcd(a, p)}$。
4. 记 $d = \frac{c}{gca(a, p)}$，最小整数解一般通过 `X = ( X % d + d ) % d` 获取。

### 2.5 中国剩余定理 CRT

当遇到线性同余方程组时，例如 “韩信点兵” 问题：

$$
\begin{cases}
x \equiv a_1 \pmod{m_1} \\
x \equiv a_2 \pmod{m_2} \\
\dots \\
x \equiv a_n \pmod{m_n}  \\
\end{cases}
$$

> **条件**：所有的模数 $m_i$ **两两互质**。

**构造思路**：

1. 计算所有模数的乘积 $M = \prod m_i$。
2. 对于第 $i$ 个方程，计算 $M_i = \frac{M}{m_i}$。
3. 因为互质，必定存在 $t_i$ 使得 $M_i \cdot t_i \equiv 1 \pmod{m_i}$。（即用 Exgcd 求 $M_i$ 在模 $m_i$ 下的逆元）。
4. 构造特解：$ans = \sum_{i=1}^{n} a_i M_i t_i \pmod M$。

{% fold info @CRT %}
```cpp
ll a[maxn], m[maxn];
ll crt( int cnt ) {
    ll M = 1, ans = 0;
    for( int i = 1; i <= cnt; ++ i ) M *= m[i];
    for( int i = 1; i <= cnt; ++ i ) {
        ll Mi = M / m[i];
        ll ti, y;
        exgcd( Mi, m[i], ti, y ); // 找逆元
        ti = ( ti % m[i] + m[i] ) % m[i];
        ll term = mul( a[i], Mi, M );
        term = mul( term, ti, M );
        ans = ( ans + term ) % M; // 累加构造解
    }
    return ( ans + M ) % M;
}
```
{% endfold %}

这也是 Exlucas 定理的最后一步推导，把大组合数拆成一堆模质数次幂的子问题后，用 CRT 将它们组合起来。

### 2.6 扩展中国剩余定理 (ExCRT)

**问题**：如果方程组中，模数 $m_i$ **不互质**怎么办？此时 $M_i$ 和 $m_i$ 不互质，逆元 $t_i$ 根本不存在，CRT 的构造法就失效，所以需要回归本质，用 Exgcd 不断合并两个方程。

> 说是 ExCRT 但实际上跟 CRT 关系不大，它的本质是 2.2 ExGCD 求解线性方程的 $n$ 次复合。

**推导**：

- 设两个方程分别是 $x \equiv a_{1} \pmod{m_{1}}, x \equiv a_{2} \pmod{m_{2}}$，

- 把它们转化为不定方程：$x = m_{1}p + a_{1} = m_{2}q + a_{2}$，其中 $p, q$ 是整数，则有 $m_{1}p - m_{2}q = a_{2} - a_{1}$。

- 由裴蜀定理，当 $(a_{2}-a_{1})$ 不能被 $gcd(m_{1}, m_{2})$ 整除时，无解。其他情况下，都可以用 Exgcd 解出一组可行解 $(p, q)$。

- 则原来的两方程组成的模方程组的解为 $x \equiv b \pmod M$，其中 $b = m_{1}p + a_{1}, M = lcm(m_{1}, m_{2})$。

对于更多的方程组，用上述方法两两合并即可。

{% fold info @ExCRT %}
```cpp
ll excrt( ll n ) {
    ll M = m[1], res = a[1];
    for( int i = 2; i <= n; ++ i ) {
        ll x, y;
        ll A = M, B = m[i], C = ( a[i] - res % B + B ) % B;
        ll g = exgcd( A, B, x, y );
        if( C % g != 0 ) return -1; // 无解
        x = mul( x, C / g, B / g );
        res += x * M;
        M *= ( B / g );
        res = ( res % M + M ) % M;
    }
    return res;
}
```
