## 4. 组合与求和

#### 4.1 数论函数与狄利克雷卷积

前三章我们对 "数字" 做乘法，这一章我们对 “函数” 做乘法。

#### 4.1.1 常见数论函数

**数论函数的定义**：定义域为正整数的函数。

若满足 $\gcd(a, b) = 1 \implies f(ab) = f(a)f(b)$，则称为 **积性函数**。

下面是三个常用的 完全积性函数 ( 即使不互质也满足拆分 )：

1. 单位函数 $\epsilon$：$\epsilon(n) = [n==1]$ 当且仅当 $n=1$ 时值为 1，其余为 0。
2. 常数函数 $1$：$1(n) = 1$ 恒等于 1。
3. 恒等函数 $id$：$id(n) = n$。

#### 4.1.2 狄利克雷卷积

**定义**：对于两个数论函数 $f$ 和 $g$，它们的狄利克雷卷积 $f * g$ 是一个新的数论函数，其定义为：

$$
(f * g)(n) = \sum_{d \mid n} f(d)g( \frac{n}{d} )
$$

> 枚举 $n$ 的所有因子 $d$，将两部分函数值相乘再求和。

**卷积的群论意义**：

- 交换律，结合律，分配律。
- **单位元**：这里就是单位函数 $\epsilon$，即 $f * \epsilon = f$。
- **逆元**：若 $f * g = \epsilon$，则称 $g$ 为 $f$ 的狄利克雷逆元。

#### 4.1.3 欧拉与莫比乌斯函数

1. **莫比乌斯函数**：它是 常数函数 $1$ 的逆元。

$$
\mu * 1 = \epsilon \iff \sum_{d \mid n} \mu(d) = [n = 1]
$$
2. **欧拉函数**：它卷 常数函数 1，刚好等于 恒等函数 $id$。

$$
\varphi * 1 = id \iff \sum_{d \mid n} = n
$$

> 注：这个 $\mu * 1 = \epsilon$ 的性质，可以把复杂的限制条件（比如互质）给强行替换，展开。

### 4.2 整除分块 / 数论分块

在推导完毕后，我们常常会遇到形如 $\sum_{i=1}^{n} \left\lfloor  \frac{n}{i}  \right\rfloor$ 的连加算式，如果 $n = 10^{12}$，直接相加必然超时。

观察后发现，对于 $\left\lfloor \frac{n}{i} \right\rfloor$ 这样的柿子，虽然 $i$ 有 $n$ 个，但它不同的取值最多只有 $2 \sqrt{ n }$ 种，并且相同的值是 **连续成块** 分布的。

对于当前处于左边界 $l$ 的块，它的右边界必定为：

$$
r = \left\lfloor  \frac{n}{ \lfloor n / l \rfloor }  \right\rfloor 
$$

如果有二维限制（ 例如 $\lfloor \frac{n}{i} \rfloor$，$\lfloor \frac{m}{i} \rfloor$ ），那么块的右边界就是两者的较小值。

#### 4.2.1 一维整除分块

**经典形式**：计算 $\sum_{i=1}^{n}f(i, \lfloor k / i \rfloor)$，其中 $k$ 是某个常数。

{% fold info @一维整除分块 %}
```cpp
ll sum = 0;
for( ll l = 1, r; l <= n; l = r + 1 ) {
    ll val = k / l;
    if (val == 0) r = n; // 之后 val 全为 0，直接跳到末尾
    else r = min( k / val, n ); // 保证 k / r == val 且 r <= n
    // 当前区间 [l, r] 内 ⌊k/i⌋ 恒为 val
    sum += f( l, r, val );
}
```
{% endfold %}

1. 计算 $\sum_{i=1}^{n} \lfloor k / i \rfloor$：

{% fold info @一维整除分块-1 %}
```cpp
ll ans = 0;
for( ll l = 1, r; l <= n; l = r + 1 ) {
    ll val = k / l;
    if( val == 0 ) r = n;
    else r = min( k / val, n );
    ans += (r - l + 1) * val;
}
```
{% endfold %}

2. 计算 $\sum_{i=1}^{n} i \cdot \lfloor k / i \rfloor$：

{% fold info @一维整除分块-2 %}
```cpp
ll ans = 0;
for (ll l = 1, r; l <= n; l = r + 1) {
    ll val = k / l;
    if( val == 0 ) r = n;
    else r = min( k / val, n );
    ans += (l + r) * (r - l + 1) / 2 * val;   // 等差数列求和
}
```
{% endfold %}

#### 4.2.2 二维整除分块

二维分块常见于计算 $\sum_{i=1}^{\min(n, m)} \lfloor n/i \rfloor \cdot \lfloor m/i \rfloor$ 的形式（ 例如杜教筛，莫比乌斯反演中的前缀和优化 ）。

**原理**：$\lfloor n / i \rfloor$ 和 $\lfloor m / i \rfloor$ 同时 保持不变的最大区间是：

$$
r = \min\left(  \frac{n}{\lfloor n / l \rfloor }, \frac{m}{\lfloor m / l \rfloor } \right)
$$
{% fold info @二维整除分块 %}
```cpp
ll sum = 0;
for (ll l = 1, r; l <= min(n, m); l = r + 1) {
    ll vn = n / l, vm = m / l;
    if (vn == 0 || vm == 0) break;
    r = min( n / vn, m / vm );
    // 注意 r 不能超过 min(n, m)
    if( r > min(n, m) ) r = min(n, m);
    sum += (r - l + 1) * f(vn, vm);  // f 为当前块贡献
}
```
{% endfold %}

1. 计算 $\sum_{i=1}^{n} \lfloor n/i \rfloor \lfloor m/i \rfloor$：

{% fold info @二维整除分块-1 %}
```cpp
if( n > m ) swap(n, m); // 保证 n <= m
ll ans = 0;
for( ll l = 1, r; l <= n; l = r + 1 ) {
    ll vn = n / l, vm = m / l;
    if( vn == 0 || vm == 0 ) break;
    r = min( n / vn, m / vm );
    ans += (r - l + 1) * vn * vm;
}
cout << ans << '\n';
```
{% endfold %}

### 4.3 莫比乌斯反演

**反演定理**：若 $F = f * 1$，则 $f = F * \mu$。

> 从代数角度看，两边同时乘上一个 $\mu$ 就行，由于 $1$ 的逆元是 $\mu$，就可以得到结果。

#### 4.3.1 例子：求互质对数量

$$
\sum_{i=1}^{n} \sum_{j=1}^{m} [\gcd(i, j) = 1]
$$
1. **替换条件**：看到 $[\cdots = 1]$，立刻想到 $\epsilon$，利用 $\mu * 1 = \epsilon$，展开：

$$
= \sum_{i=1}^{n} \sum_{j=1}^{m} \sum_{d | \gcd(i, j)} \mu(d)
$$

2. **交换求和顺序**：既然 $d \mid \gcd(i, j)$，那么 $d \mid i, d \mid j$，可以把 $d$ 提到最外面，去枚举 $d$ 的倍数。

$$
= \sum_{d=1}^{n} \mu(d) \sum_{i=1, d \mid i}^{n} 1 \sum_{j=1, d \mid j}^{m} 1
$$

3. **化简倍数个数**：在 $n$ 的范围内，$d$ 的倍数显然有 $\lfloor \frac{n}{d} \rfloor$ 个。

$$
= \sum_{d=1}^{n} \mu(d) \lfloor \frac{n}{d} \rfloor \lfloor \frac{m}{d} \rfloor 
$$

4. **计算答案**（结合数论分块）。

### 4.4 杜教筛

当我们用到莫比乌斯反演的时候，必须预处理 $\mu$ 或者 $\varphi$ 的前缀和，而线性筛的 $O(n)$ 复杂度在遇到 $N = 10^{10}$ 时就会超时。

**引子**：如果能构造一个 $S(n)$ 关于 $S( \lfloor \frac{n}{i} \rfloor)$ 的递推式就好了。

已知狄利克雷卷积：

$$
(f * g)(i) = \sum_{d \mid i} f(d)g\left( \frac{i}{d} \right) = \sum_{d \mid i} g(d)f\left( \frac{i}{d} \right)
$$

所以我们有：

$$
\sum_{i=1}^{n}(f * g)(i) = \sum_{i=1}^{n} \sum_{d \mid i} g(d) f\left( \frac{i}{d} \right)
$$

1. 交换求和顺序：固定 $d$，然后考虑 $i$ 取遍 $d$ 的倍数，令 $i = d \cdot k$。

$$
\sum_{i=1}^{n} \sum_{d \mid i} g(d)f\left( \frac{i}{d} \right) = \sum_{d=1}^{n} \sum_{k=1}^{\lfloor n / d \rfloor } g(d)f(k)
$$

2. 分离 $g(d)$，并引入 $S$

$$
\sum_{d=1}^{n} g(d) \left(  \sum_{k=1}^{\lfloor n/d \rfloor }f(k) \right) = \sum_{d=1}^{n} g(d) S\left(  \left\lfloor  \frac{n}{d}  \right\rfloor  \right)
$$

3. 分离 $d = 1$ 的项

$$
\sum_{d=1}^{n} g(d) S\left( \left\lfloor  \frac{n}{d}  \right\rfloor  \right) = g(1)S\left( \left\lfloor  \frac{n}{1}  \right\rfloor  \right) + \sum_{d=2}^{n} g(d)S\left(  \left\lfloor  \frac{n}{d}  \right\rfloor   \right)
$$

4. 移项：

$$
\begin{align}
\sum_{i=1}^{n} (f * g)(i) = g(1)S(n) + \sum_{d=2}^{n} g(d)S\left( \left\lfloor \frac{n}{d} \right\rfloor \right) \\
g(1)S(n) = \sum_{i=1}^{n}(f * g)(i) - \sum_{d=2}^{n} g(d)S\left( \left\lfloor \frac{n}{d} \right\rfloor \right)
\end{align}
$$

#### 4.4.1 欧拉与莫比乌斯函数

1. 筛 $\mu$：令 $f = \mu, g = 1$，因为 $f * 1 = \epsilon$，其前缀和恒为 1。

$$
S_{\mu}(n) = 1 - \sum_{i=2}^{n} S_{\mu}\left(  \left\lfloor  \frac{n}{i}  \right\rfloor  \right)
$$

2. 筛 $\varphi$：令 $f = \varphi, g = 1$，因为 $\varphi * 1 = id$，其前缀和等于 $\frac{n(n+1)}{2}$。

$$
S_{\varphi}(n) = \frac{n(n+1)}{2} - \sum_{i=2}^{n} S_{\varphi}\left( \left\lfloor \frac{n}{i} \right\rfloor \right)
$$

利用数论分块递归，并使用 `unordered_map` 记忆化，结合线性筛预处理前 $N^{2/3}$ 个答案，总时间复杂度优化到 $O(N^{2/3})$。

{% fold info @杜教筛 %}
```cpp
const int maxn = 5e6 + 5;

ll primes[maxn >> 1], pcnt;
bool isp[maxn];

ll phi[maxn], mu[maxn];
ll sum_mu[maxn], sum_phi[maxn];
unordered_map<ll, ll> mp_mu, mp_phi;

// 调用线性筛 (略) - 预处理 欧拉和莫比乌斯函数 以及它们的前缀和

// 杜教筛求 mu 的前缀和
ll get_sum_mu( ll n ) {
    if( n <= maxn ) return sum_mu[n];
    if( mp_mu.count( n ) ) return mp_mu[n];
    ll res = 1;
    for( ll l = 2, r; l <= n; l = r + 1 ) {
        ll val = n / l;
        if( val == 0 ) r = n;
        else r = min( n / val, n );
        res -= ( r - l + 1 ) * get_sum_mu( val );
    }
    return mp_mu[n] = res;
}

// 杜教筛求 phi 的前缀和
ll get_sum_phi( ll n ) {
    if( n <= maxn ) return sum_phi[n];
    if( mp_phi.count( n ) ) return mp_phi[n];
    ll res = n * (n + 1) / 2;
    for( ll l = 2, r; l <= n; l = r + 1 ) {
        ll val = n / l;
        if( val == 0 ) r = n;
        else r = min( n / val, n );
        res -= ( r - l + 1 ) * get_sum_phi( val );
    }
    return mp_phi[n] = res;
}

void solve( ) {
    ll n;
    cin >> n;
    ll ans1 = get_sum_phi( n );
    ll ans2 = get_sum_mu( n );
    cout << ans1 << ' ' << ans2 << '\n';
}
```
{% endfold %}

### 4.5 组合数取模

当 $n, m \leq 10^{18}$ 且需要求 $\binom{n}{m} \pmod{p}$ 时，无法直接计算阶乘，需要通过 Lucas 定理拆解。

#### 4.5.1 Lucas 定理 ( 要求 $p$ 是质数 )

把大整数 $n, m$ 转化为 $p$ 进制，然后每一位独立求组合数阶乘：

$$
\binom{n}{m} \equiv \binom{n \bmod p}{m \bmod p} \times \binom{n / p}{m / p} \pmod{p}
$$

{% fold info @Lucas定理 %}
```cpp
// 预处理 1 ~ N 范围内的 阶乘 和 阶乘逆元
ll fac[maxn], inv[maxn];
void init( ll N, ll mod ) {
    fac[0] = 1;
    for( int i = 1; i <= N; ++ i ) fac[i] = fac[i - 1] * i % mod;
    inv[N] = qpow( fac[N], mod - 2, mod );
    for( int i = N - 1; i >= 0; -- i ) inv[i] = inv[i + 1] * (i + 1) % mod;
}
// 基础组合数计算
ll nCr( ll n, ll r, ll mod ) {
    if( n < r ) return 0;
    return fac[n] * inv[r] % mod * inv[n - r] % mod;
}
// 递归形式的 lucas
ll lucas( ll n, ll r, ll p ) {
    if( r == 0 ) return 1;
    return nCr( n % p, r % p, p ) * lucas( n / p, r / p, p ) % p;
}
```
{% endfold %}

#### 4.5.2 扩展 Lucas 定理 ( 不要求 $p$ 是质数 )

若 $p$ 不是素数，由于 $\gcd(m!, p) \neq 1$，阶乘不存在逆元，无法应用 Lucas 定理。

1. 拆分（算术基本定理）：将 $p$ 质因数分解为 $p = \prod p_{i}^{a_{i}}$。
2. 消去因子：对于求 $\binom{n}{m} \pmod{p_{i}^{a_{i}}}$，单独把阶乘里面的所有质因子 $p_{i}$ 剥离，此时剩余部分必定与模数互质，逆元存在。
3. 合并 CRT：由于各个 $p_{i}^{a_{i}}$ 两两互质，可以直接使用 CRT 合并答案。

{% fold info @ExLucas定理 %}
```cpp
ll mul( ll a, ll b, ll mod ); // 防溢出乘法
ll qpow( ll a, ll k, ll mod ); // 快速幂
ll exgcd( ll a, ll b, ll &x, ll &y ); // 扩展欧几里得算法
ll inv( ll a, ll mod ); // 求逆元

// 剥离 质因子 p 的阶乘计算
ll fac( ll n, ll p, ll pk ) {
    if( n == 0 ) return 1;
    ll res = 1;
    // 计算以 pk 为周期的非 p 倍数的乘积
    for( ll i = 1; i <= pk; ++ i ) {
        if( i % p != 0 ) res = res * i % pk;
    }
    res = qpow( res, n / pk, pk );
    // 计算剩余尾部的非 p 倍数乘积
    for( ll i = 1; i <= n % pk; ++ i ) {
        if( i % p != 0 ) res = res * i % pk;
    }
    // 递归处理提取出 p 后的阶乘部分
    return res * fac( n / p, p, pk ) % pk;
}
// 获取 n! 中含质因子 p 的个数
ll cntp( ll n, ll p ) {
    ll res = 0;
    while( n ) res += n / p, n /= p;
    return res;
}
// 计算单体 C(n, r) % p^k
ll nCr_pk( ll n, ll r, ll p, ll pk ) {
    if( n < r ) return 0;
    ll a = fac( n, p, pk ), b = fac( r, p, pk ), c = fac( n - r, p, pk );
    ll k = cntp( n, p ) - cntp( r, p ) - cntp( n - r, p );
    ll ans = a * inv( b, pk ) % pk * inv( c, pk ) % pk;
    return ans * qpow( p, k, pk ) % pk;
}

ll a[maxn], m[maxn];
ll crt( int cnt ); // 中国剩余定理 (略) 整合答案

ll exlucas( ll n, ll r, ll mod ) {
    ll tmp = mod;
    int cnt = 0;
    for( ll i = 2; i * i <= tmp; ++ i ) {
        if( tmp % i == 0 ) {
            ll pk = 1;
            while( tmp % i == 0 ) pk *= i, tmp /= i;
            cnt ++;
            m[cnt] = pk, a[cnt] = nCr_pk( n, r, i, pk );
        }
    }
    if( tmp > 1 ) cnt ++, m[cnt] = tmp, a[cnt] = nCr_pk( n, r, tmp, tmp );
    return crt( cnt );
}

void solve( ) {
    ll n, r, p;
    cin >> n >> r >> p;
    cout << exlucas( n, r, p ) << '\n';
}
```
{% endfold %}

> ExLucas 不是 Lucas 的推广，它们之间的关系就像 ExCRT 跟 CRT 之间的关系。
