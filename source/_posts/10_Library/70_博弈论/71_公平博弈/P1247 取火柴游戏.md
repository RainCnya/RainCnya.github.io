# [P1247 取火柴游戏](https://www.luogu.com.cn/problem/P1247 "null")

## 1. 核心逻辑

- **模型抽象**: 标准 Nim 游戏，增加构造性输出要求。
    
- **破局路径**:
    
    1. **构造策略**: 找到一堆 $a[i]$，使得修改后的 $a'[i] = a[i] \oplus s$ 且满足 $a'[i] < a[i]$。
        
    2. **物理直觉**: 异或和 $s$ 的最高位在 $a[i]$ 中必然存在奇数个 $1$，选取其中一堆必能将其变小。

## 2. 代码实现

{% fold info @AcCode #trd %}

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int maxn = 5e5 + 50;

int a[maxn];
int n;

void solve( )
{
    cin >> n;
    int s = 0;
    for( int i = 1; i <= n; ++ i )
    {
        cin >> a[i];
        s ^= a[i];
    }
    
    if( s == 0 )
    {
        cout << "lose" << '\n';
    }
    else
    {
        for( int i = 1; i <= n; ++ i )
        {
            if( ( a[i] ^ s ) > a[i] ) continue;
            cout << ( a[i] - ( a[i] ^ s ) ) << " " << i << '\n';
            a[i] ^= s;
            break;
        }
        for( int i = 1; i <= n; ++ i )
        {
            cout << a[i];
            if( i != n ) cout << " ";
        }
    }
}

int main( )
{
    int _t = 1;
    //cin >> _t;
    while( _t -- )
    {
        solve( );
    }
    return 0;
}
```
