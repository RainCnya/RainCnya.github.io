# 数论导论与位运算

## 1. 生态位

- **识别**:
    
	1. 需要处理最基础的因倍关系或模运算律。
	
	2. 涉及高次幂取模 $a^b \pmod m$。
        
- **地位**: 它定义了离散世界运行的“物理定律”。
    
- **用途**: 确立整除的传递性、模运算的封闭性，并提供 $O(\log N)$ 的指数级降维工具。

## 2. 逻辑支点

## 2.1 物理定律

1. **整除性**：

	- **整除定义**: 设 $a, b \in \mathbb{Z}$ 且 $b \neq 0$。若存在 $k \in \mathbb{Z}$ 使得 $a = kb$，则称 $b$ 整除 $a$，记作 $b \mid a$。
    
	- **传递性**: 若 $a \mid b$ 且 $b \mid c$，则 $a \mid c$。
        
	- **线性组合性**: 若 $a \mid b$ 且 $a \mid c$，则对任意 $x, y \in \mathbb{Z}$，都有 $a \mid (xb + yc)$。

2. **模运算**：

	- **同余定义**: 若 $m \mid (a - b)$，则称 $a$ 与 $b$ 模 $m$ 同余，记作 $a \equiv b \pmod m$。
    
	- **封闭性**: 模运算对加法、减法、乘法完全透明。
    
	    - $(a \pm b) \pmod m = (a \bmod m + b \bmod m) \pmod m$
        
	    - $(a \times b) \pmod m = (a \pmod m \times b \pmod m) \pmod m$
        
	- **注意：除法不具有封闭性**！关于除法的模运算，详见 [[乘法逆元体系]]。

### 2.2 辗转相除法 

- **定义**: $\gcd(a, b)$ 为最大公约数，$\text{lcm}(a, b)$ 为最小公倍数。

- **原理**: $\gcd(a, b) = \gcd(b, a \bmod b)$。
    
- **互质**: 若 $\gcd(a, b) = 1$，则称 $a, b$ 互质。
    
- **基本公式**: $a \times b = \gcd(a, b) \times \text{lcm}(a, b)$。

### 2.3 快速幂

- **核心逻辑**: 利用倍增思想，将指数 $k$ 进行二进制拆分。
    
- **复杂度**: $O(\log b)$。

## 3. 实战部署

{% fold info @辗转相除法 %}

```cpp
// 核心逻辑：gcd(a, b) = gcd(b, a % b)
// 个人喜欢的递归写法
int gcd( int a, int b )
{
	if( b == 0 ) return a;
	else return gcd( b, a % b );
}

```

{% endfold %}

{% fold info @快速幂 %}

```cpp
// 在 O(log n) 时间内计算 (a ^ b) % m
ll qpow( ll base, ll k, ll mod )
{
    ll res = 1;
    while( k )
    {
        if( k & 1 ) res = ( res * base ) % mod;
        base = ( base * base ) % mod;
        k >>= 1;
    }
    return res;
}
```

{% endfold %}
