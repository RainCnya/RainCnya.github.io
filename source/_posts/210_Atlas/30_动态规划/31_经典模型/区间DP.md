# [L4] 区间状态 动态规划

该类 DP 的特点是，困于问题本身限制，“某类有序事件中前若干个事件的子答案” 不再能支撑状态转移，需要去寻找 “从某个元素起到另一个元素所包含所有的（连续）元素的子答案” 作为状态。

> 一般来说，每个状态会对应于原题序列上的一个区间。

区间具有良好的性质，短的区间可以扩宽成长的区间，相同长度的区间互相不包含。

这样就可以把所有状态理解成 DAG，即不会有互相到达的局面。

## 一、核心特征

- **状态定义**：一般 $f[i][j]$ 为区间状态 $[i, j]$ 的答案（最优解 / 方案数）。

- **状态转移**：$len \to len + 1$ 由小区间向大区间合并。

- **复杂度**：很显然是 $O( N^3 )$。

### 区间DP模板

{% fold info @区间DP %}
```cpp
for( int len = 2; len <= n; ++ len )
{   // 阶段：长度 len
	for( int l = 1; l + len - 1 <= n; ++ l )
	{   // 状态：左右端点 l, r
		int r = l + len - 1;
		// 决策：分割点 k
		for( int k = l; k < r; ++ k )
		{
			f[l][r] = min( f[l][r], f[l][k] + f[k + 1][r] + cost );
		}
	}
}
```
{% endfold %}

### 记忆化搜索模板

适用于 **状态稀疏** 或 **逻辑复杂**（如括号匹配、树结构解析）的问题。

{% fold info @区间DP - 记忆化搜索 %}
```cpp
// 针对 CF149D 括号染色，状态带额外限制
ll solve( int l, int r, int cl, int cr )
{
    if( l >= r ) return 0; // 边界
    if( memo[l][r][cl][cr] != -1 ) return memo[l][r][cl][cr];

    ll ans = 0;
    // 逻辑：是嵌套 (A) 还是拼接 AB
    if( match[l] == r ) {
        // 嵌套：递归内层 [l+1, r-1]
    } else {
        int k = match[l];
        // 拼接：裂解为 [l, k] 和 [k+1, r]
    }
    return memo[l][r][cl][cr] = ans;
}
```
{% endfold %}

## 二、模型分类

### 合并类模型

这类问题的本质是将相邻的两个区间合并成一个新的区间，并产生合并的代价。

- **特征**：最后一步一定是 $[l, k]$ 和 $[k + 1, r]$ 两个区间的合并。
	
- **方程**: $f[l][r] = \max (f[l][k] + f[k+1][r] + Cost)$。

- **相关例题**：

	- [[P1880 石子合并]]：基础区间DP，前缀和优化。
	- [[P1063 能量项链]]：矩阵链乘模型，注意首尾衔接。

- **技巧**：环形处理，通常将数组复制一份拼接在后面（变为 $2N$），最后在所有长度为 $N$ 的区间中统计答案。

### 对称类模型

这类问题的典例是 **回文串**，利用对称结构进行区间处理。

- **特征**：状态转移取决于两端点的性质。

- **方程**: $f[l][r] = \min (f[l+1][r] + Cost_L, f[l][r-1] + Cost_R)$。

- **相关例题**：

	- [[P2890 Cheapest Palindrome G]]：回文对称性。
	- [[P4170 涂色]]：端点状态影响转移。

## 三、进阶技巧

### 逆向思维

当正向推导 “谁先吃/谁先涂” 导致状态混乱时，可以反向思考 “谁是最后一个”。

- **相关例题**：

	- [[P3205 合唱队]]：最后一个人只能在 $l$ 或 $r$ 处。
	- [[P5851 Greedy Pie Eaters P]]：枚举位置 $k$ 最后是被哪头牛吃掉的。

### 状态相关

当 $f[l][r]$ 不足以描述状态（如受相邻颜色限制）时，增加维度扩展状态。

- **相关例题**：

	- [[CF149D Coloring Brackets]]：扩展左右端点颜色。

某些状态优化，暂时还没遇到太多。

- **相关例题**：

	- [[P3147 262144]]：最优解 $V$ 和边界状态 $N$ 的转换，复杂度 $O( V \cdot N )$。
