# 单调队列优化

## 1. 生态位

- **识别**: 转移方程形如 $f[i] = \text{opt} \{ g[j] \} + \text{extra}(i)$，其中 $j \in [L_i, R_i]$，且 $L_i, R_i$ 随 $i$ 非递减。
    
- **地位**: 决策优化的基础模型。是将 $O(K)$ 转移降维至 $O(1)$ 的维度工具。
    
- **用途**: 通过**滑动窗口维护最值**，实现 DP 状态转移的降维。
    
## 2. 逻辑支点

### 2.1 核心原理：单调队列

在决策区间 $[L_i, R_i]$ 中，如果存在两个候选点 $j_1 < j_2$，且在 $j_1$ 还没过期（即还在窗口内）时，其价值 $g[j_1]$ 已经不如 $g[j_2]$ 了，那么 $j_1$ 将永远不可能成为最优决策。 

**结论**：既然 $j_1$ 不可能产生贡献了，直接把它踢出候选名单。

### 2.2 具体推导

要使用单调队列，核心是实现 $i, j$ **依赖分离**：

$$f[i] = \text{val}[i] + \max_{L_i \leq j \leq R_i} \{ f[j] + \text{cost}[j] \}$$

- $j$ **依赖项**: $f[j] + \text{cost}[j]$ 放入单调队列维护最值。
    
- $i$ **依赖项**: $\text{val}[i]$ 在最值选出后进行叠加。

## 3. 实战部署

{% fold info @代码实现: 标准模板 %}
```cpp
// 以最大化转移为例
deque< int > q;
int l = 1; // 辅助指针，用于控制候选点入队

for( int i = 1; i <= n; ++ i )
{
    // 1. 异步入队：将新进入合法范围的决策点加入候选
    // 注意：具体逻辑根据题目 [i-R, i-L] 调整
    int cur = i - L;
    if( cur >= 0 )
    {
        while( !q.empty( ) && g( q.back( ) ) <= g( cur ) ) 
            q.pop_back( );
        q.push_back( cur );
    }

    // 2. 剔除过期决策：下标超出 [i-R, i-L] 的范围
    while( !q.empty( ) && q.front( ) < i - R ) 
        q.pop_front( );

    // 3. 状态转移：队首即为当前最优决策
    if( !q.empty( ) ) f[i] = g( q.front( ) ) + extra[i];
}
```
