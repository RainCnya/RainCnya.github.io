# [L3] 线性代数

## 向量与矩阵

### 向量

#### 定义

同时具有大小与方向的量，在 CS / CP 中，向量通常被视为一个一维的有序数表。

它可以表示空间中的一个点，也可以表示一个方向。设有一个 $n$ 维向量 $v$，记作：

$$
\vec{v} = \begin{bmatrix}
v_1 \\
v_2 \\
\vdots  \\
v_n  \\
\end{bmatrix} 
\quad \text{或} \quad \vec{v} = (v_1, v_2, \dots, v_n)^T
$$

#### 基本性质

- **加法**：

$$
\begin{bmatrix}
x_{1} \\ y_{1} \\
\end{bmatrix}
+
\begin{bmatrix}
x_{2} \\ y_{2} \\
\end{bmatrix}
=
\begin{bmatrix}
x_{1} + x_{2} \\ y_{1} + y_{2} \\
\end{bmatrix}
$$

- **数乘**：

$$
c \cdot \begin{bmatrix}
x \\ y \\
\end{bmatrix}
= \begin{bmatrix}
cx \\ cy \\
\end{bmatrix}
$$
    
- **线性组合**：给定一组向量 $\vec{v_1}, \vec{v_2}, \dots, \vec{v_k}$，对于任何一组数 $a_{i}$，称 $a_{1} v_{1} + a_{2} v_{2} + \dots + a_{m} v_{m}$ 为它们的线性组合。
	
- **线性无关**：若干个向量无法用彼此的线性组合表示，则称它们线性无关。

- **基向量**：对于任意向量 $\vec{a}$ 都存在唯一实数对满足 $\vec{a} = x \vec{e_{1}} + y \vec{e_{2}}$，则 $\vec{e_{1}}, \vec{e_{2}}$ 被称为一组基向量。
	
- **单位向量**：模为 $1$ 的向量。（ 注：模是向量的长度 ）

### 矩阵

#### 定义

矩阵是一个由 $m \times n$ 个数排成的矩形阵列。记作 $A_{m \times n}$。如果在状态压缩或图论中，通常用于表示状态的转移或图的邻接关系。

$$
A = \begin{bmatrix}
a_{11} & a_{12} & \dots & a_{1n} \\
a_{21} & a_{22} & \dots & a_{2n} \\
\vdots & \vdots & \ddots & \vdots \\
a_{m1} & a_{m2} & \dots & a_{mn}  \\
\end{bmatrix}
$$

#### 基本性质

- **矩阵加法**：$C_{i,j} = A_{i,j} + B_{i,j}$。
    
- **矩阵数乘**：$C_{i,j} = c \cdot A_{i,j}$。
    
- **矩阵转置**：行与列互换，记作 $A^T$。$(A^T)_{i,j} = A_{j,i}$。

#### 特殊矩阵

|矩阵名称|符号/性质|描述|
|---|---|---|
|**方阵**|$N \times N$|行数等于列数的矩阵。|
|**零矩阵**|$O$|所有元素均为 $0$ 的矩阵。|
|**单位矩阵**|$I$ 或 $E$|主对角线全为 $1$，其余全为 $0$ 的方阵。满足 $A I = I A = A$。|
|**对角矩阵**|$\Lambda$|除主对角线外元素全为 $0$ 的方阵。|
|**邻接矩阵**|$G$|图论中表示顶点间连通关系的矩阵。|


## 矩阵乘法

#### 定义

设 $A$ 是 $m \times p$ 的矩阵，$B$ 是 $p \times n$ 的矩阵，则它们的乘积 $C = A \times B$ 是一个 $m \times n$ 的矩阵。其元素计算公式为：行乘列之和。

$$C_{i,j} = \sum_{k=1}^{p} A_{i,k} \cdot B_{k,j}$$

#### 基本性质

- **结合律**：$(A B) C = A (B C)$。
    
- **分配律**：$A(B+C) = AB + AC$，$(A+B)C = AC + BC$。
    
- **不满足交换律**：一般情况下 $AB \neq BA$。

### 矩阵快速幂

与数论中的整数快速幂同理，由于矩阵乘法满足结合律，我们可以将 $A^n$ 的计算复杂度从 $O(n \cdot M^3)$ 降维至 $O(\log n \cdot M^3)$（其中 $M$ 为矩阵大小）。

在 CP 中，矩阵快速幂常用于求解**常系数线性齐次递推式**的第 $n$ 项，或者计算**图的长度为** $K$ **的路径方案数**。

#### 构造递推矩阵

**相关例题**：[P1962 斐波那契数列 - 洛谷](https://www.luogu.com.cn/problem/P1962)

已知 $F_n = F_{n-1} + F_{n-2}$，我们可以构造状态列向量 $\begin{bmatrix} F_n \\ F_{n-1} \end{bmatrix}$，并寻找一个转移矩阵 $A$，这里采用待定系数法：$\begin{bmatrix} F_{n} \\ F_{n-1} \end{bmatrix} = \begin{bmatrix} a & b \\ c & d \end{bmatrix} \begin{bmatrix} F_{n-1} \\ F_{n-2} \end{bmatrix}$，很容易得出这个转移矩阵 $A = \begin{bmatrix} 1 & 1 \\ 1 & 0 \end{bmatrix}$。

接着对初始向量进行一次变化，可以得到 $\begin{bmatrix} F_{3} \\ F_{2} \end{bmatrix} = A \begin{bmatrix} 1 \\ 1 \end{bmatrix}$，接着采用数学归纳得到：$\begin{bmatrix} F_{n} \\ F_{n-1} \end{bmatrix} = A^{n-2} \begin{bmatrix} 1 \\ 1 \end{bmatrix}$。

矩阵乘法满足结合律，所以可将矩阵的幂乘使用快速幂的方式来处理。

{% fold info @矩阵快速幂模板 %}
```cpp
int L = 3;
struct Matrix {
    ll m[maxn][maxn];
    void clear( ) { memset( m, 0, sizeof( m ) ); }
    void init( ) { clear( );
        for( int i = 1; i <= L; ++ i ) m[i][i] = 1; 
    }
    
    Matrix friend operator * ( const Matrix &a, const Matrix &b ) {
        Matrix res; res.clear( );
        for ( int i = 1; i <= L; i++ )
            for ( int j = 1; j <= L; j++ )
                for ( int k = 1; k <= L; k++ )
                    res.m[i][j] = ( res.m[i][j] + a.m[i][k] * b.m[k][j] ) % mod;
        return res;
    }
} A;

Matrix mqpow( Matrix a, ll k ) {
    Matrix res; res.init( );
    for( ; k; k >>= 1, a = a * a )
        if( k & 1 ) res = res * a;
    return res;
}
```

### 广义矩阵乘法

在最短路 DP 中，普通的加法被 $\min$ 取代，乘法被加法取代：$(min, +)$

$$C_{i,j} = \min_{k=1}^p \{ A_{i,k} + B_{k,j} \}$$

这种定义下的矩阵乘法**依然满足结合律**，因此可以使用矩阵快速幂求解恰好经过 $K$ 条边的最短路问题。

## 线性方程组

### 矩阵的秩

#### 定义

秩是衡量矩阵（ 或线性变换 ）“包含多少有效信息维度” 的指标，记作 $rank(A)$。

#### 基本性质

- **列空间**：矩阵所有列向量通过线性组合能到达的所有点的集合。

- **行列秩相等**：矩阵的列秩永远等于行秩。

- **满秩**：对于一个 $n$ 阶方阵 $A$，若 $rank(A) = n$，则 $A$ 为满秩矩阵。

> 满秩意味着变换**没有降维**（没有信息丢失），因此满秩矩阵式**可逆**的（ 可以通过乘以逆矩阵 $A^{-1}$ 复原 ）。

- 在 CP 中，求矩阵的秩通常使用高斯消元法。

### 行列式

#### 定义

行列式，是方阵的一种运算，对于方阵 $A$， $\det(A)$ 表示方阵 $A$ 的行列式。

- **几何**：从几何上看，它表示矩阵代表的线性变换对空间的 “缩放比例”。

- **与秩的关系**：如果 $\det(A) = 0$，说明变换后的体积被压缩成了 $0$，此时矩阵**不满秩**，且不可逆。

#### 基本性质

设 $A$ 是一个 $n$ 阶方阵：

1. **转置不变**：$\det(A) = \det(A^T)$。
    
2. **交换变号**：交换矩阵的两行（或两列），行列式的值变号。
    
3. **倍乘提公因数**：把某一行的所有元素乘 $k$，行列式的总值也乘 $k$。
    
4. **倍加不变**：把某一行的 $k$ 倍加到另一行上，行列式的值**不变**。
    
5. **上三角矩阵的行列式**：等于其主对角线元素的乘积。

- 在 CP 中，我们利用 “倍加不变” 性质，使用高斯消元法将矩阵化为上三角矩阵，求解即可。

### 高斯消元
给定一个包含 $n$ 个未知数 $m$ 个方程的线性方程组：

$$ (*) = \begin{cases}
a_{1,1}x_{1} + a_{1,2}x_{2} + \dots + a_{1,n}x_{n} = b_{1} \\ a_{2,1}x_{1} + a_{2,2}x_{2} + \dots + a_{2,n}x_{n} = b_{2} \\ \vdots \\ a_{m,1}x_{1} + a_{m,2}x_{2} + \dots + a_{m,n}x_{n} = b_{m} \\
\end{cases} $$

我们可以将其提取为**增广矩阵**：

$$(A|b) = \left( \begin{array}{cccc|c}
a_{1,1} & a_{1,2} & \dots & a_{1,n} & b_{1} \\
a_{2,1} & a_{2,2} & \dots & a_{2,n} & b_{2} \\
\vdots & \vdots & \ddots & \vdots & \vdots \\
a_{m,1} & a_{m,2} & \dots & a_{m,n} & b_{m} \\
\end{array} \right)$$

根据线性代数基本定理，对增广矩阵进行以下三种**初等行变换**，不会改变方程组的解集：

1. **交换变换**：交换两行（对应交换两个方程的位置）。
    
2. **倍法变换**：用一个非零实数 $k$ 乘以某一行（对应方程两边同乘非零数）。
    
3. **消去变换**：将一行的 $k$ 倍加到另一行（对应方程组的叠加消元）。

通过上述变换，我们可以将增广矩阵化为**行阶梯形矩阵**：

$$\left( \begin{array}{cccccc|c}
a'_{1,i_1} & a'_{1,i_2} & \dots & a'_{1,r} & \dots & a'_{1,n} & b'_1 \\
0 & a'_{2,i_2} & \dots & a'_{2,r} & \dots & a'_{2,n} & b'_2 \\
\vdots & \vdots & \ddots & \vdots & \ddots & \vdots & \vdots \\
0 & 0 & \dots & a'_{r,i_r} & \dots & a'_{r,n} & b'_r \\
0 & 0 & \dots & 0 & \dots & 0 & d_{r+1} \\
\vdots & \vdots & \vdots & \vdots & \vdots & \vdots & \vdots \\
0 & 0 & \dots & 0 & \dots & 0 & d_{m}
\end{array} \right)$$

设矩阵 $A$ 的秩为 $r(A)$，增广矩阵 $(A|b)$ 的秩为 $r(A|b)$：

- **无解**： 若 $r(A) < r(A|b)$。 

	- 具体表现为：存在某一行变换后出现 $(0, 0, \dots, 0 | d)$ 且 $d \neq 0$ 的情况，即出现 $0 = d$ 的矛盾。
    
- **有解**： 若 $r(A) = r(A|b) = r$。
    
    - **唯一解**：当 $r = n$（秩等于未知数个数）时。此时矩阵可化为单位阵。
        
    - **无穷多解**：当 $r < n$（秩小于未知数个数）时。此时有 $n-r$ 个自由变量。

{% fold info @高斯消元模板 ( $O(N^3)$ ) %}
```cpp
// 1 唯一解 | 0 无穷解 | -1 无解
int gauss( )
{
    int curi = 1;  // 当前处理行
    for( int j = 1; j <= n; ++ j )
    {
        int p = curi;  // 主元
        for( int i = curi + 1; i <= n; ++ i )
            if( fabs( a[i][j] ) > fabs( a[p][j] ) ) p = i;
	    // 这一列找不到非 0 元素
        if( fabs( a[p][j] ) < eps ) continue;
        // 交换行
        swap( a[p], a[curi] );
		// 当前行同除，令主元系数化一
        double div = a[curi][j];
        for( int k = j; k <= n + 1; ++ k ) a[curi][k] /= div;
        // 对其他行消元
        for( int i = 1; i <= n; ++ i )
        {
            if( i != curi && fabs( a[i][j] ) > eps )
            {
                double mul = a[i][j];
                for( int k = j; k <= n + 1; ++ k )
                    a[i][k] -=  a[curi][k] * mul;
            }
        }
        curi ++;
    }

    if( curi <= n )
    {
        for( int i = curi; i <= n; ++ i )
            if( fabs( a[i][n + 1] ) > eps ) return -1; // 发现 0 != 0 无解
        return 0; // 发现 0 == 0 无穷多解
    }
    return 1; // 唯一解
}
```
