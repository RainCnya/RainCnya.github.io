### 整除

#### 定义

设 $a, b \in \mathbb{Z}$ 且 $a \neq 0$，如果存在整数 $q$，使得 $b = a \cdot q$，则称 $a$ 整除 $b$，或者 $b$ 可被 $a$ 整除，记作 $a \mid b$，此时称 $a$ 是 $b$ 的因数（约数），$b$ 是 $a$ 的倍数；如果不存在这样的整数 $q$，则称 $a$ 不整除 $b$，记作 $a \nmid b$。

#### 基本性质

设 $a, b \in \mathbb{Z}$ 且 $a \neq 0$。

- **自反性**：$a \mid a$。
- **传递性**：若 $a \mid b, b \mid c$，则 $a \mid c$。
- **线性性**：若 $a \mid b$ 且 $a \mid c$，则对任意整数 $x, y$，有 $a \mid (bx + cy)$。
- **与符号无关**：若 $a \mid b$，则 $−a \mid b$ 且 $a \mid −b$。
- **比较性**：若 $a \mid b$ 且 $b \neq 0$，则 $| a | \leq | b |$。
- **遍历对称**：若 $b > 0$，则当 $a$ 遍历 $b$ 的全体**正**因数时，$\frac{b}{a}$ 亦然。

### 取整函数

#### 定义

对于 $x \in \mathbb{R}$，定义向下取整函数和向上取整函数分别为：

$$
\lfloor x \rfloor = \max \{ n \in \mathbb{Z} | n \leq x \}; \lceil x \rceil = \min \{ n \in \mathbb{Z} | n \geq x \}
$$

取整函数通常指向下取整函数，此外还有**四舍五入函数**。

$$
\lfloor x \rceil = \begin{cases}
\lfloor x + 0.5 \rfloor , x \geq 0 \\
\lceil x - 0.5 \rceil , x \leq 0
\end{cases}
$$

> 数论中不常用。注意 $\lfloor -2.5 \rceil = -3$。

#### 基本性质

设 $x, y \in \mathbb{R}, n \in \mathbb{Z}$。

- **基本不等式**：$| x | \leq x < | x | + 1, | x | - 1 < x \leq | x |$。
- **整数提出**：$\lfloor x + n \rfloor = \lfloor x \rfloor + n, \lceil x + n \rceil = \lceil x \rceil + n$。
- **反号关系**：$\lfloor -x \rfloor = -\lceil x \rceil, \lceil -x \rceil = -\lfloor x \rfloor$
- **合并公式**：$\lfloor x \rfloor + \lfloor y \rfloor = \lfloor x + y \rfloor \leq \lfloor x \rfloor + \lfloor y \rfloor + 1$
- **等价关系** 1：$n \leq x \iff n \leq \lfloor x \rfloor, n \geq x \iff n \geq \lceil x \rceil$
- **等价关系** 2：$n < x \iff n < \lfloor x \rfloor, n > x \iff n > \lceil x \rceil$

### 带余除法

#### 定义

设 $a, b \in \mathbb{Z}, a \neq 0$，则存在唯一的整数 $q$（商）和 $r$（余数），使得

$$
b = a \cdot q + r \quad ( 0 \leq r < | a | )
$$

这样的式子称为 $b$ 除以 $a$ 的带余除法算式，$q$ 称为不完全商（简称商），$r$ 称为余数。

#### 基本性质

设 $a, b \in \mathbb{Z}, a \neq 0$，$q$ 是 $b$ 除以 $a$ 的商。

- **整除判定**：$a \mid b$ 当且仅当余数为 $0$
- **余数遍历**：相邻的 $| a |$ 个整数除以 $a$，余数恰好遍历 $[0, a−1]$
- **商的估计**：若除数 $a > 0$，则商 $q = \lfloor \frac{b}{a} \rfloor$；若 $a < 0$，则商 $q = \lceil \frac{b}{a} \rceil$

- 计算机遵循 “余数与被除数同号” 规则。

|   7    |  −3   | 7 $\div$ (−3)  |  −2  |   1   |
|   −7   |   3   |  −7 $\div$ 3   |  −2  |  −1   |
|   −7   |  −3   | −7 $\div$ (−3) |  2   |  −1   |

### 整除分块

#### 核心逻辑

利用分块思想，将 $\left\lfloor  \frac{n}{k}  \right\rfloor$ 进行分块，从而在 $O( \sqrt{ N })$ 的时间内求解形如下式：

$$
\sum_{i=1}^{n} f(i) \cdot g(\lfloor \frac{n}{k} \rfloor)
$$
原理即本质不同的 $\lfloor \frac{n}{i} \rfloor$ 只有至多 $O( \sqrt{ n })$ 种。因此 $\lfloor \frac{n}{i} \rfloor$ 相同值肯定是连续的，那么就可以找出这 $O( \sqrt{ n })$ 区间：$[l_{1}, r_{1}], [l_{2},r_{2}], \dots, [l_{k}, r_{k}]$，接着求出：

$$
\sum_{i=1}^{k} \left( g( \lfloor \frac{n}{l_{i}} \rfloor ) \cdot \sum_{j=l_{i}}^{r_{i}} (f(j)) \right)
$$

若分块后可以 $O(1)$ 求出，则整除分块是 $O( \sqrt{ N })$ 的。

#### 寻找区间

对于一个 $l$ 所对应的 $\lfloor \frac{n}{l} \rfloor$，要寻找一个最大的 $r$ 使得 $\lfloor \frac{n}{l} \rfloor = \lfloor \frac{n}{r} \rfloor$，从而找到区间 $[l,r]$，则有：

$$
r = \left\lfloor \frac{n}{ \lfloor \frac{n}{l} \rfloor } \right\rfloor 
$$

而对于向上取整的情况，即对于 $l$ 对应的 $\lceil \frac{n}{l} \rceil$，要寻找一个最大的 $r$ 使得 $\lceil \frac{n}{l} \rceil = \lceil \frac{n}{r} \rceil$，从而找到区间 $[l, r]$，则有：

$$
r = \left\lfloor  \frac{n-1}{ \lfloor \frac{n-1}{l} \rfloor  }  \right\rfloor 
$$

### 快速幂

#### 核心逻辑

利用倍增思想，将指数 $k$ 进行二进制拆分，从而将 $O(k)$ 的线性乘法降维至 $O(\log k)$ 的时间复杂度内完成高次幂运算取模。

{% fold info @快速幂 %}
```cpp
// 在 O(log n) 时间内计算 (a^k) % mod
ll qpow( ll a, ll k )
{
    ll res = 1;
    while( k )
    {
        if( k & 1 ) res = ( res * a ) % mod;
        a = ( a * a ) % mod;
        k >>= 1;
    }
    return res;
}
```
{% endfold %}

## 互素与质数

### 最大公约数与最小公倍数

#### 定义（最大公约数）

设 $a_{1}, a_{2}, \dots, a_{n}$ 是 $n$ 个不全为零的整数，如果正整数 $d$ 满足：

- $d \mid a_{1}, d \mid a_{2}, \dots , d \mid a_{n}$ （ 即 $d$ 是每个 $a_{i}$ 的约数 ）
- 对任意满足 $c \mid a_{1}, c \mid a_{2}, \dots , c \mid a_{n}$ 的整数 $c$，有 $c \mid d$

则称 $d$ 为 $a_{1}, a_{2}, \dots, a_{n}$ 的最大公约数，记作：

$$
d = gcd( a_{1}, a_{2}, \dots , a_{n} ) \quad d = ( a_{1}, a_{2}, \dots, a_{n} )
$$

#### 定义（最小公倍数）

设 $a_{1}, a_{2}, \dots, a_{n}$ 是 $n$ 个非零整数，如果正整数 $m$ 满足：

- $a_{1} \mid m, a_{2} \mid m, \dots, a_{n} \mid m$ （ 即 $m$ 是每个 $a_{i}$ 的倍数 ）
- 对任意满足 $a_{1} \mid c, a_{2} \mid c, \dots, a_{n} \mid c$ 的整数 $c$，有 $m \mid c$

则称 $m$ 为 $a_{1}, a_{2}, \dots, a_{n}$ 的最小公倍数，记作：

$$
m = lcm( a_{1}, a_{2}, \dots, a_{n}) \quad m = [a_{1}, a_{2}, \dots, a_{n}]
$$

#### 基本性质

设 $a, b, c, d \in \mathbb{Z}$。

- **交换律**：$(a, b) = (b, a)$，$[a, b] = [b, a]$
- **结合律**：$(a, b, c) = ((a, b), c) = (a, (b, c))$，$[a, b, c] = [[a, b], c] = [a, [b, c]]$
- **单位元**：$(a, 0) = | a |$，$[a, 1] = | a |$
- **与符号无关**：$(a, b) = ( |a| , |b| )$，$[a, b] = [ |a|, |b| ]$
- **乘法提出**：$(ka, kb) = |k|(a, b)$，$[ka, kb] = |k|[a, b]$
- **幂提出**：$(a^{k}, b^{k}) = (a, b)^{k}$，$[a^{k}, b^{k}] = [a, b]^{k}$
- **乘积关系**：$(a, b) \times [a, b] = | a \times b |$
- $gcd$ **更相减损**：$(a, b) = (a \pm b, b) = (a, b \pm a)$
- $gcd$ **辗转相除**：$(a, b) = (a \bmod b, b) = (a, b \bmod a)$

{% fold info @辗转相除法 %}
```cpp
// 核心逻辑：gcd(a, b) = gcd(b, a % b)
int gcd( int a, int b ) {
	if( b == 0 ) return a;
	else return gcd( b, a % b );
}
```
{% endfold %}

### 互素

#### 定义

设 $a, b \in \mathbb{Z}$，若 $gcd(a, b) = 1$，则称 $a$ 与 $b$ 互素（或互质），记作：

$$
a \perp b
$$

若 $gcd(a, b) \neq 1$，则称 $a$ 与 $b$ 不互质，记作：

$$
a \not\perp b
$$

特别地，若 $gcd(a_{1}, a_{2}, \dots, a_{n}) = 1$，则称 $a_{1}, a_{2}, \dots, a_{n}$ 整体互素。

若任意两个 $a_{i}, a_{j} (i \neq j)$ 都满足 $gcd(a_{i}, a_{j}) = 1$，则称它们两两互质。

#### 基本性质

设 $a, b, c, d \in \mathbb{Z}$。

- **对称性**：若 $a \perp b$，则 $b \perp a$。
- **遍历性**：若 $a \perp b$，则相邻的 $| a |$ 个整数乘 $b$ 再除以 $a$，余数恰好遍历 $[0, a-1]$
- **与符号无关**：若 $a \perp b$，则 $-a \perp b$ 且 $a \perp -b$
- **乘法的保持**：若 $a \perp b$ 且 $a \perp c$，则 $a \perp bc$
- **倍数的约化**：若 $a \perp b$，$c \mid a$ 且 $d \mid b$，则 $c \perp d$
- **与整除的关系**：若 $a \perp b$ 且 $a \mid bc$，则 $a \mid c$

### 素数

#### 定义

设 $p \in \mathbb{Z^+}$ 且 $p > 1$，若 $p$ 的正因数只有 $1$ 和 $p$ 本身，则称 $p$ 为**素数**（或质数），否则，称 $p$ 为**合数**。

$$
P = \{ p \in \mathbb{Z^{+}} | p > 1, \text{且 p 是素数} \}
$$

> 特别地，$1$ 既不是素数也不是合数。

#### 基本性质

设 $p$ 是素数，$a, b \in \mathbb{Z}$。

- **整除性**：若 $p \nmid a$，则 $gcd(p, a) = 1$
- **整除性推广**：若 $p \mid a_{1} a_{2} \dots a_{n}$，则存在某个 $i$ 使得 $p \mid a_{i}$
- **素性判断**：若对所有满足 $1 < d \leq \sqrt{ p }$ 的整数 $d$ 都有 $d \nmid p$，则 $p$ 是素数
- **素数的个数**：素数的个数是无限的（欧几里得证明）

### 唯一分解定理与因数统计

#### 标准分解式

任意大于 $1$ 的整数 $n$ 都可以唯一地（不考虑顺序）表示为：

$$
n = p_{1}^{a_{1}} p_{2}^{a_{2}} \cdots p_{k}^{a_{k}} = \prod_{i=1}^{k} p_i^{a_i}
$$

> 其中 $p_i$ 为递增的素数，$a_i$ 为正整数指数，这也是 _唯一分解定理_。

#### 因数统计

基于上述表示法，无需 $O(N)$ 枚举即可推导出全局因数属性：

- **因数个数** $d(n)$: 每个质因数 $p_i$ 可以在因数中出现 $0 \dots a_i$ 次（共 $a_i + 1$ 种选择）。
- 
$$
d(n) = \prod_{i=1}^{k} (a_i + 1
)$$

- **因数总和** $\sigma(n)$: 利用分配律，因数之和等于各质因数等比数列之和的乘积。

$$
\sigma(n) = \prod_{i=1}^{k} \left( \sum_{j=0}^{a_i} p_i^j \right) = \prod_{i=1}^{k} \frac{p_i^{a_i+1}-1}{p_i-1}
$$

## 同余方程与中国剩余定理

### 同余

#### 定义

设 $m \in \mathbb{Z^{+}}$，若 $a, b \in \mathbb{Z}$ 满足 $m \mid (a - b)$，则称 $a$ 与 $b$ 模 $m$ 同余，记作：

$$
a \equiv b \pmod m
$$

此时 $m$ 称为 **模数**（或模），该式称为 **同余式**。若 $m \nmid (a - b)$，则称 $a$ 与 $b$ 模 $m$ 不同余，记作：

$$
a \not\equiv b \pmod m
$$

#### 基本性质

设 $a, b, c, d \in \mathbb{Z}, m \in \mathbb{Z^{+}}$

- **自反性**：$a \equiv a \pmod m$
- **对称性**：若 $a \equiv b \pmod m$，则 $b \equiv a \pmod m$
- **传递性**：若 $a \equiv b \pmod m$ 且 $b \equiv c \pmod m$，则 $a \equiv c \pmod m$
- **保加性**：若 $a \equiv b \pmod m$ 且 $c \equiv d \pmod m$，则 $a + c \equiv b + d \pmod m$
- **保乘性**：若 $a \equiv b \pmod m$ 且 $c \equiv d \pmod m$，则 $ac \equiv bd \pmod m$
- **幂等性**：若 $a \equiv b \pmod m$ 则对任意 $n \in \mathbb{N^{*}}$，有 $a^{n} \equiv b^{n} \pmod m$
- **消去律**：若 $ac \equiv bc \pmod m$ 且 $gcd(c, m) = 1$，则 $a \equiv b \pmod m$
- **与带余除法的关系**：$a \equiv b \pmod m$ 当且仅当 $a$ 和 $b$ 除以 $m$ 所得余数相同

### 裴蜀定理

#### 定理

设 $a, b$ 是不全为零的整数，则存在整数 $x, y$，使得：

$$
ax + by = gcd(a, b)
$$

特别地，$a$ 与 $b$ 互素当且仅当存在整数 $x, y$ 满足：

$$
ax + by = 1
$$

例如取 $a = 28, b = 19$，有 $gcd(28, 19)$，存在整数 $x = -2, y = 3$，使得：

$$
28 \times (-2) + 19 \times 3 = -56 + 57 = 1
$$

#### 扩展欧几里得

输入 $a, b$ 为不同时为 $0$ 的整数；输出 $g, x, y$ 满足：

$$
ax = by = g = gcd(a, b)
$$

- 若 $| b | \neq 0$，扩展欧几里得算法求出的可行解必定满足：
- 
$$
| x | \leq | b |, \quad | y | \leq | a |
$$

- 且代码 `exgcd(a,b)` 流程中，任何计算的绝对值都不超过
- 
$$
| a | + | b |
$$

- 因此，如果 $|a|,|b| \leq 10^9$，只求解 `ax + by = gcd(a,b)` 中的 $x, y$ 不会爆 `int`

> 但 ax 和 by 有可能

#### 证明

$$
\begin{align}
x_{1}b + y_{1}(a \bmod b) \\
= x_{1}b + y_{1}(a - \lfloor  \frac{a}{b} \rfloor b) \\
= y_{1}a + (x_{1} - \lfloor \frac{a}{b} \rfloor y_{1} ) b
\end{align}
$$

> 以上证明仅在 $a, b$ 非负的情况下有效，但其他情况类似。
> PS：代码中的 $a \div b$ 不等于数学公式中的 $\lfloor \frac{a}{b} \rfloor$

{% fold info @ExGCD 标准模板 (构造特解) %}
```cpp
ll exgcd( ll a, ll b, ll &x, ll &y )
{
    if( b == 0 ) { x = 1; y = 0; return a; }
    ll d = exgcd( b, a % b, y, x );
    y -= ( a / b ) * x;
    return d;
}
```
{% endfold %}

### 二元一次不等式 & 同余方程

#### 定义（二元一次方程组）

$A, B, C$ 均为整数，求出方程：

$$
A \dot x + B \cdot y = C
$$

关于 $x, y$ 的全体整数解。

#### 定义（同余方程）

$A, B, m$ 均为整数，求出方程：

$$
A \cdot x \equiv B \pmod m
$$

关于 $x$ 的全体整数解。

#### 二者等价

$$
\begin{align}
A \cdot x + B \cdot y = C & \iff B \cdot y = C - A \cdot x \\
& \iff B \mid C - A \cdot x \\
& \iff A \cdot x \equiv C \pmod B 
\end{align}
$$

#### 例子

**解**：$A \cdot x + B \cdot y = C$

- 令 $d = gcd(A, B)$，如果 $d \mid C$，那么方程无解，否则两边同时除以 $d$。

**解**：$\frac{A}{d} \cdot x + \frac{B}{d} \cdot y = \frac{C}{d}$

- `Exgcd` 找出 $\frac{A}{d} \cdot x + \frac{B}{d} \cdot y = 1$ 的一个任意解 $(x_{0}, y_{0})$

- 同时 $\frac{A}{d} \cdot x + \frac{B}{d} \cdot y = 0$ 的通解为 $( \frac{B}{d} k, - \frac{A}{d} k ), k \in \mathbb{Z}$

- 故原方程的通解为 $(\frac{C}{d} x_{0} + \frac{B}{d} k, \frac{C}{d} y_{0} - \frac{A}{d} k), k \in \mathbb{Z}$

### 逆元

#### 定义

设 $a, m \in \mathbb{Z}$，若存在整数 $x$ 使得：

$$
ax \equiv 1 \pmod m
$$

则称 $x$ 为 $a$ 模 $m$ 的**逆元**（或模逆元），记作 $a^{-1} \pmod m$ 或 $a^{-1}$

$$
a a^{-1} \equiv 1 \pmod m
$$

此时也称 $a$ 在模 $m$ 下**可逆**（或存在逆元）。

若这样的 $x$ 不存在，则称 $a$ 在模 $m$ 下**不可逆**。

#### 基本性质

设 $a, b, m \in \mathbb{Z}$

- **存在性定理**：$a$ 模 $m$ 存在逆元当且仅当 $gcd(a, m) = 1$
- **唯一性**：若 $a$ 模 $m$ 的逆元存在，则它们模 $m$ 的余数相同
- **逆元的逆元**：模 $m$ 的意义下，若 $a$ 可逆则 $a^{-1}$ 也可逆，且 $(a^{-1})^{-1} \equiv a \pmod m$
- **乘积的逆元**：若 $a$ 和 $b$ 均模 $m$ 可逆，则 $ab$ 也模 $m$ 可逆，且

$$
(ab)^{-1} \equiv b^{-1} a^{-1} \pmod m
$$

- **除法定义**：若 $b$ 在模 $m$ 下可逆，则定义：

$$
\frac{a}{b} \equiv a b^{-1} \pmod m
$$

#### 线性递推求逆元

除了使用 ExGCD 和费马小定理单次求逆元外，如果需要预处理 $1 \dots N$ 范围内所有整数的逆元，可通过 $O(N)$ 线性递推推导：

- **推导**：
		
	- 设 $m = ki + r$，$ki + r \equiv 0 \pmod m \implies r \equiv -ki \pmod m$。
	- 两边同乘 $i^{-1} \cdot r^{-1}$ 得到：$i^{-1} \equiv -k \cdot r^{-1} \pmod m$。
    
- **结论**：$inv[i] = (m - m/i) \cdot inv[m \bmod i] \pmod m$。

{% fold info @逆元全家桶 %}
```cpp
// 1. 线性递推预处理 1..n 的逆元 (O(n))
void initInv( int n, int mod )
{
    inv[1] = 1;
    for( int i = 2; i <= n; ++ i )
    {
        inv[i] = (ll)( mod - mod / i ) * inv[mod % i] % mod;
    }
}

// 2. 快速幂求逆元 (前提: mod 为质数, O(log mod))
ll getInv1( ll a, ll mod )
{
    return qpow(a, mod - 2, mod);
}

// 3. ExGCD 求逆元 (通用方案: 需保证 gcd(a, mod) == 1, O(log mod))
ll getInv2( ll a, ll mod )
{
    ll x, y;
    ll d = exgcd( a, mod, x, y );
    if( d != 1 ) return -1; // 无解
    return ( x % mod + mod ) % mod; // 确保结果为正数
}
```
{% endfold %}

### 中国剩余定理

#### 定理：

设 $m_{1}, m_{2}, \dots, m_{k}$ 是两两互质的正整数，则对于任意整数 $a_{1}, a_{2}, \dots, a_{k}$，同余方程组：

$$
\begin{cases}
x \equiv a_{1} \pmod{m_{1}} \\
x \equiv a_{2} \pmod{m_{2}} \\
\vdots \\
x \equiv a_{k} \pmod{m_{k}} \\
\end{cases}
$$

在模 $M = m_{1} m_{2} \cdots m_{k}$ 的意义下有解且有唯一解。

即 $[0, M-1]$ 内存在唯一的整数 $x$ 满足上述所有同余式。

#### 解的存在性

设 $M_{i} = \prod_{j \neq i} m_{j} = \frac{M}{m_{i}}$，显然 $gcd( m_{i}, M_{i}) = 1$。

故存在整数 $t_{i}$ 使得 $M_{i} t_{i} \equiv 1 \pmod m_{i}$，即 $t_{i}$  是 $M_{i}$ 模 $m_{i}$ 的逆元。构造：

$$
x_{0} = \sum_{i=1}^{k} ( a_{i} M_{i} t_{i} )
$$

发现它是同余方程组的一个特解，即满足所有的：

$$
x_{0} \equiv a_{i} \pmod{m_{i}}
$$

#### 解的唯一性

假设 $x$ 和 $y$ 都是同余方程组的解，即对所有的 $i$ 有：

$$
x \equiv y \equiv a_{i} \pmod{m_{i}}
$$

即 $m_{i} \mid (x - y)$ 对所有 $i$ 成立，因此 $lcm(m_{1}, m_{2}, \cdots , m_{k}) | (x - y)$。

考虑 $m_{1}, m_{2}, \cdots , m_{k}$ 两两互素，$lcm(m_{1}, m_{2}, \cdots, m_{k}) = M$。综上，通解为：

$$
x \equiv x_{0} \equiv \sum_{i=1}^{k} (a_{i} M_{i} t_{i}) \pmod M
$$

{% fold info @标准 CRT 模板 %}
```cpp
// 标准 CRT：处理两两互质的模数
ll crt( int n, ll a[], ll m[] )
{
    ll M = 1, ans = 0;
    for( int i = 1; i <= n; ++ i ) M *= m[i];

    for( int i = 1; i <= n; ++ i )
    {
        ll Mi = M / m[i];
        ll ti, y;
        // 求解 Mi * t ≡ 1 (mod mi)
        exgcd( Mi, m[i], ti, y );
        
        // 构造并累加：ans = ∑ a_i * Mi * (Mi^-1 mod mi)
        ti = ( ti % m[i] + m[i] ) % m[i];
        ans = ( ans + a[i] * Mi % M * ti % M ) % M;
    }
    return ( ans + M ) % M;
}
```
{% endfold %}

### Lucas 定理

#### 定义

设 $n, m, p \in \mathbb{Z^{+}}$，且 $p$ 为素数，则：

$$
\binom{n}{m} = \binom{ \lfloor n / p \rfloor }{ \lfloor m / p \rfloor } \binom{ n \bmod p }{ m \bmod p } \pmod p
$$

其中 $\binom{n}{m}$ 表示组合数（或者二项式系数），表示从 $n$ 个不同元素中选取 $m$ 个元素的不同方式的数目，其计算公式为：

$$
\binom{n}{m} = \frac{n!}{m! (n - m)!}
$$

特别地，当 $n < m$ 时，规定 $\binom{n}{m} = 0$。

- 考虑求出 $[2,n]$ 范围内所有素数
- 如果换个试除法判断，复杂度为 $O(n \sqrt{ n })$
- 有没有更快的方法

    进而标记 $c$ 是合数

    内层循环总会在到达 $p '$ 前中断  
    综上合数 $c = p \times m$ 由且只由 $p$ 标记

#### 积性函数的同步推导 (附)

除了求素数，利用线性筛 “**每个合数只被其最小质因子处理一次**” 的性质，还能在线性时间内顺便求出所有的积性函数（ 如欧拉函数 $\phi$、莫比乌斯函数 $\mu$ ）：

- **定义**: 若 $\gcd(a, b) = 1$，则满足 $f(ab) = f(a)f(b)$。

1. $i$ **为质数**：根据定义直接赋值。如 $\phi(p) = p-1, \mu(p) = -1$。
    
2. $\gcd(i, p_j) = 1$：利用积性性质。$f(i \cdot p_j) = f(i) \cdot f(p_j)$。
    
3. $p_j \mid i$（即 `i % primes[j] == 0`）：
    
    - $\phi(i \cdot p_j) = \phi(i) \cdot p_j$
        
    - $\mu(i \cdot p_j) = 0$（包含平方因子）
        

{% fold info @线性筛全家桶 ( 质数 + 欧拉函数 + 莫比乌斯函数 ) %}
```cpp
int primes[maxn], cnt;
bool notPrime[maxn];
int phi[maxn], mu[maxn];

void sieve( int n )
{
    notPrime[0] = notPrime[1] = true;
    phi[1] = 1;
    mu[1] = 1;

    for( int i = 2; i <= n; ++ i )
    {
        if( !notPrime[i] )
        {
            primes[++ cnt] = i;
            phi[i] = i - 1;
            mu[i] = -1;
        }
        for( int j = 1; j <= cnt && i * primes[j] <= n; ++ j )
        {
            int next = i * primes[j];
            notPrime[next] = true;

            if( i % primes[j] == 0 )
            {
                // primes[j] 是 i 的因子，不满足互质
                phi[next] = phi[i] * primes[j];
                mu[next] = 0; // 包含平方因子 p^2
                break; // 核心：保证每个合数只被最小质因子筛掉
            }
            
            // primes[j] 小于 i 的最小质因子，利用积性性质
            phi[next] = phi[i] * (primes[j] - 1);
            mu[next] = -mu[i];
        }
    }
}
```
{% endfold %}

### 欧拉函数
#### 定义
设 $n \in \mathbb{Z^{+}}$，欧拉函数 $\phi(n)$ 定义为不超过 $n$ 且与 $n$ 互质的正整数的个数，即：

$$
\phi(n) = | \{ 1 \leq k \leq n \mid gcd(k, n) = 1 \} |
$$

特别地，规定 $\phi(1) = 1$。

#### 基本性质
设 $k, m, n \in \mathbb{Z^{+}}$

- **积性函数**：若 $gcd(m, n) = 1$，则 $\phi(mn) = \phi(n) \cdot \phi(m)$，可用中国剩余定理证明
- **素数的情形**：$\phi(p^{k}) = p^{k} - p^{k-1} = p^{k-1}(p - 1)$
- **一般公式**：若 $n = p_{1}^{a_{1}} p_{2}^{a_{2}} \cdots p_{k}^{a_{k}}$，则：

$$
\phi(n) = \prod_{i=1}^{k} p_{i}^{a_{i}-1} (p_{i} - 1) = n \prod_{i=1}^{k} (\frac{p_{i}-1}{p_{i}})
$$

- **高斯定理**：$\sum_{d \mid n} \phi(d) = n$

> 证明：枚举因子 $d$，恰好有 $\phi(d)$ 个不超过 $n$ 的正整数 $k$ 满足 $gcd(k, n) = \frac{n}{d}$

{% fold info @欧拉函数计算 %}
```cpp
// O( sqrt(n) ) 计算 phi(n)
phi = n;
for( int i = 2; i * i <= n; ++ i )
{
	if( n % i == 0 )
	{
		phi = phi / i * (i - 1);
		while( n % i == 0 ) n /= i;
	}
}
if( n != 1 ) phi = phi / n * (n - 1);
```
{% endfold %}

### 同余类

#### 定义

设 $m \in \mathbb{Z^{+}}$，对于任意整数 $a$，定义集合：

$$
\bar{a} = \{ x \in \mathbb{Z} \mid x \equiv a \pmod m \}
$$

称为 $a$ 模 $m$ 的**同余类**（或剩余系），其中 $a$ 称为该同余类的**代表元**。所有模 $m$ 同余类的集合记为：

$$
\mathbb{Z}_{m} = \{ \overline{0}, \overline{1}, \dots, \overline{m-1} \}
$$

#### 基本性质

设 $m \in \mathbb{Z^{+}}, a, b \in \mathbb{Z}$。

- **划分性质**：不同的同余类互不相交，且全体同余类的并集为 $\mathbb{Z}$。
- **代表元的选取**：每个同余类有无数个代表元，通常选取 $0, 1, \dots, m-1$ 作为标准代表元。
- **运算的良定性**：在 $Z_{m}$ 上可定义加法和乘法：

$$
\bar{a} + \bar{b} = \overline{a + b}, \quad \bar{a} \cdot \bar{b} = \overline{a \cdot b}
$$

> 这些运算与代表元的选取无关（良定义）
- **可逆元**：在 $\mathbb{Z}_{m}$ 中，$a$ 可逆当且仅当 $gcd(a, m) = 1$，可逆元的格式为 $\phi(m)$。

### 剩余类

#### 定义

设 $m \in \mathbb{Z^{+}}$，若一个大小为 $m$ 的整数集合 $\{ a_{1}, a_{2}, \dots, a_{m} \}$ 其元素模 $m$ 两两不同余，则称之为模 $m$ 的一个**完全剩余系**，通常取  $\{ 0, 1, \dots, m-1 \}$ 作为标准完全剩余系。

若一组整数 $b_{1}, b_{2}, \dots, b_{\phi(m)}$ 满足：

- 对每个 $b_{i}$，有 $gcd(b_{i}, m) = 1$
- 当 $i \neq j$ 时，$b_{i} \neq b_{j} \pmod m$

则称 $\{ b_{1}, b_{2}, \dots, b_{\phi(m)} \}$ 为模 $m$ 的一个**简化剩余系**（或既约剩余系）。

#### 基本性质

设 $m \in \mathbb{Z^{+}}, a_{i}, b_{i}, c, k \in \mathbb{Z}$

- **平移不变（完全）**：若 $\{ a_{1}, a_{2}, \dots, a_{m} \}$ 是模 $m$ 的完全剩余系，则对任意则合格你数 $c$，$\{ a_{1} + c, a_{2} + c, \dots, a_{m} + c \}$ 也是
- **数乘不变（完全）**：若 $\{ a_{1}, a_{2}, \dots, a_{m} \}$ 是模 $m$ 的完全剩余系且 $gcd(k, m) = 1$，则 $\{ ka_{1}, ka_{2}, \dots, ka_{m} \}$ 也是
- **数乘不变（简化）**：若 $\{ b_{1}, b_{2}, \dots, b_{\phi(m)} \}$ 是模 $m$ 的简化剩余系且 $gcd(k, m) = 1$，则 $\{ kb_{1}, kb_{2}, \dots, kb_{\phi(m)} \}$ 也是
- **威尔逊定理**：若 $p$ 为质数，则 $1, 2, \dots, p-1$ 构成模 $p$ 的简化剩余系，且有：

$$
(p - 1) ! \equiv -1 \pmod p
$$

### 欧拉定理

#### 定理（欧拉定理）

若 $a, n \in \mathbb{Z^{+}}$，且 $gcd(a, n) = 1$，则：

$$
a^{\phi(n)} \equiv 1 \pmod n
$$

#### 定理（费马小定理）

若 $a, p \in \mathbb{Z^{+}}$ 且 $p$ 是质数，则：

$$
a^{p-1} \equiv 1 \pmod p
$$

> 费马小定理是欧拉定理的素数特殊情况。

#### 费马小定理推论

若 $a, p \in \mathbb{Z^{+}}$，且 $p$ 是质数，则：

$$
a^{-1} \equiv a^{p-2} \pmod p
$$

因此，可以用快速幂求模质数下的逆元。

#### 证明

设与 $n$ 互质的所有正整数为 $r_{1}, r_{2}, \dots, r_{\phi(n)}$，它们构成模 $n$ 的一个简化剩余系。由于 $gcd(a, n) = 1$，当这些数分别乘以 $a$ 后，得到的数：

$$
ar_{1}, ar_{2}, \dots, ar_{\phi(n)}
$$

在模 $n$ 意义下仍然构成一个简化剩余系（即它们模 $n$ 两两不同余，且每个数仍然与 $n$ 互质）。因此，这两个集合在模 $n$ 意义下是相同的，于是就有：

$$
\prod_{i=1}^{\phi(n)}(r_{i}) \equiv \prod_{i=1}^{\phi(n)}(ar_{i}) \pmod n \implies \prod_{i=1}^{\phi(n)}(r_{i}) \equiv a^{\phi(n)} \prod_{i=1}^{\phi(n)}(r_{i}) \pmod n
$$

每个 $r_{i}$ 与 $n$ 互质，所以 $\prod_{i=1}^{\phi(n)}(r_{i})$ 也与 $n$ 互质，两边同时约去，得到 $a^{\phi(n)} \equiv a \pmod n$

### 扩展欧拉定理

#### 定理

设 $a, m$ 为正整数，$b$ 为非负整数，则：

- 若 $gcd(a, m) = 1$，则：
- 
$$
a^{b} \equiv a^{n \bmod \phi(m) } \pmod{m}
$$

- 若 $gcd(a, m) \neq 1$ 且 $b < \phi(m)$，则直接计算 $a^{k} \pmod m$
- 若 $gcd(a, m) \neq 1$ 且 $b \geq \phi(m)$，则：

$$
a^{b} \equiv a^{ b \bmod \phi(m) + \phi(m)} \pmod m
$$
