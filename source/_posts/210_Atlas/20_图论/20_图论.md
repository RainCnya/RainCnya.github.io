# [L2] 20_图论

> **核心哲学**：图论不是在画点和线，而是在建立一套“状态”与“转换”的规则。

## 1. 知识图谱与演化

- **核心工具**: 最短路建模、约束转化、流量平衡、结构压缩。
    
- **演化路径**:
	    
    1. **连通与遍历**：能不能走通？代价是多少？
        
    2. **最短路径**：在能走通的基础上，寻找代价最小或收益最高的走法。
        
    3. **资源分配**：从“一个人走”演变为“一堆资源流过”，解决有上限限制的冲突。
        
    4. **拓扑降维**：利用缩点和树形处理，把复杂的图简化成简单的树形或线形。

## 2. 决策矩阵

- **题目涉及“代价最小”、“距离最近”**: 优先选最短路。如果涉及变量之间的不等式约束（如 $x_i - x_j \le k$），采用 [[差分约束]] 建模。
    
- **题目涉及“资源分配”、“任务匹配”且存在上限冲突**: 考虑 [[网络流]]。如果是简单的配对问题，用 [[二分图匹配]]；如果涉及具体的成本，用 [[费用流]]。
    
- **题目要求处理“必经点”、“必经边”或者环的结构**: 采用连通性分析。通过[[Tarjan SCC 与缩点]] 将有环图变成无环图，或者用 [[圆方树]] 处理仙人掌图。
    
- **状态值域极大但余数有限**: 采用 [[同余最短路]]。
    

## 3. 核心板块索引

### 3.1 [L3] 21 图论基础

- [[存图与遍历]]：如何存图，如何遍历。
	
- [[最短路体系]]: Dijkstra (P1) 与 SPFA (P2) 的博弈。
    
- [[Note] 差分约束]: 逻辑不等式向物理边的翻译。
    
- [[Note] 同余最短路]: 剩余系空间下的到达性。

- [[Note] K 短路]: 寻找代价前 K 小的路径集合。
    

### 3.2 [L3] 22 树论专题

- [[Note] LCA]: 寻找最近公共祖先，倍增/树链剖分。

- [[Note] 生成树]: Kruskal 与矩阵树定理。

- [[Note] 点分治]: 处理大规模路径统计的分治法。
         
- [[Note] 树的直径与重心]: 识别树的物理核心与最长骨架。
	

### 3.3 [L3] 23 连通性专题

- [[Note] 有向图连通性]: SCC 缩点与 2-SAT。
    
- [[Note] 无向图连通性]: BCC 点双边双与圆方树。

- [[Note] SCC 缩点]: 将互相可达的点集视为超级节点。
	    
- [[Note] 支配树]: 分析必经点与节点间的支配层级。
	    
- [[Note] 圆方树]: 将环路结构投影到树上处理。

### 3.4 [L3] 24 匹配专题

- [[Note] 二分图基础]: 染色法判定与匈牙利算法。
    
- [[Note] 一般图匹配]: 带花树算法。
	
- [[Note] KM 算法]: 解决带有权值的完美匹配。
    

### 3.5 [L3] 25 网络流专题

- [[Note] 最大流与最小割]: 核心逻辑是“最大流量受限于最小代价的切断”。
    
- [[Note] 最小割建模]: 解决“二者选其一”的经典工具。

- [[Note] 费用流]: 在保证吞吐量最大的前提下，寻找成本最低的输送路径。
    

## 4. 依赖与规划

- **前置依赖**: 链式前向星、Dijkstra、拓扑排序。
    
- **学习路径**:
    
    - **P1 底座**：基础最短路 + LCA + 树的直径。
        
    - **P2 中坚**：SCC 缩点 + 二分图匹配 + 最大流。
        
    - **P3 进阶**：圆方树 + 费用流建模。
