算法学习笔记：差分约束系统

一、 初步认知

领域：图论 (Graph Theory)
子领域：领域应用 / 线性规划特例
典型设问：给定一组形如 xi−xj≤ckx_i - x_j \le c_kxi​−xj​≤ck​ 的不等式，判断是否有解，或求出一组满足条件的解（通常是最大值或最小值）。

差分约束系统 是线性规划的一种特殊形式。虽然本质是代数问题，但由于其不等式形式与最短路算法中的三角不等式 (dist[v]≤dist[u]+wdist[v] \le dist[u] + wdist[v]≤dist[u]+w) 高度一致，因此常转化为图论问题求解。

二、 主题渗透

1. The Struggle (模拟推演)

核心转化：

  • 代数视角：xi−xj≤wx_i - x_j \le wxi​−xj​≤w
  • 移项变形：xi≤xj+wx_i \le x_j + wxi​≤xj​+w
  • 图论视角：存在一条边 j→wij \xrightarrow{w} ijw​i，表示 iii 的值受限于 jjj 的值加 www。
▶
模拟推演 (负环与矛盾)

场景：

  1. xB−xA≤3  ⟹  xB≤xA+3x_B - x_A \le 3 \implies x_B \le x_A + 3xB​−xA​≤3⟹xB​≤xA​+3 (Edge: A→3BA \xrightarrow{3} BA3​B)
  2. xC−xB≤−5  ⟹  xC≤xB−5x_C - x_B \le -5 \implies x_C \le x_B - 5xC​−xB​≤−5⟹xC​≤xB​−5 (Edge: B→−5CB \xrightarrow{-5} CB−5​C)
  3. xA−xC≤1  ⟹  xA≤xC+1x_A - x_C \le 1 \implies x_A \le x_C + 1xA​−xC​≤1⟹xA​≤xC​+1 (Edge: C→1AC \xrightarrow{1} AC1​A)

图论分析：
构成了环 A→B→C→AA \to B \to C \to AA→B→C→A。
环的总权值：3+(−5)+1=−13 + (-5) + 1 = -13+(−5)+1=−1。这是一个负环。

代数矛盾：
将三个不等式相加：
(xB−xA)+(xC−xB)+(xA−xC)≤3−5+1(x_B - x_A) + (x_C - x_B) + (x_A - x_C) \le 3 - 5 + 1(xB​−xA​)+(xC​−xB​)+(xA​−xC​)≤3−5+1
0≤−10 \le -10≤−1
结论：逻辑矛盾。因此，图中存在负环   ⟺  \iff⟺ 不等式组无解。

2. Hack 思维 (Generator 视角)

  • 图不连通：不等式组可能只约束了局部变量（如 x1,x2x_1, x_2x1​,x2​ 相关，x3,x4x_3, x_4x3​,x4​ 相关，但两组之间无限制）。
    • 应对：建立超级源点 0，向所有 1…n1 \dots n1…n 连一条权值为 0 的边（xi≤x0+0x_i \le x_0 + 0xi​≤x0​+0），从 0 开始跑 SPFA。或者简单地将所有点初始入队。
  • 最大值 vs 最小值：
    • 求 xix_ixi​ 的最大值 →\to→ 转化为 xi≤…x_i \le \dotsxi​≤… (上界) →\to→ 最短路。
    • 求 xix_ixi​ 的最小值 →\to→ 转化为 xi≥…x_i \ge \dotsxi​≥… (下界) →\to→ 最长路。

三、 核心思想

"代数几何化 (Mapping Inequalities to Edges)"

利用最短路的三角不等式性质，将约束条件转化为边权，将逻辑矛盾转化为负权环。

2. 部署细节

  • 不等式标准化：

    • A−B≥C  ⟺  B−A≤−C  ⟹  A - B \ge C \iff B - A \le -C \impliesA−B≥C⟺B−A≤−C⟹ add(A, B, -C)

    • A−B≤C  ⟹  A - B \le C \impliesA−B≤C⟹ add(B, A, C)

    • A=B  ⟺  A−B≤0∧B−A≤0  ⟹  A = B \iff A - B \le 0 \land B - A \le 0 \impliesA=B⟺A−B≤0∧B−A≤0⟹ add(B, A, 0), add(A, B, 0)

  • 正解构造：SPFA 跑完后，dist[i] 就是一组满足条件的可行解（通常是满足约束下的最大值，相对于超级源点）。
