[Leaf] [CF2183C] War Strategy

CF2183C War Strategy

1. 题面梗概

一条有 nnn 个基地的线段，你的基地在第 kkk 个节点，每天你的基地都会刷新一个人。

每天你可以选择一个基地，以及对应基地的任意数量的士兵，向左或者向右一个基地移动。

求 mmm 天后，你的士兵最多覆盖多少个基地。

2. 逻辑推导

首先最终 mmm 天后，驻扎士兵的基地一定是一个包含 kkk 的区间，证明略。

那么我们假设这段区间为 [k−a,k+b][k-a,k+b][k−a,k+b]。 0≤a≤k−1,0≤b≤n−k0 \leq a \leq k-1, 0 \leq b \leq n - k0≤a≤k−1,0≤b≤n−k 。

所以，如果我们前 aaa 天不动，那么第 aaa 天大本营就有 aaa 个士兵。
然后我们把 aaa 个士兵移到 k−1k-1k−1， a−1a-1a−1 个士兵移到 k−2k-2k−2。以此类推。
接着再把 bbb 个士兵向右移动，可能要等，也可能不等。

因为我们至少需要 a+ba + ba+b 天来获得额外的 a+ba + ba+b 个士兵。
综上至少需要 a + b + max( a, b ) - 1; 天。

我们考虑贪心求解。如果 k−1<n−kk - 1 < n - kk−1<n−k，我们把 k→n−k+1k \to n - k + 1k→n−k+1。
我们定义两个变量a = 0 | b = 0，然后不断轮流增加a | b，统计最大值答案即可。
