P3199 最小圈

Luogu-P3199 / TZOJ-6544[HNOI2009] 最小圈

题目简述

难度：提高+(26822) | 题解格式：V6.3

给定一个 nnn 个点 mmm 条边的带权有向图，求图中所有环的最小平均权值（即环上边权之和除以环的边数）。

  • 数据范围：n≤3000,m≤10000n \le 3000, m \le 10000n≤3000,m≤10000，∣w∣≤107|w| \le 10^7∣w∣≤107。

解题思路

1. 初步认知

  • 直观想法：直接搜索所有的环并计算平均值。但环的数量可能是指数级的，显然不可行。
  • 瓶颈分析：我们需要一种方法来判定“是否存在一个平均值小于 XXX 的环”，而不是直接求解。这提示我们使用二分答案。

2. 主题渗透 (正解)

算法：0/1 分数规划 + DFS-SPFA (O(K⋅N⋅M)O(K \cdot N \cdot M)O(K⋅N⋅M)，其中 KKK 为二分次数)

  • 基本概念：将“求最优比率”转化为“判定性问题”。
  • 数学推导：
    假设我们要判定是否存在一个环 CCC，其平均值小于 λ\lambdaλ：

    ∑e∈Cwe∣C∣<λ\frac{\sum_{e \in C} w_e}{|C|} < \lambda ∣C∣∑e∈C​we​​<λ

    变形为：

    ∑e∈Cwe<λ⋅∣C∣\sum_{e \in C} w_e < \lambda \cdot |C| e∈C∑​we​<λ⋅∣C∣

    ∑e∈C(we−λ)<0\sum_{e \in C} (w_e - \lambda) < 0 e∈C∑​(we​−λ)<0

    结论：如果将图中每条边的权值 www 减去 λ\lambdaλ，若新图中存在负环，则说明存在平均值小于 λ\lambdaλ 的环。
  • 实现细节：
    • 二分：对答案 λ\lambdaλ 进行实数二分。
    • 判负环：使用 DFS 版 SPFA。相比 BFS 版，DFS 一旦进入负环就能通过递归栈快速发现，非常适合二分检测的场景。

#include<bits/stdc++.h>using namespace std;using ll = long long;using PID = pair<int, double>;const int maxn = 3000 + 5;const int inf = 1e9;vector<PID> adj[maxn];double dis[maxn];bool vis[maxn]; // 记录当前递归栈中是否包含该节点 (instack)int n, m;// DFS-SPFA 判负环// u: 当前节点, lim: 当前二分的平均值 lambdabool dfs( int u, double lim ){    vis[u] = 1; // 标记 u 在当前递归栈中        for( auto [v, w] : adj[u] )    {        // 【核心推导】        // 判定是否存在 sum(w) / k < lim        // <=> sum(w) < k * lim        // <=> sum(w - lim) < 0        // 所以将边权视为 w - lim，检查是否存在负环        w -= lim;                 if( dis[u] + w < dis[v] )        {            dis[v] = dis[u] + w;                        // 如果 v 已经在递归栈中，说明绕回了祖先，且路径权值和更小 -> 发现负环            if( vis[v] ) return 1;                        // 继续深搜            if( dfs( v, lim ) ) return 1;        }    }    vis[u] = 0; // 回溯：离开当前递归栈    return 0;}// check 函数：检查是否存在平均值 < lim 的环bool check( double lim ){    // SPFA 初始化    // 对于判负环，dis 初始值不重要，设为 0 即可    for( int i = 1; i <= n; ++ i )    {        dis[i] = 0;        vis[i] = 0;    }    // 图可能不连通，需要遍历所有点尝试作为起点    // (一般为了效率可以加个全局 visited 数组优化，但这题数据不卡)    for( int i = 1; i <= n; ++ i )    {        if( dfs( i, lim ) ) return 1;    }    return 0;}int main( ){    // IO 优化    cin >> n >> m;    for( int i = 1; i <= m; ++ i )    {        int u, v;        double w;        cin >> u >> v >> w;        adj[u].push_back({ v, w });    }    // 二分答案范围：边权范围是 [-1e7, 1e7]    double l = -1e7, r = 1e7;    double ans = 0;    // 实数二分固定循环次数 (60~100次) 以保证精度并避免死循环    for( int k = 1; k <= 100; ++ k )    {        double mid = ( l + r ) / 2.0;        if( check( mid ) )         {            // 存在负环 => 存在平均值 < mid 的环            // 说明最小平均值可能更小，向左逼近            ans = mid;             r = mid;        }        else         {            // 不存在负环 => 所有环平均值 >= mid            l = mid;        }    }    cout << fixed << setprecision( 8 )  << ans << '\n';    return 0;}

四、 结语

本题是 0/1 分数规划 在图论中的经典应用。

难点在于将代数不等式 ∑(wi−λ)<0\sum (w_i - \lambda) < 0∑(wi​−λ)<0 转化为图论中的 负环判定 模型。

同时，DFS-SPFA 虽然求最短路不稳定，

但在这种只需“判定存在性”的场景下，效率远高于 BFS 版本。
