算法学习笔记：图论/最短路 - 体系综述

一、 初步认知

  • 领域层级：
    • 父领域： 图论 (Graph Theory)
    • 子领域： 最短路 (Shortest Path)
  • 命名： 最短路径算法体系 (Shortest Path Algorithm System)
  • 定义： 在加权图中，寻找从一个节点到另一个节点总权值最小的路径。
  • 两大分支：
    • 单源最短路 (SSSP)： 求一个起点到其他所有点的距离。（代表：Dijkstra, SPFA）
    • 多源最短路 (APSP)： 求任意两点之间的距离。（代表：Floyd-Warshall）

二、 主题渗透

1. 核心机制：松弛 (Relaxation)

  • 原理： 所有最短路算法的基石。通过一个中间节点 uuu 来检查是否能缩短 uuu 的邻居 vvv 的距离。
  • 公式： if (dis[v] > dis[u] + w) dis[v] = dis[u] + w;
  • 算法差异： 算法本质差异在于决定松弛的顺序和松弛的频率。

2. 算法选型策略 (Decision Tree)

场景特征 推荐算法 核心机制 最坏复杂度
单源、非负权边 Dijkstra (堆优化) 贪心 + 优先队列 O(Mlog⁡N)O(M \log N)O(MlogN)
单源、有负权边 SPFA / Bellman-Ford 队列优化 + 迭代松弛 O(NM)O(NM)O(NM)
多源、小图 (N≤500N \le 500N≤500) Floyd-Warshall 动态规划 O(N3)O(N^3)O(N3)
单源、无权图 BFS 队列 + 层次遍历 O(N+M)O(N+M)O(N+M)

三、 核心思想

  • 边松弛 (Edge Relaxation)：利用三角不等式 (dis[v]≤dis[u]+wdis[v] \le dis[u] + wdis[v]≤dis[u]+w) 不断逼近真实的最短距离。
  • 最优子结构 (Optimal Substructure)：最短路径上的子路径也是最短路径。

四、 实战部署

1. 选型指南

  • 遇到负权边： 仅限 SPFA/Bellman-Ford。
  • 追求速度 (非负权)： 优先 Dijkstra。
  • 处理闭包关系： Floyd-Warshall (常用于传递闭包)。

2. 通用初始化

  • 距离数组： 使用 long long dis[N]，初始值设为 101810^{18}1018 级别的 INF。
  • 自环处理： 邻接矩阵初始化时 d[i][i]=0d[i][i]=0d[i][i]=0。

六、 领域拓展

  • 负环检测： SPFA 的副作用，可用于判断是否存在套利空间或矛盾约束（差分约束系统）。
  • 差分约束： 将形如 xi−xj≤kx_i - x_j \le kxi​−xj​≤k 的不等式组转化为最短路问题求解。
