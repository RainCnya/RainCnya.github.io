## 1. 生态位

* **定义**: 管理元素所属集合的数据结构，支持**合并 (Union)** 与 **查询 (Find)**。
* **解决痛点**:
    * **动态连通性**: 判断图中两点是否连通 (Kruskal)。
    * **关系传递**: 朋友的朋友是朋友，敌人的敌人是朋友 (扩展域)。
    * **离线动态图**: 处理删边/删点问题。

## 2. 逻辑支点

* **路径压缩 (Path Compression)**: 查询时直接将节点挂到祖先下，均摊复杂度 $\alpha(N)$ 。
* **时光倒流 (Time Reversal)**:
    * 问题: 图中不断**删除**点/边，查询连通块数量。
    * 转化: 并查集只支持合并，不支持分裂。**将操作序列反过来**，变成初始为空(或终态)，不断**添加**点/边。
* **扩展域 (Extended Domain)**:
    * 将每个点 $x$ 拆分为 $x_{friend}$ 和 $x_{enemy}$ (通常用 $x$ 和 $x+N$ 表示)。
    * 逻辑: "x 与 y 是敌人" $\iff$ ``merge(x, y+N)` 和 ``merge(y, x+N)` [cite: 3679]。

## 3. 实战部署

### 3.1 扩展域并查集 (P1525)

* **场景**: 关押罪犯。仇恨值大的必须分开。
* **策略**: 按仇恨值从大到小排序。对于仇恨 $(u, v)$，尝试把 $u$ 和 $v$ 分开关押。如果发现 $u$ 和 $v$ 已经在同一集合（不可避免地在一起），则该仇恨值为答案。

{% fold info @Code: Extended DSU %}
```cpp
// x: self, x+n: enemy
// Rule: Enemy of Enemy is Friend
sort( edges, edges + m ); // 降序

for( auto e : edges ) 
{
    int u = e.u, v = e.v;
    if( find(u) == find(v) ) 
    {
        cout << e.w << '\n'; // 冲突，输出最大怨气值
        return;
    }
    // u 和 v 分开 -> u 和 v的敌人是一伙
    merge( u, v + n );
    merge( v, u + n );
}
cout << 0 << '\n';
```

{% endfold %}

### 3.2 离线 + 时光倒流 (P1197)

- **场景**: 星球大战。依次摧毁星球，求连通块数。
    
- **策略**: 先标记所有被摧毁的星球，计算**最终状态**的连通块数。然后逆序恢复星球，每恢复一个，连通块数可能减少 9。
