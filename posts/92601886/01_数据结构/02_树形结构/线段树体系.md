## 1. 生态位

* **定义**: 基于**二分**思想，将序列划分为 $O(\log N)$ 层树形结构，支持复杂的区间修改与查询。
* **解决痛点**:
    * **复杂区间操作**: 乘法、加法、开方、最值等。
    * **信息合并**: 最大子段和、最长连续 0/1 等需要左右子区间拼接的信息。

## 2. 逻辑支点

* **懒标记 (Lazy Tag)**:
    * **延迟更新**: 标记只打在 $O(\log N)$ 个极大区间上，仅在访问子节点时下传 (`push_down`)。
    * **优先级**: 当存在加法与乘法标记时，**先乘后加** ($val \times mul + add$) 以维护运算律。
* **信息上浮 (Push Up)**:
    * 核心在于如何由 `ls` 和 `rs` 的信息合并出 `p` 的信息。
    * **GSS 模型 (最大子段和)**: 
        $$MaxSum(p) = \max( MaxSum(ls), MaxSum(rs), MaxSuf(ls) + MaxPre(rs) )$$ 。

## 3. 实战部署

### 3.1 乘加混合模板 (P3373)

{% fold info @Code: Mul & Add Tag %}
```cpp
// 维护 sum, mul_tag, add_tag
void push_down( int p, int l, int r ) 
{
    int mid = (l + r) >> 1;
    // 1. Apply Mul Tag
    if( mul[p] != 1 ) 
    {
        mul[ls(p)] = mul[ls(p)] * mul[p] % mod;
        add[ls(p)] = add[ls(p)] * mul[p] % mod;
        sum[ls(p)] = sum[ls(p)] * mul[p] % mod;
        // ... same for rs ...
        mul[p] = 1;
    }
    // 2. Apply Add Tag
    if( add[p] ) 
    {
        add[ls(p)] = (add[ls(p)] + add[p]) % mod;
        sum[ls(p)] = (sum[ls(p)] + add[p] * (mid - l + 1)) % mod;
        // ... same for rs ...
        add[p] = 0;
    }
}
```

{% endfold %}

### 3.2 区间最大子段和 (GSS / P4513)

- **策略**: 维护四个值：`sum` (区间和), `lmax` (最大前缀), `rmax` (最大后缀), `dat` (最大子段和) 4。
    

{% fold info @Code: GSS PushUp %}

```cpp
struct Node { int sum, lmax, rmax, dat; };

Node merge( Node L, Node R ) 
{
    Node res;
    res.sum = L.sum + R.sum;
    res.lmax = max( L.lmax, L.sum + R.lmax );
    res.rmax = max( R.rmax, R.sum + L.rmax );
    res.dat = max( max( L.dat, R.dat ), L.rmax + R.lmax );
    return res;
}
```

{% endfold %}

- **势能分析线段树**:
    
    - **区间开方 (P4145)**: 只有当 $val > 1$ 时才递归修改。每个数最多被开方 6 次 5。
