## 1. 生态位

* **定义**: 利用 $x \& -x$ (Lowbit) 性质，将前缀和拆分为 $O(\log N)$ 个子区间的数据结构。
* **解决痛点**:
    * **动态前缀和**: 单点修改 + 前缀查询，常数远小于线段树。
    * **逆序对**: 配合离散化，$O(N \log N)$ 解决逆序对计数 。
* **局限**: 维护的信息必须满足**结合律**且**可差分** (如和、异或)，难以维护区间最值。

## 2. 逻辑支点

* **Lowbit 拆分**:
    * $c[x]$ 维护的区间是 $(x - \text{lowbit}(x), x]$ 。
    * **父子关系**: 树形结构中 $x$ 的父节点是 $x + \text{lowbit}(x)$ 。
* **差分转化**:
    * **区间修改 + 单点查询**: 维护原数组的**差分数组** $d[i] = a[i] - a[i-1]$。
    * 原数组 $a[x] = \sum_{i=1}^x d[i]$，即差分数组的前缀和。

## 3. 实战部署

### 3.1 基础模板 (P3374)

{% fold info @Code: BIT Standard %}
```cpp
int c[maxn];
int lowbit( int x ) { return x & -x; }

// 单点修改: a[x] += v
void add( int x, int v ) 
{
    for( ; x <= n; x += lowbit(x) ) 
        c[x] += v;
}

// 前缀查询: sum(1...x)
int ask( int x ) 
{
    int res = 0;
    for( ; x; x -= lowbit(x) ) 
        res += c[x];
    return res;
}
```

{% endfold %}

### 3.2 逆序对 (P1908)

- **策略**: 离散化 -> 从左往右扫 -> 统计比当前数大的已出现个数 -> 加入当前数。
    

{% fold info @Code: Inversion Pair %}

```cpp
// a[] 已离散化为 1...m
for( int i = 1; i <= n; ++ i ) 
{
    add( a[i], 1 );
    // 总数 i 减去 小于等于 a[i] 的数量 = 大于 a[i] 的数量
    ans += i - ask( a[i] ); 
}
```

{% endfold %}
