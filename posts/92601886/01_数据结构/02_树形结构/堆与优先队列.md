## 1. 生态位

* **定义**: 一棵完全二叉树，满足父节点权值优于子节点 (大根/小根)。
* **解决痛点**:
    * **动态极值**: $O(\log N)$ 插入与删除， $O(1)$ 查询最值。
    * **贪心引擎**: 如 Huffman 编码 (合并果子)、Dijkstra。
    * **动态中位数**: 利用对顶堆实时维护第 $K$ 大元素。

## 2. 逻辑支点

* **Structure Invariant**: 
    * `priority_queue` 默认为**大根堆**。
    * 小根堆写法: `priority_queue<int, vector<int>, greater<int>>` 。
* **对顶堆 (Dual Heap)**:
    * 维护一个大根堆 `L` (存较小的一半) 和一个小根堆 `R` (存较大的一半)。
    * **平衡**: 保持 `L.size() == R.size()` 或 `L.size() == R.size() + 1`。
    * **中位数**: `L.top()` 即为中位数。

## 3. 实战部署

### 3.1 对顶堆求中位数 (P1168)

* **策略**: 动态维护两个堆，每插入一个数，根据大小推入对应堆，并调整平衡。

{% fold info @Code: Running Median %}
```cpp
priority_queue<int> L; // Max Heap
priority_queue<int, vector<int>, greater<int>> R; // Min Heap

void add( int x ) 
{
    if( L.empty() || x <= L.top() ) L.push( x );
    else R.push( x );
    
    // Rebalance: L can have at most 1 more element than R
    if( L.size() > R.size() + 1 ) 
    { 
        R.push( L.top() ); L.pop(); 
    }
    if( R.size() > L.size() ) 
    { 
        L.push( R.top() ); R.pop(); 
    }
}
```

{% endfold %}

### 3.2 多路归并 (P1631)

- **场景**: $N$ 个有序序列，求前 $N$ 小的元素和。
    
- **策略**: 初始将所有序列的第一个元素入堆，取出最小的后，将该元素所在序列的“下一个”入堆。
