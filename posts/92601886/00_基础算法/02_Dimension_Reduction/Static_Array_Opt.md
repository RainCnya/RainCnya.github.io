## 1. 生态位

* **定义**: 处理**静态数组**区间查询与修改的“对偶技巧”。
* **前置依赖**: [容斥原理]
* **解决痛点**: 
    * **前缀和**: $O(1)$ 查询静态区间和。
    * **差分**: $O(1)$ 执行区间修改。
    * **SOS DP**: 解决子集求和 (Sum Over Subsets) 问题。

## 2. 逻辑支点

* **对偶性**: 差分是前缀和的**逆运算**。
    * $D[i] = A[i] - A[i-1] \iff A[i] = \sum D[j]$。
* **维度压缩**:
    * 将“区间操作”压缩为“端点操作”。

### 3.2 高维前缀和 (SOS DP) 模板

* **定义**: 求 $F[mask] = \sum_{i \subseteq mask} A[i]$。
* **策略**: **逐维处理**。将 $N$ 维空间的容斥转化为 $N$ 次线性扫描。

{% fold info @Code: SOS DP %}
```cpp
// dp[mask] 初始存 A[mask]
for( int i = 0; i < n; ++ i ) // 枚举维度 (第 i 位)
{
    for( int mask = 0; mask < (1 << n); ++ mask ) 
    {
        // 如果 mask 包含第 i 维 (即第 i 位为 1)
        if( (mask >> i) & 1 ) 
        {
            // 累加 "第 i 位为 0" 的对应状态
            dp[mask] += dp[mask ^ (1 << i)];
        }
    }
}
```

{% endfold %}

### ⚠️ 边界与 Hack

- **[坑点]**: SOS DP 的外层循环必须是**枚举维度** `i`，内层枚举状态 `mask`。顺序反了就变成了普通的逻辑运算，无法传递状态。
