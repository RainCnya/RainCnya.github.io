## 1. 生态位

* **定义**: 利用序列的**单调性**，通过两个指针的**同向**或**相向**移动，将 $O(N^2)$ 降维至 $O(N)$。
* **前置依赖**: [排序], [前缀和]
* **解决痛点**: 
    * **区间搜索**: 寻找满足条件的最短/最长连续子区间。
    * **组合优化**: 两个有序序列的匹配或合并。
* **竞品对比**:
    * vs **二分**: 双指针是动态的 $O(N)$，二分是静态的 $O(N \log N)$。

## 2. 逻辑支点

* **核心直觉**: **"Never Turn Back" (绝不回头)**。
    * 当 $R$ 右移导致状态变更时，$L$ 只需要跟随右移，无需回退。
* **物理模型**: **滑动窗口**的伸缩。
    * **进窗 (Expand)**: $R$ 主动摄入。
    * **出窗 (Shrink)**: $L$ 被动排出以维持合法性。

## 3. 实战部署

### 3.1 尺取法模板 (同向双指针)

* **策略**: 寻找和 $\ge K$ 的**最短**子数组。

{% fold info @Code: Sliding Window %}
```cpp
int l = 1, sum = 0, ans = n + 1;
for( int r = 1; r <= n; ++ r ) 
{
    sum += a[r]; 
    while( sum >= K ) 
    {
        ans = min( ans, r - l + 1 );
        sum -= a[l]; 
        l ++; 
    }
}
```

{% endfold %}

### ⚠️ 边界与 Hack

- **[坑点 1]**: 若数组含**负数**，单调性失效，需改用前缀和+单调队列。
    
- **[坑点 2]**: 求“最长”合法区间时，逻辑通常是 `while(invalid) L++`。
    

## 4. 知识粘附

- **变体模型**:
    
    - **前后缀分解 + 双指针**: (P3143) 选取两个不重叠区间使和最大。
        
        - 预处理 `pre[i]` (以 i 结尾的最优解) 和 `suf[i]` (以 i 开头的最优解)。
            
        - 答案为 $\max(pre[i] + suf[i+1])$。其中 `pre/suf` 的计算利用双指针维护。
            
    - **相向双指针**: 2-Sum 问题，或接雨水问题。
        
- **母题索引**:
    
    - **[Luogu P1638]**: 逛画展 - 尺取法
