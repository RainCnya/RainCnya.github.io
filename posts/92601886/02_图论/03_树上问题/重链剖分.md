## 1. 生态位

* **定义**: 将边分为**重边**和**轻边**，将树剖分为 $O(\log N)$ 条链，使其“线性化”。
* **解决痛点**:
    * **树上路径修改**: 结合线段树，维护路径上的点权/边权。
    * **LCA**: 相比倍增，常数更小，且支持修改。

## 2. 逻辑支点

* **重儿子**: 子树大小 (`size`) 最大的子节点。
* **重链连续性**: 重链上的 DFS 序是**连续**的。这是线段树能维护区间的根本原因。
* **跳链逻辑**: 
    * 若 $u, v$ 不在同一重链，选择链头 (`top`) 深度较深的点，跳到 `top` 的父节点。
    * 循环直到二者在同一条链。

## 3. 实战部署

{% fold info @Code: HLD Template %}
```cpp
// DFS 1: sz, dep, fa, son
void dfs1( int u, int f, int d )
{
    dep[u] = d; fa[u] = f; sz[u] = 1;
    int max_sz = -1;
    for( int v : adj[u] )
    {
        if( v == f ) continue;
        dfs1( v, u, d + 1 );
        sz[u] += sz[v];
        if( sz[v] > max_sz ) { max_sz = sz[v]; son[u] = v; }
    }
}

// DFS 2: top, dfn (Priority: Heavy Son)
void dfs2( int u, int tp )
{
    top[u] = tp; dfn[u] = ++ idx; 
    if( ! son[u] ) return;
    dfs2( son[u], tp ); // 重儿子优先
    for( int v : adj[u] )
    {
        if( v != fa[u] && v != son[u] ) dfs2( v, v );
    }
}

// LCA
int lca( int u, int v )
{
    while( top[u] != top[v] )
    {
        if( dep[ top[u] ] < dep[ top[v] ] ) swap( u, v );
        u = fa[ top[u] ];
    }
    return dep[u] < dep[v] ? u : v;
}
```
