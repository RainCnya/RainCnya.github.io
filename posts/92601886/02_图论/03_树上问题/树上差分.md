## 1. 生态位

* **定义**: 差分思想在树上的推广。
* **解决痛点**: **离线**批量处理路径上的区间修改（$+K$），最后一次性查询。

## 2. 逻辑支点

* **影响的抵消**: 在路径两端施加影响，在 **LCA** 处消除影响。
* **点差分**:
    * 修改 $u \to v$ 的点权。
    * 操作: `cnt[u]++`, `cnt[v]++`, `cnt[lca]--`, `cnt[fa[lca]]--`。
* **边差分**:
    * 修改 $u \to v$ 的边权（下放给深度深的节点）。
    * 操作: `cnt[u]++`, `cnt[v]++`, `cnt[lca]-=2`。

## 3. 实战部署

{% fold info @Code: Tree Difference %}
```cpp
// 点差分示例: diff[u]++
void add_path( int u, int v, int lca_node )
{
    diff[u] ++;
    diff[v] ++;
    diff[lca_node] --;
    diff[ fa[lca_node] ] --;
}

// 自底向上求前缀和
void dfs_sum( int u, int f )
{
    for( int v : adj[u] )
    {
        if( v == f ) continue;
        dfs_sum( v, u );
        diff[u] += diff[v];
    }
}
```
