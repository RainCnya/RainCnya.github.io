## 1. 生态位

* **树的直径**: 树上**最长**的简单路径。
    * **痛点**: 决定了树的“跨度”极限。常用于计算树上最远点对。

* **树的重心**: 删除该点后，最大子树的节点数最小的点。
    * **痛点**: 树的分治中心。点分治的核心，保证递归层数稳定在 $O(\log N)$。
## 2. 逻辑支点

* **直径的端点特质**: 从树上**任意**一点 $u$ 出发，能走到的最远节点 $v$，一定是直径的一个端点。
    * *算法*: 两次 DFS。第一次找端点 $u$，第二次找另一端点 $v$。

* **重心的平衡特质**: 重心 $G$ 满足 $\max_{v \in son(G)} \{ size(v), N - size(G) \}$ 最小。
    * *性质*: 以重心为根，所有子树大小 $\le N/2$。

## 3. 实战部署

{% fold info @Code: Tree Diameter & Centroid %}
```cpp
// 1. 直径 (Two-DFS)
int max_len = 0, endpoint = 0;
void dfs_diam( int u, int fa, int dist )
{
    if( dist > max_len ) { max_len = dist; endpoint = u; }
    for( auto &edge : adj[u] )
    {
        if( edge.v != fa ) dfs_diam( edge.v, u, dist + edge.w );
    }
}

// 2. 重心 (Centroid)
int sz[maxn], max_part[maxn]; 
int root = 0; // max_part[0] = INF

void get_root( int u, int fa, int total_n )
{
    sz[u] = 1; max_part[u] = 0;
    for( int v : adj[u] )
    {
        if( v == fa ) continue;
        get_root( v, u, total_n );
        sz[u] += sz[v];
        max_part[u] = max( max_part[u], sz[v] );
    }
    max_part[u] = max( max_part[u], total_n - sz[u] );
    if( max_part[u] < max_part[root] ) root = u;
}
```
