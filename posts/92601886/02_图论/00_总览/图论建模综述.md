## 1. 生态位

* **定义**: 将问题的 **“状态”映射为节点**，将 **“关系/限制”映射为边**。
* **物理图 vs 代数图**:
    * **物理图**: 节点代表位置，边代表通路。
    * **代数图**: 节点代表数值/状态，边代表转移逻辑/约束。
* **解决痛点**:
    * **复杂约束**: 不等式组 $\to$ 负环检测。
    * **无限状态**: 数轴 $\to$ 同余环。
    * **动态过程**: 时间 $\to$ 阶段 (Floyd) 或 层级 (分层图)。

## 2. 逻辑支点

* **三角不等式映射**:
    * $x_v - x_u \le w \iff x_v \le x_u + w$。对应边 $u \to v$，权值 $w$。（详见 **[差分约束系统](../01_最短路/差分约束系统.md)**）
* **同余折叠 (Modulo Folding)**:
    * 问题: 求 $a_1 x_1 + \dots = K$ 的可行性。
    * 转化: 选最小系数 $m$ 为模数，节点 $0 \dots m-1$ 构成环。
* **时间/阶段展开**:
    * 问题: 动态加点的最短路。
    * 转化: 利用 Floyd 最外层循环 $k$ 的物理意义（允许经过前 $k$ 个点）。

## 3. 实战部署

### 3.1 同余最短路模型 ([P2371](https://www.luogu.com.cn/problem/P2371))

{% fold info @Code: Modulo Shortest Path %}
```cpp
// m = a[1] (smallest coefficient)
// Nodes: 0..m-1
// dist[i]: 模 m 余 i 的数中，能凑出的最小值

void solve( ) 
{
    // ... Dijkstra Init ...
    while( ! q.empty( ) ) 
    {
        int u = q.top( ).second; q.pop( );
        if( vis[u] ) continue; 
        
        // Try all other coefficients as edges
        for( int i = 2; i <= n; ++ i ) 
        {
            int w = a[i];
            int v = ( u + w ) % m; // Modulo Transition
            if( dist[u] + w < dist[v] ) 
            {
                dist[v] = dist[u] + w;
                q.push( {dist[v], v} );
            }
        }
    }
}
```

{% endfold %}

### 3.2 动态时间轴 Floyd ([P1119](https://www.luogu.com.cn/problem/P1119))

{% fold info @Code: Incremental Floyd %}

```C++
// Queries sorted by time T
int k = 0;
for( auto& q : queries ) 
{
    // Unlock nodes up to time T
    while( k < n && node_time[k] <= q.T ) 
    {
        // Use k as the intermediate pivot
        for( int i = 0; i < n; ++ i )
        {
            for( int j = 0; j < n; ++ j )
            {
                dist[i][j] = min( dist[i][j], dist[i][k] + dist[k][j] );
            }
        }
        k ++;
    }
    // Answer query using current dist[][]
}
```
