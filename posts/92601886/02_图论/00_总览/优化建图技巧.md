## 1. 生态位

* **定义**: 通过引入**虚拟节点**或数据结构，将 $O(N^2)$ 的暴力连边降为 $O(N)$ 或 $O(N \log N)$。
* **解决痛点**: 集合连边、区间连边、位运算隐式图。

## 2. 逻辑支点

* **虚拟节点 (Hub)**:
    * 场景: 集合 $A$ 向集合 $B$ 连边。
    * 策略: 建立中转点 $O$，连边 $A \to O$ 和 $O \to B$。
* **线段树优化**:
    * 场景: 点向区间连边。
    * 策略: 建立入树和出树，叶子为实际点。点 $u$ 连向入树的区间节点。
* **隐式位运算图**:
    * 场景: 边权由 $u \oplus v$ 决定。
    * 策略: 仅当 $u, v$ 二进制差一位时连边，最短路即为逐位修正的过程。

## 3. 实战部署

{% fold info @Code: Virtual Node ([P1983](https://www.luogu.com.cn/problem/P1983)) %}
```cpp
// 虚拟汇点: 优化 "未停靠集合" -> "停靠集合" 的连边
void build_virtual( int train_id )
{
    int virtual_node = n + train_id; 
    for( int i = start_stat; i <= end_stat; ++ i )
    {
        if( ! is_stop[i] ) add_edge( i, virtual_node, 0 );
        else add_edge( virtual_node, i, 1 );
    }
}
```
