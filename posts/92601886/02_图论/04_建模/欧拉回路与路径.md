## 1. 生态位

* **定义**: 经过图中**每条边恰好一次**的路径。
* **解决痛点**: “一笔画”问题，DNA 序列重组，词语接龙。

## 2. 逻辑支点

* **度数不变量**:
    * **有向图回路**: 所有点 `in == out`。
    * **有向图路径**: 恰有 1 点 `out = in + 1` (起)，1 点 `in = out + 1` (终)。
* **Hierholzer 算法 (圈套圈)**:
    * 核心: **回溯时记录**。
    * 只有当一个点无路可走（所有关联边都已遍历）时，才将其压入栈。最后逆序输出。

## 3. 实战部署

{% fold info @Code: Hierholzer Vector %}
```cpp
// cur[u]: 当前弧优化，防止重复遍历边
void dfs( int u )
{
    while( cur[u] < adj[u].size( ) )
    {
        int v = adj[u][ cur[u] ];
        cur[u] ++; // 逻辑删边
        dfs( v );
    }
    ans.push( u ); // 回溯入栈
}

// 主逻辑: 判度数 -> 找起点 -> DFS -> 逆序输出
```

{% endfold %}
