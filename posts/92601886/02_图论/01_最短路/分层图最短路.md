## 1. 生态位

* **定义**: 将图复制 $K+1$ 层，层间通过特定规则（如免费/状态改变）单向连接。
* **解决痛点**: 二维 Dijkstra 状态。例如有 $K$ 次免费机会。

## 2. 逻辑支点

* **状态升维**: $dist[u][k]$ 表示到达 $u$ 且消耗 $k$ 次机会。
* **层间连边**:
    * 层内: $u_k \to v_k$ (权值 $w$)。
    * 层间: $u_k \to v_{k+1}$ (权值 $0$ 或折扣)。

## 3. 实战部署

{% fold info @Code: Layered Dijkstra %}
```cpp
// id(u, k) = u + k * n
for( int k = 0; k <= K; ++ k )
{
    // 1. 层内
    add( id(u, k), id(v, k), w );
    // 2. 层间 (免费)
    if( k < K ) add( id(u, k), id(v, k+1), 0 );
}
// Ans: min( dist[t][0...K] )
```

{% endfold %}
