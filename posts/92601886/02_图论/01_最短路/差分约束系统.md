## 1. 生态位

* **定义**: 将形如 $X_u - X_v \le W$ 的不等式组转化为最短路问题。
* **转化**: $X_u \le X_v + W \iff dist[u] \le dist[v] + w$。对应边 $v \to u$，权值 $W$。

## 2. 逻辑支点

* **三角不等式**: 最短路满足 $dist[v] \le dist[u] + w$。
* **负环判定**: 若图中存在负环，说明不等式组有冲突，**无解**。
* **超级源点**: 建立 $0$ 号点连向所有点，防止图不连通。

## 3. 实战部署

```cpp
// 核心: SPFA 判负环
bool spfa( int n )
{
    // ... init ...
    while( !q.empty() )
    {
        int u = q.front(); q.pop(); in_queue[u] = 0;
        for( auto &e : adj[u] )
        {
            if( dist[e.v] > dist[u] + e.w )
            {
                dist[e.v] = dist[u] + e.w;
                cnt[e.v] = cnt[u] + 1;
                if( cnt[e.v] > n ) return false; // 负环 -> 无解
                if( !in_queue[e.v] ) { q.push(e.v); in_queue[e.v] = 1; }
            }
        }
    }
    return true;
}
```
