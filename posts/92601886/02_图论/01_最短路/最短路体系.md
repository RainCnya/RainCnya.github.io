## 1. 生态位

* **定义**: 在加权图中，寻找从源点到目标点总权值最小的路径。
* **核心机制**: **松弛 (Relaxation)**。
* **算法选型**:

| 算法 | 核心逻辑 | 适用场景 | 复杂度 |
| :--- | :--- | :--- | :--- |
| **Dijkstra** | **贪心** + 优先队列 | **非负权**单源最短路 | $O(M \log N)$ |
| **SPFA** | **队列** + 反复入队 | **负权**边/判负环 | Avg $O(kM)$, Max $O(NM)$ |
| **Floyd** | **DP** + 阶段划分 | **多源**最短路 / 传递闭包 | $O(N^3)$ |

## 2. 逻辑支点

* **Dijkstra (确定的贪心)**
    * 将节点分为“已确定”和“未确定”。每次选出距离最小的未确定点 $u$，**$u$ 的最短路不可能再被后续节点优化**（基于非负权前提）。
* **SPFA (纠错的代价)**
    * 允许节点多次入队。如果 $u$ 的距离被更新（纠错），那么 $u$ 的邻居可能也需要随之更新。
* **Floyd (阶段性中转)**
    * $dp[k][i][j]$ 表示“仅允许经过节点 $1 \dots k$ 作为中转”的最短路。

## 3. 实战部署

### 3.1 Dijkstra (标准模板)

{% fold info @Code: Dijkstra Heap %}
```cpp
// SSSP Template (Non-negative weights)
void dijkstra( int st ) 
{
    for( int i = 1; i <= n; ++ i ) dist[i] = INF, vis[i] = 0;
    dist[st] = 0;
    
    priority_queue< PII, vector< PII >, greater< PII > > pq;
    pq.push( {0, st} );
    
    while( ! pq.empty( ) ) 
    {
        auto [d, u] = pq.top( ); pq.pop( );
        
        if( vis[u] ) continue; // Lazy deletion
        vis[u] = 1;
        
        for( auto [v, w] : adj[u] ) 
        {
            if( dist[u] + w < dist[v] ) 
            {
                dist[v] = dist[u] + w;
                pq.push( {dist[v], v} );
            }
        }
    }
}
```

### 3.2 SPFA (判负环)

{% fold info @Code: SPFA Queue %}

```C++
// Negative Cycle Detection
bool spfa( int st ) 
{
    queue< int > q;
    q.push( st ); vis[st] = 1; dist[st] = 0;
    
    while( ! q.empty( ) ) 
    {
        int u = q.front( ); q.pop( );
        vis[u] = 0; // Not in queue now
        
        for( auto [v, w] : adj[u] ) 
        {
            if( dist[u] + w < dist[v] ) 
            {
                dist[v] = dist[u] + w;
                cnt[v] = cnt[u] + 1;
                if( cnt[v] > n ) return true; // Negative Cycle
                
                if( ! vis[v] ) 
                {
                    q.push( v );
                    vis[v] = 1;
                }
            }
        }
    }
    return false;
}
```

### 3.3 Floyd (最小环变体)

{% fold info @Code: Floyd Variant %}



```C++
// Floyd with Min-Cycle
// k 必须在最外层
for( int k = 1; k <= n; ++ k ) 
{
    // 1. Check Min Cycle (using nodes 1..k-1)
    for( int i = 1; i < k; ++ i )
    {
        for( int j = i + 1; j < k; ++ j )
        {
             ans = min( ans, dist[i][j] + mat[i][k] + mat[k][j] );
        }
    }
             
    // 2. Standard Relaxation
    for( int i = 1; i <= n; ++ i )
    {
        for( int j = 1; j <= n; ++ j )
        {
            dist[i][j] = min( dist[i][j], dist[i][k] + dist[k][j] );
        }
    }
}
```
