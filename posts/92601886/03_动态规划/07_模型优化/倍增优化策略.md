## 1. 生态位

* **定义**: 利用 $2^k$ 的跳跃特性，快速计算线性或环形结构上的**多次状态转移**。
* **解决痛点**:
    * **长路径转移**: 需要跳跃 $N$ 次，倍增降至 $\log N$。
    * **环形覆盖**: 求最少多少个区间能覆盖环 (P6902)。

## 2. 逻辑支点

* **核心直觉**: **"Jump to the Limit" (极限跳跃)**。
    * 如果我们知道“跳 1 步最远到哪”，我们就能知道“跳 2 步最远到哪” ($f[i][1] = f[f[i][0]][0]$)。
* **模型**:
    * $f[i][j]$: 从点 $i$ 出发，跳 $2^j$ 步能到达的最远位置 (或最优状态)。

## 3. 实战部署

### 3.1 环形区间覆盖 (P6902)

* **转化**:
    1.  **破环成链**: 复制一倍 $1 \dots 2N$。
    2.  **贪心预处理**: $f[i][0]$ = 从 $i$ 出发用 1 个区间能覆盖到的最远右端点 (双指针求出)。
    3.  **倍增**: 建立 ST 表。
    4.  **查询**: 枚举起点，倍增跳跃，直到覆盖长度 $\ge N$。

{% fold info @Code: Doubling Coverage %}
```cpp
// 1. Precalc f[i][0] with Two Pointers
// ...
// 2. Build Table
for( int j = 1; j <= 19; ++ j )
    for( int i = 1; i <= 2*n; ++ i )
        f[i][j] = f[ f[i][j-1] ][ j-1 ];

// 3. Query
int ans = INF;
for( int i = 1; i <= n; ++ i ) // Enumerate start point
{
    int curr = i, res = 0;
    for( int j = 19; j >= 0; -- j )
    {
        if( f[curr][j] < i + n ) 
        {
            curr = f[curr][j];
            res += (1 << j);
        }
    }
    ans = min( ans, res + 1 );
}
```
