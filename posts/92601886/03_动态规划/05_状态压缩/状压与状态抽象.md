## 1. 生态位

* **定义**: 通过**二进制位**或**数学抽象**来压缩高维状态。
* **解决痛点**:
    * **数值特性**: 转移条件依赖于数值的二进制位 (P4310)。
    * **对称冗余**: 状态看似不同但功能等价 (P2051)。

## 2. 逻辑支点

### 2.1 Bitwise Domain (从位置到值域)

* **场景**: P4310 绝世好题。求子序列满足 $a_i \& a_{i-1} \ne 0$。
* **转化**: 暴力 DP $dp[i]$ (以位置 $i$ 结尾) 为 $O(N^2)$。
* **优化**: 
    * 两个数按位与不为 0 $\iff$ 它们至少共享一个为 1 的二进制位。
    * **状态**: $dp[k]$ 表示以“第 $k$ 位为 1 的数”结尾的最长子序列长度。
    * 复杂度降为 $O(N \log (\max A))$。

### 2.2 Symmetry Abstraction (对称性抽象)

* **场景**: P2051 汉诺塔 III / 放置炮兵。
* **直觉**: 列是**无标号**的。对于下一行而言，它不关心“第 1 列放了 1 个”，只关心“**有多少列**放了 1 个”。

## 3. 实战部署

### 3.1 P4310 位运算优化模板

{% fold info @Code: Bitwise LIS %}
```cpp
// dp[j]: max length ending with a number whose j-th bit is 1
for( int i = 1; i <= n; ++ i ) 
{
    int now = 1;
    // 1. Check history: inherit from any bit j that a[i] has
    for( int j = 0; j <= 30; ++ j ) 
    {
        if( ( a[i] >> j ) & 1 ) 
            now = max( now, dp[j] + 1 );
    }
            
    // 2. Update future: update all bits j that a[i] has
    for( int j = 0; j <= 30; ++ j )
    {
        if( ( a[i] >> j ) & 1 )
            dp[j] = max( dp[j], now );
    }
            
    ans = max( ans, now );
}
```
