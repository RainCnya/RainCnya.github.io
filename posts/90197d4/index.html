计算机导论

第一章 - 绪论

1. 计算机基本概念

  1. 定义：计算机是一种信息处理工具，它能自动、高效、精确地对信息进行存储、传输和加工。
  2. 特点：记忆能力强；计算精度高；处理速度快；自动化。
  3. 应用：数值计算；信息处理；辅助设计/制造/教学；人工智能；网络应用；过程控制。
  4. 发展史：
    • 算盘 →\to→ 计算尺 →\to→ 手摇机械 →\to→ 电动机械
    • 机械计算机 →\to→ 电子计算机（电子管 →\to→ 晶体管 →\to→ 集成电路 →\to→ 大规模集成）
  5. 趋势：巨型化；微型化；网络化；智能化
  6. IT行业：…\dots…

第二章 - 计算机基础知识

1. 计算与图灵机

  • 计算的实质：符号串的变换。即从一个已知的初始符号串开始，在有限步骤内按照一定规则进行变换最终得到目标字符串的过程。
  • 图灵机原理：由一条无限长的纸带、一个读写头、一个状态寄存器和一套控制规则组成。
  • 关系：可计算性   ⟺  \iff⟺ 图灵可计算性。

2. 进制转换

  • 意义：二进制便于电路实现；八进制和十六进制是方便人类对长二进制串进行书写和阅读。
  • 转换：R→10R \to 10R→10 按权展开； 10→R10 \to R10→R 除基取余。

*3. 机器数与补码

  • 形式：原码、反码、补码。

  • 转换：

    • 若 x>0x > 0x>0，即正数：原码 === 反码 === 补码；
    • 若 x<0x < 0x<0，即负数：反码 === 原码除符号位全取反，补码 === 反码加 111。

  • 为什么要用补码？

    1. 统一加减法：A−B=A+(−B)A - B = A + (-B)A−B=A+(−B)。
    2. 0 的表示唯一：在补码中，0 只有一种表示形式（000000000000000000000000）。
    3. 符号位参与运算：不需要单独判断正负。

4. 数字化过程：

  • ASCII 码：7 位二进制表示 128 个字符，高位补 0。
  • 声音数字化：采样（时间离散化） →\rightarrow→ 量化（幅度离散化） →\rightarrow→ 编码（存入计算机）。
  • 图像数字化：对连续的图像进行采样和量化。位图的基本单位是像素（Pixel）。

第三章 - 计算机系统结构

1. 冯·诺依曼体系

  • 五大部件：运算器、控制器、存储器、输入设备、输出设备。

2. 操作系统

  • 定义：管理和控制计算机软硬件资源，为用户提供便捷使用计算机的程序的集合。
  • 作用：调度、分配和管理所有的硬件设备和软件系统，使其统一协调地运行，以满足用户实际操作的需求。
  • 类型：批处理操作系统；分时操作系统；网络操作系统；实时操作系统；嵌入式操作系统。
  • 功能：处理机管理；存储器管理；文件系统管理；设备管理。
  • 地位：配置在计算机硬件上的第一层软件，是对硬件系统的首次扩充，负责配置和管理计算机上的各种资源。

3. 软件工程

  • 软件危机：由于软件规模和复杂度剧增，导致开发成本高、质量差、进度难以控制的现象。

  • 概念：为了解决“软件危机”。用工程化的方法开发和维护软件。

  • 要素：方法；工具；过程。

  • 方法：生命周期法；快速原型法；面向对象方法；敏捷开发方法；

  • 过程：问题定义；可行性研究；需求分析；总体设计；详细设计；编码和单元测试；综合测试；软件维护；

第四章 - 典型问题

1. 学科形态

  • 理论：将实际问题转换为计算机能处理的形式并对处理结果进行分析。
  • 抽象：便于计算机处理的形式研究有关问题的基本原理，具有构造性数学的特征。
  • 设计：实现问题求解的系统的设计与实现。

2. 计算可行性

  • P 类问题：所有在多项式时间内求解的问题。
  • NP 类问题：所有在多项式时间内验证的问题。

3. 典型问题：

  • 七桥问题 / 哈密尔顿回路问题 / 中国邮局问题：图论。
  • 梵天塔（汉诺塔）问题：递归。
  • 证比求易算法：P≠NPP \neq NPP=NP。
  • 旅行商问题：组合优化问题。
  • 哲学家共餐问题：操作系统。
  • 两军问题：网络协议。
  • 图灵测试 / 中文屋子问题 / 博弈问题：人工智能

第五章 - 核心概念

1. 算法

  • 概念：按照一定规则解决某类问题的明确有限的步骤。
  • 特性：有穷性；确定性；零个或多个输入；至少一个输出；有效性。
  • 复杂度：时间；空间。
  • 典型算法： 穷举法；递归；回溯；贪心；分治；随机…

2. 数据结构

  • 概念：相互之间存在一种或多种特定关系的数据元素的集合。
  • 线性表 / 栈 / 队列 / 树 / 图。

3. 数据库

  • 概念：按照数据结构来组织、存储和管理数据的仓库。
  • 数据库管理系统（DBMS）：是位于用户与操作系统之间的一层数据管理软件，它是数据库系统的核心。
    • 常用数据库：DB2，Oracle，Informix，Sybase，SQL Server。

4. 计算机网络

  • 概念：讲分布在不同的地理位置上，且具有独立功能的若干台计算机及其外围设备，通过通行设备和线路连接起来，在网络操作系统、网络管理系统及网络通信协议的管理和协调下，实现资源共享和信息传递的计算机系统。

  • 分类：局域网（LAN）、城域网（MAN）、广域网（WAN）。

  • 互联网：国际计算机互联网络。

  • 互联网应用：WWW 服务、电子邮件、文件传输、远程登录、即时通讯服务。

5. 物联网、大数据、云计算……

  • 物联网（IoT）：感知、传输、应用。

  • 大数据：4V 特征（规模性、多样性、高速性、价值密度低）。

  • 云计算：按需分配服务（IaaS, PaaS, SaaS）。

  • 人工智能（AI）：机器学习、深度学习、神经网络。

第六章 - 信息安全

1. 安全概念与防范

  • 基本内容：机密性、完整性、可用性、抗抵赖性。

  • 防范策略：身份认证、访问控制、加密技术、数字签名。

2. 计算机病毒

  • 概念：人为编制的具有破坏性、自我复制能力的指令或代码。

  • 特点：潜伏性、传染性、触发性、破坏性。

3. 法律与伦理

  • 职业道德：保护隐私、尊重产权、不传播有害信息。

  • 法规：遵守《计算机软件保护条例》等国家法律法规。
