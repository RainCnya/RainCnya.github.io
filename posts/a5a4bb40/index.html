[Leaf] [ABC442F] Diagonal Separation 2

F - Diagonal Separation 2

1. 题面梗概

给你一个 N×NN \times NN×N 的网格。你要给它重新染色，使得它满足两个条件：

  1. 行约束: 每一行必须是“左边全是白，右边全是黑”。

  2. 列约束: 每一列必须是“上边全是白，下边全是黑”。

问：最少需要改动多少个格子的颜色？

2. 逻辑推导

2.1 分析

从行约束入手，设第 iii 行分界点为 kik_iki​，即 (i,1⋯ki)(i, 1 \cdots k_i)(i,1⋯ki​) 是白色，(i,ki+1⋯N)(i, k_i + 1 \cdots N)(i,ki​+1⋯N) 是黑色。

接着考虑列约束，如果格子 (i,j)(i, j)(i,j) 是黑色，那么它下方的所有格子 (i+1,j),(i+2,j)…(i + 1, j), (i + 2, j) \dots(i+1,j),(i+2,j)… 都是黑色。

这说明了什么？说明如果第 iii 行从第 jjj 列开始变黑，那么第 i+1i + 1i+1 行在第 jjj 列也必须是黑的。这就意味着：ki+1≤kik_{i+1} \leq k_{i}ki+1​≤ki​，kkk 数组是非递增的序列。

![[Pasted image 20260125124842.png]]

结论：问题的本质就是寻找一组 k1,k2,…,knk_1, k_2, \dots, k_nk1​,k2​,…,kn​，使得修改代价最少。

2.2 DP

遇到这种情况，我们考虑用 DP 来解决最优化问题。

定义：dp[i][j] 表示处理完前 iii 行，且第 iii 行分界点为 jjj 时的最小代价。

转移：dp[i][j]=Cost(i,j)+mink=jN(dp[i−1][k])dp[i][j] = Cost(i, j) + min_{k = j}^{N}(dp[i-1][k])dp[i][j]=Cost(i,j)+mink=jN​(dp[i−1][k])。

  • 其中 Cost(i,j)Cost(i, j)Cost(i,j) 表示把第 iii 行变为 前 jjj 白，后 N−jN-jN−j 黑 的代价，这一项可以用前缀和快速查询。
  • 然后就是 min⁡k=jN(dp[i−1][k])\min_{k = j}^{N}(dp[i-1][k])mink=jN​(dp[i−1][k]) 这一项，前面提到 kkk 数组是非递增的，所以我们只需要从后面的转移。

瓶颈：如果暴力转移势必会在 min⁡\minmin 这一项中浪费大量时间，导致复杂度退化为 O(N3)O(N^3)O(N3)，那样就 TLE 了。

观察：

{dp[i][n]=Cost(i,n)+min⁡(dp[i][n])dp[i][n−1]=Cost(i,n−1)+min⁡(dp[i][n−1],dp[i][n])dp[i][n−2]=Cost(i,n−2)+min⁡(dp[i][n−2],dp[i][n−1],dp[i][n])\begin{cases} dp[i][n] = Cost(i, n) + \min(dp[i][n]) \\ dp[i][n-1] = Cost(i, n-1) + \min(dp[i][n-1], dp[i][n]) \\ dp[i][n-2] = Cost(i, n-2) + \min(dp[i][n-2], dp[i][n-1], dp[i][n]) \end{cases} ⎩⎨⎧​dp[i][n]=Cost(i,n)+min(dp[i][n])dp[i][n−1]=Cost(i,n−1)+min(dp[i][n−1],dp[i][n])dp[i][n−2]=Cost(i,n−2)+min(dp[i][n−2],dp[i][n−1],dp[i][n])​

发现了吗？min⁡k=jN(dp[i−1][k])\min_{k=j}^{N}(dp[i-1][k])mink=jN​(dp[i−1][k]) 是一个典型的后缀最小值，所以在处理每一行时，我们只需要预处理出上一行 DP 的后缀最小值数组，即可将转移复杂度从 O(N3)O(N^3)O(N3) 降维到 O(N2)O(N^2)O(N2)。

## 4. 复盘

  • 复杂度: 时间复杂度 O(N2)O(N^2)O(N2)，空间复杂度 O(N2)O(N^2)O(N2)。

  • 碎碎念:

    • 转化：题目描述的条件看起来有点抽象，但说白了其实就是，在网格中划出一条单调下降的阶梯线。线的左上方全是白，右下方全是黑。

    • 坑点: 注意分界点 kkk 的范围是 [0,N][0, N][0,N]，共有 N+1N+1N+1 种状态（从全黑到全白），初始化和循环边界不要写错。
