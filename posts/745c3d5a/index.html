什么是有限小数？对于最简分数 xy\frac{x}{y}yx​ 来说，当且仅当 y=2a5by = 2^a 5^by=2a5b 的形式。

记 v∈S={2a⋅5b∣a,b≥0}v \in S = \{ 2^a \cdot 5^b \mid a, b \geq 0 \}v∈S={2a⋅5b∣a,b≥0}，那么答案就是：

Ans=∑i=1n∑j=1n[igcd⁡(i,j)=v]Ans = \sum_{i=1}^n \sum_{j=1}^n \left[ \frac{i}{\gcd(i, j)} = v \right] Ans=i=1∑n​j=1∑n​[gcd(i,j)i​=v]

发现 iii 一定是 vvv 的倍数，那么可以接着化简：

Ans=∑k=1n/v∑j=1n[gcd(vk,j)=v]Ans = \sum_{k=1}^{n / v} \sum_{j = 1}^{n} [gcd(vk, j) = v] Ans=k=1∑n/v​j=1∑n​[gcd(vk,j)=v]

然后把后面一个和式中的 kkk 除掉，就有：

Ans=∑k=1n/v∑j=1n/k[gcd(v,j)=1]Ans = \sum_{k=1}^{n / v} \sum_{j=1}^{n / k} [ gcd(v, j) = 1] Ans=k=1∑n/v​j=1∑n/k​[gcd(v,j)=1]

由于只有 2,52, 52,5 两个因子，不妨用容斥原理展开：

Ans=∑k=1n/v[⌊n/k⌋−[a>0]⌊n/k⌋2−[b>0]⌊n/k⌋5+[ab>0]⌊n/k⌋10]Ans = \sum_{k=1}^{n / v} \left[ \lfloor n / k \rfloor - [a>0]\frac{ \lfloor n / k \rfloor }{2} - [b>0] \frac{ \lfloor n / k \rfloor }{5} + [ab>0] \frac{ \lfloor n / k \rfloor }{10} \right] Ans=k=1∑n/v​[⌊n/k⌋−[a>0]2⌊n/k⌋​−[b>0]5⌊n/k⌋​+[ab>0]10⌊n/k⌋​]
