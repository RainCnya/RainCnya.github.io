春天替我记下了这次远行

春天替我记下了这次远行

CCPC Final 河南南阳游记

去年冬天写下愿望时，我并没有想到，它会在这个春天兑现。

于是四月末，我踏上了去河南南阳的列车。

Day -1 | 出发 ( 2026/04/24 )

早上十点半出发，一路坐到晚上八点。

中途在南京中转时，我看到好几组三人同行的小队，背着电脑包，神情匆匆。我大胆猜测，他们大概也是去打 CCPC Final 的，不过看起来不像是浙江的，如果是浙江的话，他们应该去打省赛了。

那一瞬间有种奇妙的感觉：原来在不同城市、不同学校，也有很多人正沿着各自的轨道，奔赴同一个终点。

中午在动车站附近找吃的，结果绕了半天没找到什么像样的东西，最后将就吃了一碗小馄饨。吃完第一反应是：早知道吃泡面了。吃一堑，吃一堑。

晚上到南阳站，一出站就感受到主办方的热情。志愿者一路引导、安排接驳、帮忙对接，热情得让我甚至觉得“有点太热情了”。

到酒店的时候每个人发了一个大礼包，哇，一整个大包的物资，太丰盛了！后来看到群里有人感慨：“南阳理工是我第二母校。” 我看着这句话，释怀的笑了。

Day 0 | 热身赛与开幕式 ( 2026/04/25 )

第二天是开幕式和热身赛。

我也是开幕式的时候才知道，今年 Final 办在这里，是因为十年前第一届 Final 就在这里举办。今年刚好十周年。

这种感觉很妙，像回到故事开始的地方。

学校还发了很多饭券，企业宣讲也安排得满满当当，只能说主办方真的太实在了。

下午热身赛四道题，难度不算特别夸张，但非常精准地暴露了我们队伍的四个问题：

A 题：读题读错了。
B 题：1(ll)<<50 不开 long long 见祖宗。
C 题：贪心正着贪和反着贪结果不同。
D 题：交互题记得写 endl。

晚上回酒店，我紧急突袭了一下数论，把一些公式推导过程重新理了一遍。后来事实证明——完全没用。但赛前的人，总喜欢抓点什么让自己安心。

Day 1 | 正式赛 ( 2026/04/26 )

正式赛了，好紧张，从昨天晚上开始怕，怕自己连第一题都签不出来。

开赛后读题，脑子里只有一句话，这都啥跟啥啊。不过好在偷听旁边有人说 K 是签到，我们立刻开写。

最开始两发贪心思路都有问题。那段时间大概是我全场最紧张的时候，反复讨论、确认、推翻、重写，生怕第一题就翻车。最后改完第二版思路大概是 1h 左右了。

AC。

那一刻，我真的松了一大口气。如果这题再不过，3 发 WA，压力可能直接爆炸。

L 与 G

后面我们开了 L 和 G。

L 一眼看像线性基，但题目里“本质不同”的定义我们光理解就用了很久。我们调了很久，最后还是没做出来。现在回头看，不是运气问题，是对线性基理解确实还不够到位。

G 题看到子段划分时，我下意识觉得是二维 DP。但 n 的范围是 1e6。这个数字一出来，我就开始怀疑它不是 DP 了。讨论之后，我们一致判断它更像构造题，于是走了分类讨论模拟的路线。

最后因为代码写得比较 Dirty，加上时间不够，最后一分钟没交上去。虽然交上去也未必过。但没交上去，终究是另一种遗憾，最后的题解环节，发现这题的正解居然是 DP，好吧。

后半场

比赛打到后半段时，人已经有点恍惚了。可能是起太早了。可能是红牛喝太早了。也可能是大脑负载太久了。签到题之后，我明显感觉疲惫感在往上爬。那种状态很难描述——坐在那里，知道时间还在走，但思维已经不像开局时那样锐利了。

结束

直到比赛结束那一刻，我才真正感到如释重负。抬头看四周，很多队伍的神情也差不多。差距确实存在，但也没有想象中那么遥不可及。原来顶尖赛场上，大家都在和题目、时间、状态，以及自己同时交手。

而我们队坚持思考到了最后。
某种程度上来说，我们已经是冠军了。

春天

在主办方学校里，我拍到了大片盛开的红花和草地。风吹过去的时候，整个校园散发着生命的力量。那一刻我忽然觉得，这趟旅程的意义，也许从来不只在榜单上。我见过更高的山，也见过更远的光。知道了差距在哪里，也知道了下一步该往哪里走。

Day 2 | 归途 ( 2026/04/27 )

从河南回来的路上，我很平静。

结果不算圆满，故事也没有传奇般落幕。

但这趟旅途，值得珍藏；这段经历，值得回味。

我见过更大的世界。

而我带着春天回来了。
