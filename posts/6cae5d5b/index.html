差分约束系统 (Difference_Constraints)

差分约束系统 (Difference Constraints)

1. 生态位

  • 定义: 差分约束系统是将代数中的不等式组 Xu−Xv≤wX_u - X_v \le wXu​−Xv​≤w 映射为图论中的边约束，利用最短路算法的松弛性质来寻找一组可行解。它实现了代数约束向拓扑路径的转化，是处理线性不等式组一致性判定的核心工具。

  • 解决痛点: 解决了多个变量相互制约下的取值问题，能高效判定约束冲突（负环判定）并给出极端情况下的可行解。

2. 逻辑支点

  • 核心不变量: 三角不等式 dist[v]≤dist[u]+wdist[v] \le dist[u] + wdist[v]≤dist[u]+w。最短路在达到稳态时，所有的边都必然满足该性质，这与不等式 Xv≤Xu+wX_v \le X_u + wXv​≤Xu​+w 完美对偶。

  • 判定准则: 若图中存在负环，则说明不等式组存在逻辑闭环冲突（如 A<B<C<AA < B < C < AA<B<C<A），系统无解。

  • 渗透点 (Penetration): 想象每一个变量是一个高度，每一个不等式是一根限制长度的绳子。如果绳子太短无法连接所有点并满足拓扑关系，就会产生负环。

4. 知识粘附

  • 变体模型:

    • 最大/最小可行解: 初始值 inf 对应限制下的最大解；初始 0 配合超级源点对应相对最小解。

    • 前缀和转化: 将区间和问题 [L,R]≤K[L, R] \le K[L,R]≤K 转化为 S[R]−S[L−1]≤KS[R] - S[L-1] \le KS[R]−S[L−1]≤K 的标准差分形式。
