扩展欧几里得与裴蜀定理

1. 生态位

  • 识别:

    1. 求解二元一次不定方程 ax+by=cax + by = cax+by=c 的整数解。

    2. 求解线性同余方程 ax≡c(modm)ax \equiv c \pmod max≡c(modm)。

    3. 模数 mmm 不是质数，且需要求解逆元。

  • 地位: 数论体系的“解方程工具”。它在 GCD 的基础上增加了对线性组合系数 (x,y)(x, y)(x,y) 的构造能力，是连接整除与方程的关键。

  • 用途: 提供了一种确定性的递归构造法，在 O(log⁡min⁡(a,b))O( \log \min(a, b) )O(logmin(a,b)) 时间内拉出一组整数特解。

2. 逻辑支点

2.1 裴蜀定理：存在性判定

  • 内容：对于不全为 000 的整数 a,ba, ba,b，方程 ax+by=cax + by = cax+by=c 有整数解的充分必要条件是 ccc 是 gcd(a,b)gcd(a, b)gcd(a,b) 的倍数。
  • 意义：(a,b)(a, b)(a,b) 能够线性组合出的最小正整数就是它们的 gcdgcdgcd，如果要求组合出 ccc，那么 ccc 就必须是 gcdgcdgcd 的倍数。

2.2 核心原理：递归构造

我们通过递归寻找 ax+by=gcd(a,b)ax + by = gcd(a, b)ax+by=gcd(a,b) 的一组特解。

边界：当 b=0b = 0b=0 时，gcd(a,0)=agcd(a, 0) = agcd(a,0)=a，显然有 a×1+0×0=aa \times 1 + 0 \times 0 = aa×1+0×0=a，即 x=1,y=0x = 1, y = 0x=1,y=0。

假设我们已经求出了下一层 bx′+(a mod b)y′=gcd(b,a mod b)bx' + ( a \bmod b )y' = gcd( b, a \bmod b )bx′+(amodb)y′=gcd(b,amodb) 的解。

ay′+b(x′−⌊a/b⌋⋅y′)=gcd(a,b)ay' + b( x' - \lfloor a/b \rfloor \cdot y') = gcd(a, b) ay′+b(x′−⌊a/b⌋⋅y′)=gcd(a,b)
