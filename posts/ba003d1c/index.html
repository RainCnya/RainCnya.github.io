C语言程序设计

C 语言程序设计

1. 基础内容

  • 变量：能改 | 字母、数字(不能开头)、下划线。

  • 常量：不能改 | Const / Define

  • Scanf | Printf：格式化输入输出 %d %f %c

2. 数据类型与运算符

  • 整型：short int 2 | int 4 | long long 8

  • 浮点数：float (单精度) 4 | double (双精度) 8

  • 字符型：char 1

  • Sizeof： 计算变量所占字节数。

  • 计算运算符：+ - * / % 单元运算符 | ++i i++ += -= 双元运算符

  • b = (1, 3, 5, 7, 9); b=?→b=9b = ? \to b = 9b=?→b=9

3. 分支和循环结构

  • 分支语句：if 单分支 | if-else 双分支 | switch case: ...; break; default

  • 运算符：

    • 比较运算符：< <= > >= != ==
    • 逻辑运算符：&& || !
    • 三目运算符：A ? B : C

  • 循环语句：for | while | do-while | break | continue

4. 数组

是存放相同元素的集合。

  • 一维数组 / 二维数组 / 字符数组 / 字符串

    • char s[6] = "Hello"; 字符串(最后有一个\0)
    • char s[5] = {'H','e','l','l','o'} 字符数组

  • 字符串四个函数：include<string.h>

    • strlen(s1)：求s1 长度。
    • strcpy(s2, s1)：把 s1 内容拷贝到 s2 中。
    • strcat(s2, s1)：把 s1 内容连接到 s2 后。
    • strcmp(s2, s1)：比较 s1 和 s2。相等 000 | 大于 111 | 小于 −1-1−1。

5. 函数

  • 函数构成：

    • 函数名：
    • 参数：形式参数；实际参数
    • 返回值：数据类型

  • 局部变量和全局变量

6. 指针

  • 二级指针：int a = 3; int* p1 = &a; int** p2 = &p1;

  • 指针与数组：

    • 一维数组：数组名是首元素的地址——常量。 a[i] == *(a+i)
    • 二维数组：数组名是首行的地址——常量。 a[2][1] == *(*(a+2)+1)
    • 行指针：int a[3][3]; int (*p)[3] = a;。

  • 函数指针：int add( int x, int y ); int (*p)(int x, int y) = &add;

7. 自定义类型

  • 结构体(Struct)：struct book { int id, char name[20], float cost );

    • struct book bk = { 1001, "C语言", 19,9 };

    • 结构体指针：struct book* p = &bk;

    • 结构体访问：p->id | (*p).id | bk.id
