E. CCPC String

1. 题意

给定一个字符串，含有 C | P | ?，问最多能组成多少个 CCPC，也可以是 CCCCPCC，等。

2. 思路

发现核心限制条件其实是 P 的位置，所以重点在判断两个 P 之间的距离。在字符串前后各加一个 P 作为哨兵，然后二分查询每个不是 C 的前后继 P 即可。总贡献值为 min(left/2,right)min( left / 2, right )min(left/2,right)。

F. Chase Game 3

1. 题意

在一个 nnn 个点的图上，有两条链：L1L_1L1​ 是标准的 1↔2↔⋯↔n1 \leftrightarrow 2 \leftrightarrow \dots \leftrightarrow n1↔2↔⋯↔n；L2L_2L2​ 是按排列顺序连接的 p1↔p2↔⋯↔pnp_1 \leftrightarrow p_2 \leftrightarrow \dots \leftrightarrow p_np1​↔p2​↔⋯↔pn​。 Q 在 L1L_1L1​ 上走，C 在 L2L_2L2​ 上走。两人轮流移动（或原地不动）。
问： 是否对于任意的初始位置，C 都存在必胜策略能在有限步内与 Q 相遇？

2. 思路

把样例的图画出来，然后计算两个相邻下标点之间，L1 和 L2 两条链的距离。注意到，L1 的距离为 1，然后如果存在 L2 的距离为 3 及以上的，都是不可能的情况。大胆猜测这就是正解，考虑验证，的确可行，如果距离大于 2，Q 就可以在两个点之间来回横跳，而 C 需要更多的时间去追赶。

L. Completely Multiplicative Function

1. 题意

要求构造一个长度为 nnn 的 (1,−1)(1, -1)(1,−1) 序列，满足下列要求：

  1. f(x)={−1,1}f(x) = \{-1, 1\}f(x)={−1,1}。
  2. f(x)f(y)=f(xy)f(x)f(y) = f(xy)f(x)f(y)=f(xy)。
  3. f(1)+f(2)+⋯+f(n)=kf(1) + f(2) + \cdots + f(n) = kf(1)+f(2)+⋯+f(n)=k。

2. 思路

这个 f(x)f(y)=f(xy)f(x)f(y) = f(xy)f(x)f(y)=f(xy)，下意识联想到数论中的积性函数，然后我就想到欧拉函数，然后就跑偏了。大胆猜测贪心，−1-1−1 的位置大概率和质数有关，假设我修改 kkk 为 −1-1−1，那么 k3,k5,…k^3, k^5, \dotsk3,k5,… 都是 −1-1−1，同时所有包括质因数 kkk 的奇数次幂的数也会被反转。

然后分析题意，什么情况下是不可行的，假设有 aaa 个 1，bbb 个 −1-1−1，就可以得到下列方程，a−b=k,a+b=na - b = k, a + b = na−b=k,a+b=n，求解得到 a=n+k2,b=n−k2a = \frac{n+k}{2}, b = \frac{n-k}{2}a=2n+k​,b=2n−k​，发现如果 nnn 和 kkk 的奇偶性不同，无法得到整数的 aaa 和 bbb，此时不可能构造。

接着考虑如何贪心反转，从小到大反转，还是从大到小反转。注意到 >n2> \frac{n}{2}>2n​ 的质数的贡献值一定为 1，因为最小的质数是 2，乘起来会超过 nnn；而 ≤n2\leq \frac{n}{2}≤2n​ 的质数，数字越小，贡献值越大。答案呼之欲出，从小到大贪心，然后用 >n2> \frac{n}{2}>2n​ 的质数补充剩余空隙。

注意特判最后 −1-1−1 的数量是不是等于 bbb，如果不等于，也是不可以构造的。

J. Best Carry Player 3

1. 题意

给定 X,Y,KX, Y, KX,Y,K，每次可以进行下列三种操作，问至少几次可以把 XXX 转化为 YYY。XXX 加一；XXX 减一；XXX 异或 [0,k][0,k][0,k] 中的一个数。

2. 思路

题目问最小操作次数，首选 BFS 暴力，显然不会这么简单，那就需要研究异或（位运算）的性质了。然后发现，关键点在于这个异或上，也就是说，在 kkk 最高位及其以下的比特位，都是可以通过异或来进行修改的，但是更高位的就只能通过 +1+1+1 来处理了。

接着就可以把给定的 XXX 划分为两块，记 LLL 是大于 kkk 的最小 2 的幂，也就是 2m+12^{m+1}2m+1，（mmm 是 kkk 的最高位数）那么任何数字 NNN 都可以写成 N=H⋅L+RN = H \cdot L + RN=H⋅L+R，HHH 是高位的部分，而 RRR 是低位的部分。

接着处理低位的情况，在一个块的内部，从 aaa 到 bbb 的最少步数 dist(a,b)dist(a, b)dist(a,b) 为以下三种情况：0 步：a=ba = ba=b；1 步：∣a−b∣=1|a - b| = 1∣a−b∣=1 或者 (a⊕b)≤k(a \oplus b) \leq k(a⊕b)≤k；2 步：剩余情况。

然后贪心，如果 X,YX, YX,Y 不在同一块内，就需要先让 XXX 的地位变成 L−1L-1L−1，然后再通过 +1+1+1 的操作来进行进位。如果中间隔着很多完整的块，每个块的代价都是固定的。需要从 000 走到 L−1L-1L−1，再 +1+1+1，单块代价：w=dist(0,L−1)+1w = dist(0, L-1) + 1w=dist(0,L−1)+1，总代价就是 (Hy−Hx−1)⋅w(H_{y} - H_{x} - 1) \cdot w(Hy​−Hx​−1)⋅w。最后补齐 XXX 的低位即可：dist(0,Y%L)dist(0, Y \% L)dist(0,Y%L)。

C. DFS Order 3

1. 题意

给定一棵 nnn 个节点的树，分别从 1→n1 \to n1→n 点开始的 DFS 序，要求构造出原树。

2. 思路

本题需要用到两个 DFS 的定理（当然我赛前不知道，没写出来）。

  1. DFS 序中最后一个点一定是叶子节点。
  2. DFS 序中第二个点一定与第一个点直接相连。

因为保证存在一颗满足条件的树，不妨假设 1 为根节点，然后枚举叶子节点，从 1 出发的 DFS 序中，从后往前删点，同时添加边即可。

注意细节，既然 vvv 是叶子节点，那么根据引理1 它在整棵树中只有一个邻居，接着根据 引理2 以 vvv 为根节点的 DFS，D[v][2] 一定就是它的相邻点。
