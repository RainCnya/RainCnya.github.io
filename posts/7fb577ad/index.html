P1967 货车运输 (最大生成树 + LCA)

Luogu-P1967 货车运输

1. 核心逻辑

问题本质

瓶颈路问题 (Bottleneck Path Problem)：在带权图中，寻找两点间的一条路径，使得路径上的最小边权最大。

破局思路

  1. 贪心重构 (Kruskal):

    • 想要路径上的最小边权尽可能大，意味着我们需要优先选择边权大的边来连接连通块。
    • 这等价于构造最大生成树 (Maximum Spanning Tree, MaxST)。
    • 根据 MST 的性质，两点在 MST 上的唯一路径，即为原图中的瓶颈路（Maximin Path）。

  2. 树上查询 (LCA +倍增):

    • 原图转化为最大生成森林后，问题变为：查询树上 u,vu, vu,v 路径间的最小边权。
    • 利用 LCA (最近公共祖先) 配合 倍增数组 min_w[u][i]（表示 uuu 向上跳 2i2^i2i 步路径上的最小边权）即可在 O(log⁡N)O(\log N)O(logN) 内完成单次查询。

  3. 算法形态:

    • 预处理: Kruskal 重构图 -> 构建最大生成森林。
    • DFS: 处理每个连通块，初始化倍增表 f 和 min_w。
    • Query: LCA 求解 u,vu, vu,v 路径最小值。

// P1967 货车运输// Key Logic: MaxST (Kruskal) + LCA (Binary Lifting)// Airy Style v3.1#include<bits/stdc++.h>using namespace std;using ll = long long;using PII = pair<int, int>;const int maxn = 1e4 + 50;const int maxm = 5e4 + 50;const int inf = 0x3f3f3f3f;struct Edge {    int u, v, w;    bool operator < ( const Edge &other ) const     {        return w > other.w; // 最大生成树，按权值降序    }} edges[maxm];int fa[maxn];int f[maxn][20];     // LCA 倍增祖先int min_w[maxn][20]; // 路径最小边权int dep[maxn];bool vis[maxn];vector<PII> adj[maxn];int n, m, q;int find( int x ){    if( x == fa[x] ) return fa[x];    else return fa[x] = find( fa[x] );}void kruskal( ){    for( int i = 1; i <= n; ++ i ) fa[i] = i;    sort( edges + 1, edges + m + 1 );    for( int i = 1; i <= m; ++ i )    {        auto [u, v, w] = edges[i];        int ru = find( u ), rv = find( v );        if( ru != rv )        {            fa[ru] = rv;            adj[u].push_back({ v, w });            adj[v].push_back({ u, w });        }    }}void dfs( int u, int father, int w ){    vis[u] = 1;    dep[u] = dep[father] + 1;    f[u][0] = father;    min_w[u][0] = w;    for( int i = 1; i < 20; ++ i )    {        f[u][i] = f[ f[u][i-1] ][i-1];        min_w[u][i] = min( min_w[u][i-1], min_w[ f[u][i-1] ][i-1] );    }        for( auto [v, val] : adj[u] )    {        if( v != father )         {            dfs( v, u, val );        }    }}int query( int x, int y ){    int ans = inf;    if( dep[x] < dep[y] ) swap( x, y );        // 1. Align depth    for( int i = 19; i >= 0; -- i )    {        if( dep[ f[x][i] ] >= dep[y] )        {            ans = min( ans, min_w[x][i] );            x = f[x][i];        }    }    if( x == y ) return ans;    // 2. Jump together    for( int i = 19; i >= 0; -- i )    {        if( f[x][i] != f[y][i] )        {            ans = min( ans, min_w[x][i] );            ans = min( ans, min_w[y][i] );            x = f[x][i];            y = f[y][i];        }    }        // 3. Final step to LCA    ans = min( ans, min_w[x][0] );    ans = min( ans, min_w[y][0] );    return ans;}int main( ){    ios::sync_with_stdio(0);    cin.tie(0); cout.tie(0);    cin >> n >> m;    for( int i = 1; i <= m; ++ i )    {        int u, v, w;        cin >> u >> v >> w;        edges[i] = { u, v, w };    }    kruskal( );    // 森林处理：可能存在不连通的情况    memset( min_w, 0x3f, sizeof( min_w ) );    for( int i = 1; i <= n; ++ i )    {        if( !vis[i] )         {            dfs( i, 0, inf );        }    }    cin >> q;    for( int i = 1; i <= q; ++ i )    {        int x, y;        cin >> x >> y;        if( find( x ) != find( y ) )        {            cout << -1 << '\n';        }        else        {            cout << query( x, y ) << '\n';        }    }    return 0;}

3. 归档备注

  • 复杂度: Time: O(Mlog⁡M+Qlog⁡N)O(M \log M + Q \log N)O(MlogM+QlogN) | Space: O(Nlog⁡N)O(N \log N)O(NlogN).

  • 易错点:

    • 非连通图: 图可能不是连通的，必须处理为森林（在主函数循环调用 DFS）。

    • 并查集状态: 询问前需再次使用 find 判断连通性（虽然 MST 已建好，但直接用并查集判断最快）。

    • 初始化: min_w 需初始化为无穷大，避免 LCA 跳出边界时取到 0。
