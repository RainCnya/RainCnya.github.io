算法学习笔记：倍增 (Binary Lifting) -V3.2

一、 初步认知 (Macro View)

1. 叫什么 (Identity)

  • 中文名： 倍增
  • 英文名： Binary Lifting

2. 什么地位 (Status)

  • 所属领域： 算法思想 / 预处理
  • 核心地位： 一种利用“二进制”和“动态规划”思想，将 O(N)O(N)O(N) 的操作（如 K 级跳跃）通过 O(Nlog⁡N)O(N \log N)O(NlogN) 的预处理，优化到 O(log⁡N)O(\log N)O(logN) 的高效技巧。

3. 有什么用处 (Utility)

  • 解决的核心问题： 快速查询“ KKK 级祖先”或“区间 KKK 级信息”。
  • 典型应用场景：
    • LCA (最近公共祖先)： 快速查询树上两点的 LCA ( O(log⁡N)O(\log N)O(logN) )。
    • ST 表 (稀疏表)： 快速查询静态区间最值 (RMQ) ( O(1)O(1)O(1) )。

{% fold info @RMQ问题 %}

RMQ (Range Minimum/Maximum Query)，中文称为 “区间最值查询”。

这是一个非常经典的算法问题，其核心任务是：

给定一个静态数组（即数组元素不会被修改），你需要高效地回答若干次查询：每次查询给定一个区间 [l,r][l, r][l,r]，要求快速返回该区间内的最大值（或最小值）。

核心特点：

  • 静态 (Static)： 数组一旦给定，就不会再有“修改元素”的操作。
  • 查询 (Query)： 重点在于“查询”速度，通常需要回答大量（例如 10510^5105 次）的查询。

经典解法：

  • 朴素解法： 每次查询都遍历 [l,r][l, r][l,r]，复杂度 O(N)O(N)O(N)，总复杂度 O(QN)O(QN)O(QN)，会超时。
  • ST 表 (Sparse Table)： 这是 RMQ 问题的标准解法。它利用倍增思想，通过 O(Nlog⁡N)O(N \log N)O(NlogN) 的预处理，可以实现**O(1)O(1)O(1)** 的单次查询。
  • 线段树 (Segment Tree)： 也可以解决，预处理 O(N)O(N)O(N)，查询 O(log⁡N)O(\log N)O(logN)。但对于 RMQ 这个问题，ST 表的 O(1)O(1)O(1) 查询是其最大优势。

{% endfold %}

二、 主题渗透 (Theme Osmosis)

1. 核心原理 (The Eureka Moment)

  • 底层思想： 二进制拆分 + 动态规划 (DP)。
  • 关键推导：
    • 瓶颈： 朴素算法的瓶颈在于“一步一步”上跳，太慢了。
    • 核心： 任何一个 kkk 都可以被二进制拆分为 k=2a+2b+…k = 2^a + 2^b + \dotsk=2a+2b+…。
      • 例如，跳 131313 步   ⟹  \implies⟹ 13=8+4+1=23+22+2013 = 8 + 4 + 1 = 2^3 + 2^2 + 2^013=8+4+1=23+22+20。
      • 我们可以通过预处理，使自己拥有“跳 2k2^k2k 步”的能力。
    • 状态定义 (DP)：
      • LCA (K 级祖先)： fa[u][k] 表示节点 u 向上跳 2k2^k2k 步 能到达的祖先。
      • ST 表 (RMQ)： st[i][k] 表示区间 [i,i+2k−1][i, i + 2^k - 1][i,i+2k−1] 的最值。
    • 状态转移 (DP)：
      • LCA： u 跳 2k2^k2k 步 = u 先跳 2k−12^{k-1}2k−1 步，再从新位置跳 2k−12^{k-1}2k−1 步。
        • fa[u][k] = fa[ fa[u][k-1] ][k-1]
      • ST 表： 长度为 2k2^k2k 的区间 = 拼合两个 2k−12^{k-1}2k−1 的区间。
        • st[i][k] = min( st[i][k-1], st[i + (1<<(k-1))][k-1] )
    • Base Case：
      • LCA： fa[u][0] (跳 20=12^0=120=1 步) = parent[u] (父节点)。
      • ST 表： st[i][0] (长度 20=12^0=120=1) = a[i] (元素本身)。

2. 模拟演示 (Visualization)

{% fold info @模拟 (LCA 查询) %}

  • 任务： 预处理 fa 表，查询 lca(u, v)。

  • 模型： fa[u][k] = fa[ fa[u][k-1] ][k-1]

  • 1. 预处理：

    • dfs 获知 dep[u] 和 fa[u][0] (父节点)。
    • k=1：fa[u][1] = fa[fa[u][0]][0] (u 的爷爷)。
    • k=2：fa[u][2] = fa[fa[u][1]][1] (u 的 22=42^2=422=4 辈祖先)。
    • … 以此类推。

  • 2. 查询 lca(u, v) (假设 dep[u] > dep[v])

  • a. 拉平深度：

    • 计算 diff = dep[u] - dep[v]。
    • (例如 diff=13)
    • k 从 0 循环到 maxlg：
      • if (diff & (1<<0)) (k=0, 13&1=1)   ⟹  \implies⟹ u = fa[u][0] (跳1步)。
      • if (diff & (1<<1)) (k=1, 13&2=0)   ⟹  \implies⟹ 不跳。
      • if (diff & (1<<2)) (k=2, 13&4=4)   ⟹  \implies⟹ u = fa[u][2] (跳4步)。
      • if (diff & (1<<3)) (k=3, 13&8=8)   ⟹  \implies⟹ u = fa[u][3] (跳8步)。
    • 最终 u 跳了 1+4+8=131+4+8=131+4+8=13 步，与 v 同深度。

  • b. 同步跳跃：

    • k 从 maxlg 递减到 0：
      • if (fa[u][k] != fa[v][k]) (说明 LCA 还在上面)
      • u = fa[u][k]; v = fa[v][k]; (一起跳)

  • c. 获取答案：

    • 循环结束， uuu 和 vvv 在 LCA 的正下方，fa[u][0] 即为 LCA。

{% endfold %}

3. 详细实现过程 (Step-by-Step)

应用一：LCA (倍增)

  1. 初始化 (DFS):

    • 此步骤的目的是获取每个节点的深度 dep[u] 和直接父节点 fa[u][0]（即 202^020 祖先）。

    {% fold info @Init(dfs) %}

    // dfs( s, 0 ) 从根节点 s (父节点为 0) 开始
void dfs( int u, int father )
{
    // 1. 记录父节点 (Base Case)
    // 节点 u 的 2^0 祖先就是它的父节点 father
    fa[u][0] = father;
    // 2. 记录深度
    dep[u] = dep[father] + 1;
    // 3. 递归遍历子节点
    for( int v : adj[u] )
    {
        if( v != father ) // 避免反复横跳
            dfs( v, u );
    }
    return ;
}

    {% endfold %}

  2. 预处理:

    {% fold info @Prefix %}

    void preFix( )
{
    // j 表示 k，跳跃倍数
    for( int j = 1; j < maxlg; ++ j )
        // i 表示 u，当前位置
        for( int i = 1; i <= n; ++ i )
            // 核心转移：u 跳 2^j 步 = u 先跳 2^(j-1) 步，
            // 再从新位置 fa[i][j-1] 跳 2^(j-1) 步
            fa[i][j] = fa[ fa[i][j-1] ][j-1];
    return ;
}

    {% endfold %}

  3. 查询:

    • 此步骤在 O(log⁡N)O(\log N)O(logN) 内找到 lca(u, v)。

    {% fold info @Query %}

    int getlca( int u, int v)
{
    // 1. 拉平深度：确保 u 是更深的那个节点
    if( dep[u] < dep[v] )   swap( u, v );
   
    // 2. 拉平深度：计算高度差
    int diff = dep[u] - dep[v];

    // 3. 拉平深度：u 利用二进制拆分向上跳 diff 步
    for( int i = 0; (1 << i) <= diff; ++ i ) // 遍历 k (i)
    {
        if( (1 << i) & diff ) // 检查 diff 的第 k 位是否为 1
            u = fa[u][i]; // 如果是，u 就跳 2^k 步
    }

    // 4. 特判：如果跳完后 u == v，说明 v 是 u 的祖先
    if( u == v )    return u;

    // 5. 同步跳跃：k (i) 必须从大到小
    for( int i = maxlg - 1; i >= 0; -- i )
    {
        // 如果 2^k 祖先不同，说明 LCA 还在上面
        if( fa[u][i] != fa[v][i] )
        {
            // 放心跳：u 和 v 一起跳到LCA的下一层
            u = fa[u][i];
            v = fa[v][i];
        }
    }

    // 6. 获取答案：
    // 循环结束，u 和 v 是 LCA 的子节点，
    // 它们的父节点 (fa[u][0]) 就是 LCA
    return fa[u][0];
}

    {% endfold %}

应用二：ST 表 (RMQ)

  1. 初始化：

    • 此步骤预处理 ⌊log⁡2(i)⌋\lfloor \log_2(i) \rfloor⌊log2​(i)⌋ 的值，并读入 a[i] 作为 st[i][0]。

    {% fold info @Init %}

    // 1. 预处理 log2 数组 (O(N))
for( int i = 2; i <= n; ++ i )
    lg[i] = lg[i >> 1] + 1; // log(i) = log(i/2) + 1

// 2. 初始化 (k=0)
// 读入原数组 a[i]，并将其存为 ST 表中长度为 2^0=1 的区间
for( int i = 1; i <= n; ++ i )
    cin >> st[i][0];

    {% endfold %}

  2. 预处理 (DP):

    {% fold info @Prefix %}

    // k (j) 必须是外层循环
for( int j = 1; j <= maxlg; ++ j )
{
    // i 是内层循环
    // 必须保证 [i, i + 2^j - 1] 不超过 n
    for( int i = 1; i + (1 << j) - 1 <= n; ++ i )
    {
        // 核心转移：[i, 2^j] 区间的最值 =
        // max( 左半边 [i, 2^(j-1)], 右半边 [i + 2^(j-1), 2^(j-1)] )
        st[i][j] = max( st[i][j-1], st[i + (1 << (j - 1)) ][j-1] );
    }
}

    {% endfold %}

  3. 查询:

    • 此步骤在 O(1)O(1)O(1) 内回答 max(l, r)。

    {% fold info @Query %}

    // 1. 获取查询区间
int l, r;
cin >> l >> r;
// 2. 计算区间长度 len = r - l + 1 并 O(1) 查表得到 k = floor(log2(len))
int k = lg[ r - l + 1 ];
// 3. 获取答案
// 我们用两个长度为 2^k 的区间覆盖 [l, r]：
// 区间 1: [l, l + 2^k - 1] -> st[l][k]
// 区间 2: [r - 2^k + 1, r] -> st[r - (1 << k) + 1][k]
// 这两个区间可能有重叠，但不影响 max/min 的结果
cout << max( st[l][k], st[r - (1 << k) + 1][k] ) << '\n';

    {% endfold %}

2. 核心注意点 (Key Pitfalls)

  1. maxlg 可能涉及 MLE。

    • maxn = 1e5   ⟹  \implies⟹ log⁡2(105)≈16.6\log_2(10^5) \approx 16.6log2​(105)≈16.6   ⟹  \implies⟹ maxlg 至少应为 17。
    • maxn = 1e6   ⟹  \implies⟹ log⁡2(106)≈19.9\log_2(10^6) \approx 19.9log2​(106)≈19.9   ⟹  \implies⟹ maxlg 至少应为 20。
    • 一般来说设置 const int maxlg = 20 是一个非常好的、适用于 N≤106N \le 10^6N≤106 的取值。

  2. k（跳跃倍数） 必须是外层循环

四、 代码模板 (Code Template)

{% fold info @倍增LCA %}

// 假设 fa[maxn][maxlg], dep[maxn] 已定义
// 假设 adj[] 邻接表已填充

void dfs( int u, int pa )
{
    fa[u][0] = pa;
    dep[u] = dep[pa] + 1;
    for( int v : adj[u] )
    {
        if( v != pa )
            dfs( v, u );
    }
    return ;
}

void preFix( int n )
{
    for( int j = 1; j < maxlg; ++ j )
        for( int i = 1; i <= n; ++ i )
            fa[i][j] = fa[ fa[i][j-1] ][j-1];
    return ;
}

int getlca( int u, int v)
{
    if( dep[u] < dep[v] )  swap( u, v );

    int diff = dep[u] - dep[v];
    // 拉平深度
    for( int i = 0; (1 << i) <= diff; ++ i )
    {
        if( (1 << i) & diff )
            u = fa[u][i];
    }

    if( u == v )    return u;

    // 同步上跳
    for( int i = maxlg - 1; i >= 0; -- i )
    {
        if( fa[u][i] != fa[v][i] )
        {
            u = fa[u][i];
            v = fa[v][i];
        }
    }
    return fa[u][0];
}

{% endfold %}

{% fold info @倍增ST表 %}

// 假设 st[maxn][maxlg] (ST表), a[maxn] (原数组) 已定义
// 假设 log2[maxn] (预处理的 log) 已计算

void preFix( int n )
{
    // Base Case
    for (int i = 1; i <= n; ++i)
    {
        st[i][0] = a[i];
    }

    // Prefix
    for (int k = 1; k <= log2[n]; ++k)
    {
        for (int i = 1; i + (1 << k) - 1 <= n; ++i)
        {
            st[i][k] = min(st[i][k-1], st[i + (1 << (k-1))][k-1]);
        }
    }
}

int query(int l, int r)
{
    int k = log2[r - l + 1];
    return min(st[l][k], st[r - (1 << k) + 1][k]);
}
