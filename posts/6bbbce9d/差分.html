算法学习笔记：差分数组 (Difference Array) -V3.2

一、 初步认知 (Macro View)

1. 叫什么 (Identity)

  • 中文名： 差分数组
  • 英文名： Difference Array

2. 什么地位 (Status)

  • 所属领域： 基础算法 / 数据预处理
  • 核心地位： 与前缀和互为逆运算，是解决区间修改问题的最基础、最高效的工具。

3. 有什么用处 (Utility)

  • 解决的核心问题： 高效地对序列进行多次区间修改（例如：区间加值）。
  • 典型应用场景：
    • 频繁的区间增减操作，但最终只需要一次查询结果。
    • 二维差分（用于子矩阵的修改）。

二、 主题渗透 (Theme Osmosis)

1. 核心原理 (The Eureka Moment)

  • 底层思想： 逆运算 + 标记思想。
    • 利用前缀和的逆运算，将“一段区间的共同变化”转化为“起点和终点的单点变化”。
  • 关键推导：
    • 定义： dif[i]=a[i]−a[i−1]dif[i] = a[i] - a[i-1]dif[i]=a[i]−a[i−1] （规定 a[0]=0a[0]=0a[0]=0）。
    • 性质： 原数组 aaa 是差分数组 difdifdif 的前缀和，即 a[i]=∑k=1idif[k]a[i] = \sum_{k=1}^{i} dif[k]a[i]=∑k=1i​dif[k]。
    • 操作核心： 区间 [L,R][L, R][L,R] 全部 +v+v+v   ⟺  \iff⟺ dif[L]+=vdif[L] += vdif[L]+=v 且 dif[R+1]−=vdif[R+1] -= vdif[R+1]−=v。

3. 详细实现过程 (Step-by-Step)

  1. 初始化 (Initialization)：
    • 遍历原数组 aaa，根据公式 dif[i]=a[i]−a[i−1]dif[i] = a[i] - a[i-1]dif[i]=a[i]−a[i−1] 构造初始差分数组。
  2. 区间修改 (Modification)：
    • 对于每次操作 [L,R,V][L, R, V][L,R,V]，仅需 O(1)O(1)O(1) 地修改两个端点：dif[L]+=Vdif[L] += Vdif[L]+=V 和 dif[R+1]−=Vdif[R+1] -= Vdif[R+1]−=V。
  3. 还原结果 (Restoration)：
    • 修改完成后，对 difdifdif 数组求一次前缀和，即可还原出最终的 aaa 数组。

三、 总结分析 (Analysis & Comparison)

1. 复杂度分析 (Complexity)

2. 核心注意点 (Key Pitfalls)

  • 边界问题： 右端点的修改位置是 R + 1，而不是 R。因此差分数组的大小通常需要开到 N + 2 以防止越界。
  • 适用条件： 适用于离线场景（先修改完，再查询）。如果需要一边修改一边查询，则需要使用更高级的数据结构（如线段树、树状数组）。
  • 1-based 索引： 构造时 dif[i]=a[i]−a[i−1]dif[i] = a[i] - a[i-1]dif[i]=a[i]−a[i−1]，还原时 a[i]=a[i−1]+dif[i]a[i] = a[i-1] + dif[i]a[i]=a[i−1]+dif[i]。a[0]a[0]a[0] 视为 0。

四、 代码模板 (Code Template)

{% fold info @Template %}

int main( )
{
    int n, q;
    cin >> n >> q;

    // 1. 输入与初始构造
    for(int i = 1; i <= n; ++i)
    {
        cin >> a[i];
        // 构造差分数组
        dif[i] = a[i] - a[i-1];
    }

    // 2. 执行区间修改
    while( q -- )
    {
        int l, r, v;
        cin >> l >> r >> v;
        // 核心操作：起点 +v, 终点后一位 -v
        dif[l] += v;
        dif[r + 1] -= v;
    }

    // 3. 还原最终结果 (求前缀和)
    for( int i = 1; i <= n; ++ i )
    {
        // 求前缀和，b[i] 为还原后的 a[i]
        b[i] = b[i-1] + dif[i];
    }
    return 0;
}
