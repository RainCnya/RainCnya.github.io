算法学习笔记：单调栈 (Monotonic Stack) -V3.0

一、 初步认知 (Macro View)

1. 叫什么 (Identity)

  • 中文名： 单调栈
  • 英文名： Monotonic Stack

2. 什么地位 (Status)

  • 所属领域： 数据结构 / 优化技巧
  • 核心地位： 能够在 O(N)O(N)O(N) 时间内解决 “寻找序列中每个元素左/右侧第一个比它大/小的元素” 这类问题。
    • 是处理 柱状图最大矩形 和 下一更大元素 等问题的核心工具。

3. 有什么用处 (Utility)

  • 解决的核心问题： 将涉及到“寻找左右边界”的 O(N2)O(N^2)O(N2) 嵌套循环优化到 O(N)O(N)O(N)。
  • 典型应用场景：
    • 柱状图中的最大矩形问题。
    • 数组中下一个更大/更小元素问题。
    • 动态规划或其它算法中的优化，用于快速查找最优边界。

二、 主题渗透 (Theme Osmosis)

1. 核心原理 (The Eureka Moment)

  • 底层思想： 空间换时间 + 过滤冗余信息。
  • 关键推导： 栈只存储有潜力成为未来元素边界的索引。
    • （以维护单调递增栈为例，求左侧第一个更小值）：
    • 1. 淘汰（栈顶）： 当一个新元素 a[i]a[i]a[i] 准备入栈时，如果它比栈顶元素 a[stk.top()] 更小（或等于），说明栈顶元素 既不如此时新，也不如此时优。
      • 栈顶元素 a[stk.top()] 永远不可能成为 iii 之后任何元素的“左侧第一个更小值”（因为 a[i]a[i]a[i] 更近且更小）。
      • 因此，栈顶元素 a[stk.top()] 必须出栈。
    • 2. 寻找（新栈顶）： 持续弹出栈顶，直到栈顶元素 a[stk.top()] 严格小于 a[i]a[i]a[i]（或栈空）。此时，这个新的栈顶 stk.top() 就是 a[i]a[i]a[i] 的“左侧第一个更小值”的索引。

3. 详细实现过程 (Step-by-Step)

以 求解左侧第一个小于 a[i]a[i]a[i] 的元素 L[i]L[i]L[i] （单调递增栈）为例：

  1. 初始化 (Initialization)： 创建一个空栈 stk。
  2. 选择模型 (Model)：
    • 求第一个更小 (L/R)： 使用单调递增栈（弹出 ≥\ge≥ 的元素）。
    • 求第一个更大 (L/R)： 使用单调递减栈（弹出 ≤\le≤ 的元素）。
  3. 遍历 (Iteration)：
    • 求 L (左边界)： 正序遍历数组 aaa，从 i=1→ni=1 \to ni=1→n。
    • 求 R (右边界)： 倒序遍历数组 aaa，从 i=n→1i=n \to 1i=n→1。
  4. 维护单调性 (Pop)： while (!stk.empty() && a[stk.top()] ... a[i]) { stk.pop(); }
  5. 确定边界 (Get Answer)：
    • 栈空   ⟹  \implies⟹ 边界为哨兵（L 侧为 0，R 侧为 n+1n+1n+1）。
    • 栈不空   ⟹  \implies⟹ 边界为 stk.top()。
  6. 入栈 (Push)： 将当前下标 iii 压入栈中。

三、 总结分析 (Analysis & Comparison)

1. 复杂度分析 (Complexity)

维度 复杂度 说明
时间复杂度 O(N)O(N)O(N) NNN 为元素总数。每个元素的索引最多入栈一次，出栈一次。均摊复杂度为 O(1)O(1)O(1)。
空间复杂度 O(N)O(N)O(N) 在最坏情况下（例如数组完全单调递增），栈 stk 需要存储所有 NNN 个元素的索引。

2. 核心注意点 (Key Pitfalls)

  • 边界处理： L[j]L[j]L[j] 的边界是 0；R[j]R[j]R[j] 的边界是 M+1M+1M+1。
  • 栈的单调性：
    • 求第一个小于： 使用单调递增栈（弹出 ≥\ge≥ 的）。
    • 求第一个大于： 使用单调递减栈（弹出 ≤\le≤ 的）。
  • 存储内容： 栈中必须存储索引，而不是元素的值，否则无法确定边界的位置。
  • 重复值处理： 弹出条件用 ≥\ge≥ 还是 >>> 取决于题目定义，看题目要求是否严格单调。

四、 代码模板 (Code Template)

stack<int> stk;
// 求 L_small (正向遍历)
for(int i = 1; i <= n; ++i)
{
    // 弹出所有 >= a[i] 的元素
    while ( !stk.empty( ) && a[stk.top()] >= a[i] )
        stk.pop();

    if ( stk.empty( ) ) L_small[i] = 0;
    else L_small[i] = stk.top( );

    stk.push(i);
}
