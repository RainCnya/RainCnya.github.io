图论建模综述 (Graph Core Synthesis)

1. 生态位

  • 定义: 图论不仅是研究“城市与道路”的物理学科，更是处理离散状态与关系的通用代数语言。建模的核心在于将问题的 “状态”映射为节点，将 “关系/限制”映射为边。
  • 物理图 vs 代数图:
    • 物理图 (Physical Graph): 节点代表实体位置，边代表物理连接（如地图导航、网络布线）。
    • 代数图 (Algebraic Graph): 节点代表数值状态，边代表转移逻辑或约束条件（如不等式组、同余类、游戏状态）。
  • 解决痛点:
    • 复杂约束可视化: 将抽象的代数矛盾（如不等式无解）转化为几何结构（负环）。
    • 无限状态折叠: 将无限的数轴通过“取模”折叠为有限的环（同余最短路）。
    • 动态过程静态化: 利用分层图或 Floyd 的阶段性处理时间维度。

2. 逻辑支点

  • 核心直觉: Graph as a Relation Map (图即关系)。
    • 只要满足 StateB←f(StateA,Cost)State_B \leftarrow f(State_A, Cost)StateB​←f(StateA​,Cost) 形式的转移，皆可视为图。
  • 三大建模范式:
    1. 三角不等式映射 (Triangle Inequality Mapping):
      • 代数形式: xv−xu≤w  ⟺  xv≤xu+wx_v - x_u \le w \iff x_v \le x_u + wxv​−xu​≤w⟺xv​≤xu​+w。
      • 图论形式: 最短路松弛条件 dist[v]≤dist[u]+wdist[v] \le dist[u] + wdist[v]≤dist[u]+w。
      • 结论: 每一个形如 A−B≤CA-B \le CA−B≤C 的不等式，都是一条从 BBB 指向 AAA 的边。
    2. 同余折叠 (Modulo Folding):
      • 问题: 求解 a1x1+⋯+anxn=Ka_1 x_1 + \dots + a_n x_n = Ka1​x1​+⋯+an​xn​=K 的可行性，其中 KKK 极大。
      • 转化: 选取最小的 aia_iai​ 作为模数 mmm。若能凑出余数为 rrr 的最小数 ValminVal_{min}Valmin​，则所有 Valmin+k⋅mVal_{min} + k \cdot mValmin​+k⋅m 均可凑出。
      • 模型: 节点 0…m−10 \dots m-10…m−1 构成一个环形图。
    3. 时间/阶段展开 (Time Expansion):
      • 问题: 动态加点/边的最短路，或有步数限制的最短路。
      • 转化: 利用 Floyd 算法最外层循环 kkk 的物理意义（允许经过前 kkk 个点/第 kkk 个阶段）。

  • 母题: [Luogu P1119] 灾后重建

⚠️ 边界与 Hack

  • 差分约束:
    • 符号方向: 极易搞反。记住口诀：“谁小谁是被指向的终点” (A≤B+W  ⟹  B→AA \le B+W \implies B \to AA≤B+W⟹B→A)。
    • 超级源点: 若图不连通，SPFA 无法遍历所有点，必须建立虚点连接所有节点。
  • 同余最短路:
    • 模数选择: 必须选最小的 aia_iai​，否则点数爆炸导致 TLE。
  • 最短路计数:
    • 0 边权: 若存在 0 边，最短路图可能包含环，导致 DP 或记忆化搜索死循环。需特判或缩点。
