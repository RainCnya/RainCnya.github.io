P1993 小 K 的农场

Luogu-P1993 小 K 的农场

题目简述

难度：提高 | 题解格式：V6.3

给定 nnn 个农场和 mmm 个约束条件。设 xix_ixi​ 为农场 iii 的作物数量。判断以下约束系统是否存在非负整数解：

  1. xa−xb≥cx_a - x_b \ge cxa​−xb​≥c
  2. xa−xb≤cx_a - x_b \le cxa​−xb​≤c
  3. xa=xbx_a = x_bxa​=xb​
  • 数据范围：N,M≤5×103N, M \le 5 \times 10^3N,M≤5×103。

解题思路

1. 初步认知

  • 直观想法：这是一个标准的不等式组求解问题。由于只涉及两项的差 (xa−xbx_a - x_bxa​−xb​), 可以转化为最短路模型。
  • 瓶颈分析：如果存在逻辑矛盾（例如 A>B,B>C,C>AA > B, B > C, C > AA>B,B>C,C>A），在图论中表现为负环。我们需要检测这种矛盾。

2. 主题渗透 (正解)

算法：差分约束系统 + SPFA 判负环 (O(NM)O(NM)O(NM) 最坏)

  • 基本概念：将所有约束条件转化为最短路算法的三角不等式形式：xv≤xu+wx_v \le x_u + wxv​≤xu​+w。若约束图存在负环，则 xi<xix_i < x_ixi​<xi​，产生矛盾，无解。

  • 建模转换：
    我们将所有约束转化为 xv≤xu+wx_v \le x_u + wxv​≤xu​+w (对应边 u→wvu \xrightarrow{w} vuw​v):

    • Type 1 (xa−xb≥cx_a - x_b \ge cxa​−xb​≥c)：⇒xb≤xa−c\Rightarrow x_b \le x_a - c⇒xb​≤xa​−c. 建边 a→ba \to ba→b, 权值 −c-c−c。
    • Type 2 (xa−xb≤cx_a - x_b \le cxa​−xb​≤c)：⇒xa≤xb+c\Rightarrow x_a \le x_b + c⇒xa​≤xb​+c. 建边 b→ab \to ab→a, 权值 ccc。
    • Type 3 (xa=xbx_a = x_bxa​=xb​)：⇒xa≤xb+0\Rightarrow x_a \le x_b + 0⇒xa​≤xb​+0 且 xb≤xa+0x_b \le x_a + 0xb​≤xa​+0. 建边 a↔ba \leftrightarrow ba↔b, 权值 000。

  • SPFA 策略：

    • 全图连通性：图可能不连通。为了覆盖所有约束，初始时将所有节点 (1∼N1 \sim N1∼N) 入队，且 dist[i]=0dist[i]=0dist[i]=0。这等价于建立了一个连接所有点的超级源点。
    • 判环逻辑：记录每个点的松弛次数 cnt[v]。若 cnt[v] > n，说明存在负环，系统无解。

#include<bits/stdc++.h>using namespace std;using ll = long long;using PII = pair<int, int>;const int maxn = 5e3 + 5;vector<PII> adj[maxn];int dist[maxn];int cnt[maxn];bool vis[maxn]; // 标记是否在队列中 (in_queue)int n, m;void add( int u, int v, int w ){    adj[u].push_back({ v, w });}bool spfa( ){    queue<int> q;    // 关键：初始化所有点入队，保证处理所有连通块    for( int i = 1; i <= n; ++ i )    {        dist[i] = 0; // 初始距离设为 0 (无需 INF)        vis[i] = 1;        cnt[i] = 0;        q.push( i );    }    while( !q.empty( ) )    {        int u = q.front( );        q.pop( );        vis[u] = 0;                for( auto [v, w] : adj[u] )        {            if( dist[u] + w < dist[v] )            {                // 【核心推导】松弛：x_v <= x_u + w                dist[v] = dist[u] + w;                cnt[v] = cnt[u] + 1;                                // 判负环：最短路经过的边数超过 N                if( cnt[v] > n ) return 1; // 存在负环 -> 矛盾 -> No                if( !vis[v] )                {                    q.push( v );                    vis[v] = 1;                }            }        }    }    return 0; // 无负环 -> Yes}int main( ){    ios::sync_with_stdio(0);    cin.tie(0); cout.tie(0);        cin >> n >> m;    for( int i = 1; i <= m; ++ i )    {        int k, a, b, c;        cin >> k;        if( k == 1 )         {            // Type 1: x_a - x_b >= c  =>  x_b <= x_a - c            cin >> a >> b >> c;            add( a, b, -c );        }        else if( k == 2 )         {            // Type 2: x_a - x_b <= c  =>  x_a <= x_b + c            cin >> a >> b >> c;            add( b, a, c );        }        else         {            // Type 3: x_a = x_b  =>  x_a <= x_b + 0 & x_b <= x_a + 0            cin >> a >> b;            add( a, b, 0 );            add( b, a, 0 );        }    }    // SPFA 返回 true 表示有负环 (矛盾)    if( spfa( ) ) cout << "No" << '\n';    else cout << "Yes" << '\n';    return 0;}

四、 结语

差分约束系统是图论建模的基础思想之一。

其本质在于将代数不等式组转化为图的结构，

利用最短路算法的收敛性来判断系统是否存在矛盾。

掌握不等式到边权和方向的转换是解决此类问题的关键。
