P1993 小K的农场 (Difference_Constraints)

Luogu-P1993 小K的农场

1. 核心逻辑

  • 问题本质: 给定一系列变量间的差值约束，判定是否有解。

  • 破局转换:

    1. 代数转几何：将 xa−xb≤wx_a - x_b \le wxa​−xb​≤w 映射为图论中的边 b→wab \xrightarrow{w} abw​a。

    2. 标准转化：

      • xa−xb≥c  ⟹  xb−xa≤−c  ⟹  b←−cax_a - x_b \ge c \implies x_b - x_a \le -c \implies b \xleftarrow{-c} axa​−xb​≥c⟹xb​−xa​≤−c⟹b−c​a。

      • xa−xb≤c  ⟹  b→cax_a - x_b \le c \implies b \xrightarrow{c} axa​−xb​≤c⟹bc​a。

      • xa=xb  ⟹  b→0ax_a = x_b \implies b \xrightarrow{0} axa​=xb​⟹b0​a 且 a→0ba \xrightarrow{0} ba0​b。

    3. 冲突判定：如果图中存在负环，则意味着约束条件存在逻辑坍缩，即无解。

    4. 超级源点：建立 0 号点向所有点连边权为 0 的边，确保判负环过程能覆盖所有联通分量。
