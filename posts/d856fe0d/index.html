[L2] 20_图论

[L2] 20_图论

核心哲学：图论不是在画点和线，而是在建立一套“状态”与“转换”的规则。

1. 知识图谱与演化

  • 核心工具: 最短路建模、约束转化、流量平衡、结构压缩。

  • 演化路径:

    1. 遍历：能不能走通？（ DFS / BFS ）

    2. 最短路：寻找代价最小或收益最高的走法。（ Dijkstra / SPFA ）

    3. 流与匹配：资源分配与冲突解决。( MaxFlow/MinCost )

    4. 结构转化：将有环图转化为DAG或树。( SCC/圆方树 )。

2. 决策矩阵

  • 代价最小 / 距离最近 / 差分约束：最短路体系。

  • 资源分配 / 二选一模型 / 费用控制：[[网络流专题]]。

  • 必经点 / 环结构处理：[[连通性专题]] (缩点/圆方树)。

  • 最近公共祖先 / 树上路径统计：[[树论专题]] (LCA/点分治)。

  • 二分图下的最大配对：[[匹配专题]]。

3. 核心板块索引

3.1 [L3] 21 图论基础

  • 存图与遍历：如何存图，如何遍历。

  • 拓扑排序：DAG 上的最长路。

  • 欧拉回路与路径：一笔画问题。

  • 最短路体系： Dijkstra, SPFA, Floyd.

  • [[Note] 差分约束]：变量不等式组的图论建模。

  • [[Note] 同余最短路]：剩余系空间下的到达性。

  • [[Note] K 短路]：寻找代价前 K 小的路径集合。

3.2 [L3] 22 树论专题

  • LCA 体系：寻找最近公共祖先，倍增/树链剖分。

  • 树上差分：树上的前缀和差分。

  • 树链剖分：将非线性的树结构转化为线性的数组结构。

  • 生成树体系: Kruskal，Prim。

  • 直径与重心：树的骨架与平衡点。

  • [[Note] 点分治]：树上大规模路径统计。

  • [[Note] 虚树]：压缩树上关键点。

  • [[Note] 矩阵树定理]：一张图的生成树个数计数问题。

3.3 [L3] 23 连通性专题

  • SCC 缩点: 有向图强连通分量。

  • [[Note] BCC 双连通]: 割点、桥。

  • [[Note] 圆方树]: 仙人掌图与广义圆方树。

  • [[Note] 2-SAT]: 布尔可满足性问题。

3.4 [L3] 24 匹配专题

  • [[Note] 二分图匹配]: 匈牙利算法。

  • [[Note] KM 算法]: 带权二分图最佳匹配。

3.5 [L3] 25 网络流专题

  • [[Note] 最大流]: Dinic 算法。

  • [[Note] 费用流]: MCMF.

  • [[Note] 最小割建模]: 最大权闭合子图等模型。

  • [[Note] 上下界网络流]: 流量守恒的扩展。
