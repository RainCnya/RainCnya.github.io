ABC441E A > B substring

1. 题面梗概（翻译）

给定一个仅包含 A, B, C 的字符串 SSS，长度为 NNN。求有多少个子串满足其中 A 的出现次数严格大于 B 的出现次数。

2. 逻辑推导

问有多少个区间内 A 的数量大于 B。

我们只关心 A 和 B 的相对数量。令 A = 1, B = -1, C = 0。

对于一个子串 [l,r][l,r][l,r] 满足条件   ⟺  \iff⟺ ∑k=1r(cnt[k])>0\sum_{k=1}^{r}(cnt[k]) > 0∑k=1r​(cnt[k])>0

既然要用区间和，不妨用前缀和优化。

那么问题就转化为了，找到 i<ji < ji<j 使得 s[j]−s[i−1]>0  ⟺  s[i−1]<s[j]s[j] - s[i-1] > 0 \iff s[i-1] < s[j]s[j]−s[i−1]>0⟺s[i−1]<s[j]，咦，看起来这就是 "顺序对" 啊。

那么这里有两种方案了。"树状数组" / "归并排序"。

我才用树状数组来统计答案，应该是比较经典的写法。
