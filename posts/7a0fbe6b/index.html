2.2 分析

当 n≠2kn \neq 2^kn=2k 时，设 nnn 的最高位为 2k2^k2k，定义余数为 r=n−2kr= n - 2^kr=n−2k，发现 n⊕2k=rn \oplus 2^k = rn⊕2k=r。

而通过上面的分析，我们发现有三个位置很特殊 i=1,i=2k,i=ri = 1, i = 2^k, i = ri=1,i=2k,i=r。

  • 若要解决 i=1i = 1i=1，可以简要构造 k⊕(k+1)=1k \oplus (k + 1) = 1k⊕(k+1)=1。
  • 若要解决 i=2ki = 2^ki=2k，可以利用 n,rn, rn,r。
  • 若要解决 i=ri = ri=r，可以利用 n,2kn, 2^kn,2k。

  1. 而因为 r<2kr < 2^kr<2k，所以可以让 pr=np_r = npr​=n，根据题意，只需要把 2k2^k2k 放在 rrr 后面的某个位置即可。
