[Leaf] [CF2187D] Shortest Statement Ever

CF2187D - Shortest Statement Ever

1. 题面梗概

中译中: 给定两个数 x,yx, yx,y，找出两个数 p,qp, qp,q，满足 p&q=0p \& q = 0p&q=0，使得 ∣x−p∣+∣y−q∣| x - p | + | y - q |∣x−p∣+∣y−q∣ 的值最小。

2. 逻辑推导

本题存在 数位DP 的通解，以及 贪心构造 的特解。

2.1 切入点

p&q=0p \& q = 0p&q=0 这个约束条件提示我们，每一个二进制位 iii 都是不可共享的资源。对于第 iii 位，要不给 ppp，要不给 qqq，要不都没有。

既然我们的目标是最小化 ∣x−p∣+[y−q∣|x - p| + [y - q|∣x−p∣+[y−q∣，当 x,yx, yx,y 的第 iii 位都是 111 呢？这是最棘手的情况，必须有一个数作出让步。

既然是让步，那么就存在两种情况，向下截断，或是向上进位。

那么问题就变成了，在 向下截断 带来的变小，和 向上进位 带来的变大，二者中找到平衡点。

2.2 数位DP

既然涉及到进位这种问题，数位DP是很显然的写法。通常的数位DP 是从高位到低位，但本题要处理加减法，状态比较麻烦，所以我们采用从低位到高位的DP设计。

我们先处理目标函数的这个绝对值，min⁡(∣x−p∣+∣y−q∣)\min(|x - p| + |y - q|)min(∣x−p∣+∣y−q∣)，可以转化为 min⁡((x−p)+(y−q),(x−p)−(y−q),−(x−p)+(y−q),−(x−p)−(y−q))\min({(x - p) + (y - q)}, {(x - p) - (y - q)}, {-(x - p) + (y - q)}, {-(x - p) - (y - q)})min((x−p)+(y−q),(x−p)−(y−q),−(x−p)+(y−q),−(x−p)−(y−q))。

其实就是枚举四种情况的符号，然后跑4遍DP即可。

定义：dp[i][cx][cy] 表示处理到第 iii 位，xxx 的进位值 cxcxcx，yyy 的进位值 cycycy。

以计算 (p−x)( p - x )(p−x) 为例：

  1. 当前位：假设上一位传来的进位是 last_cx。当前位我们需要决策 pip_ipi​ 是 0/10/10/1。当前位实际数值差位：val=last_cx+pi−xival = last\_cx + p_i - x_ival=last_cx+pi​−xi​。

  2. 贡献：当前位对答案的贡献就是剩下的余数：cost = (val & 1) * (1 << i)。

  3. 进位：新产生的进位为 new_cx = val >> 1。

2.3 贪心与构造

直觉上其实有一个猜想：最优解大概率在边界上，即 p=xp = xp=x，或者 q=yq = yq=y。

若我们固定 p=xp = xp=x，那么只需要寻找 ∣q−y∣|q - y|∣q−y∣ 的最小值，且 q&x=0q \& x = 0q&x=0。我们只需要避开 xxx 中二进制为 111 的位置即可。

同样还是两种思路，向下取整，向上进位，计算完答案后取 min⁡\minmin 即可。
