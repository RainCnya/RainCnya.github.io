一、错排

Ex. 1 P4071 排列计数

1. 题面梗概

求有多少种从 111 到 nnn 的排列 aaa，满足序列中恰好有 mmm 个位置 iii，使得 ai=ia_{i} = iai​=i。

多测 T≤5×105,n,m≤106T \leq 5 \times 10^5, n,m \leq 10^6T≤5×105,n,m≤106

2. 逻辑推导

不妨先来考虑 m=0m = 0m=0 的情况，也就是所有数字全部错位的情况。

发现很难直接用组合数表示出本题的答案，因此考虑使用其他方式进行求解。不妨设有 nnn 个数的时候答案为 dnd_{n}dn​，则尝试寻找 ddd 数组的递推关系。

考虑第 ana_{n}an​ 的取值，显然，ana_{n}an​ 有 n−1n - 1n−1 种取值，且每种取值等价。不妨设 an=1a_{n} = 1an​=1，那么来考虑 a1a_{1}a1​ 的取值，则有以下两种情况：

  1. a1=na_{1} = na1​=n。在这种情况下，a1a_{1}a1​ 和 ana_{n}an​ 是互相对应的关系，可以直接去掉这两个数字，剩下的就是 2→n−12 \to n-12→n−1 的数字，答案就是 dn−2d_{n-2}dn−2​。

  2. a1≠na_{1} \neq na1​=n。在这种情况下，a1a_{1}a1​ 和 ana_{n}an​ 就不是互相对应的关系了，也就是剩下来 1→n−11 \to n-11→n−1 的数字错排序，答案就是 dn−1d_{n-1}dn−1​。

所以可以得出 nnn 个数完全错拍的递推式为：dn=(n−1)(dn−1+dn−2)d_{n} = (n-1)(d_{n-1} + d_{n-2})dn​=(n−1)(dn−1​+dn−2​)。

接着我们分析 m≠0m \neq 0m=0 的情况，也就是有 mmm 个数时完全对应的，那么 nnn 选 mmm 就是经典的 CnmC_{n}^{m}Cnm​，然后剩下的 n−mn - mn−m 个数完全错排，最终答案就是：

Cnm×dn−mC_{n}^{m} \times d_{n-m} Cnm​×dn−m​

由于询问的次数很多，所以需要预处理出 ddd 数组的值，而对于组合数，这里可以采用逆元加速。

二、卡特兰数

Ex. 1 P1044 栈

1. 题意梗概

有一个空栈，有 nnn 个元素依次按照顺序入栈，但在入栈的过程中可能随时会有元素出栈的情况。求最终合法的出栈顺序方案数。

n≤18n \leq 18n≤18

2. 逻辑推导

不妨设 CnC_{n}Cn​ 为元素个数为 nnn 时的答案，考虑采用递推的方式进行求解。

考虑编号为 111 的元素出栈的位置。容易发现，如果 111 号元素是第 kkk 个出栈的，那么 222 到 kkk 的元素在此时必然已经出栈，而后 n−kn-kn−k 个元素在此时还没有进栈。

由于前后两部分的出栈顺序同样满足先入先出的条件，因此编号为 222 到 kkk 的元素的出栈顺序种类数量为 Ck−1C_{k-1}Ck−1​ 种，而后 n−kn-kn−k 个元素的出栈顺序恰为 Cn−kC_{n-k}Cn−k​ 种。

所以执行需要将 kkk 取遍 1→n1 \to n1→n 即为最终的答案：

Cn=∑k=1n(Ck−1×Cn−k)C_{n} = \sum_{k=1}^{n}( C_{k-1} \times C_{n-k} ) Cn​=k=1∑n​(Ck−1​×Cn−k​)

也可以写成：

Cn=∑i=0n−1(Ci×Cn−i−1)C_{n} = \sum_{i=0}^{n-1} (C_{i} \times C_{n-i-1}) Cn​=i=0∑n−1​(Ci​×Cn−i−1​)

其中 C0=1C_{0} = 1C0​=1，那么求出 CnC_{n}Cn​ 即为答案。

三、斯特林数

Ex.1 P1287 盒子与球

1. 题面梗概

有多少种方案将 nnn 个互不相同的球放入 rrr 个互不相同的盒子，使得没有一个盒子为空。

0≤r≤n≤100 \leq r \leq n \leq 100≤r≤n≤10

2. 逻辑推导

换一个角度分析，如果所有的盒子都是相同的，本题应该如何处理。

考虑递推，设 S(n,r)S(n, r)S(n,r) 表示有 nnn 个球 rrr 个盒子，且盒子相同的答案。不妨将球按 111 到 nnn 编号，考虑编号为 nnn 的球的放置情况.

  1. 该球独占一个盒子。此时剩下的 n−1n-1n−1 个球应当共享 r−1r-1r−1 个盒子，故方案数为 S(n−1,r−1)S(n-1, r-1)S(n−1,r−1)。

  2. 该球与其他球共享一个盒子。此时不妨先摆好其他球（ S(n−1,r)S(n-1, r)S(n−1,r) 种情况 ）。接着，由于编号为 nnn 的球可以放置在 rrr 个盒子中，故通过乘法原理，方案数为 r×S(n−1,r)r \times S(n-1, r)r×S(n−1,r)。

由加法原理，得出 S(n,r)S(n, r)S(n,r) 的递推式为 S(n,r)=S(n−1,r−1)+r×S(n−1,r)S(n, r) = S(n-1, r-1) + r \times S(n-1,r)S(n,r)=S(n−1,r−1)+r×S(n−1,r)。

接下来考虑盒子不同时的情况，可以将其分成两步：

  1. 决定盒子的顺序，显然摆放盒子的方式有 r!r!r! 种。
  2. 将球放入盒子中，由于此时盒子的顺序已经确定，故此时的答案为：S(n,r)S(n, r)S(n,r)。

由乘法原理，得到最终答案为 r!×S(n,r)r! \times S(n, r)r!×S(n,r)。
