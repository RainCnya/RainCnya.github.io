TZOJ-2993 Brackets

一、 题目大意

给定一个只包含 (, ), [ 和 ] 四种字符的字符串。

要求找到一个最长子串 (Substring)，这个子字符串是一个合法的括号序列。

并且输出这个最长的合法子串。

二、 思路分析与关键点

1. 暴力（Naive Solution）

最直接的思路是：

  1. 枚举所有可能的子字符串 (i,j)(i, j)(i,j)。O(N2)O(N^2)O(N2)。
  2. 对于每一个子串，使用栈来检查其是否为合法的括号序列。O(N)O(N)O(N)。
  • 复杂度： 显然，总时间复杂度为 O(N3)O(N^3)O(N3)。
  • 结论： 对于本题 N=105N=10^5N=105 的数据，O(N3)O(N^3)O(N3) 必定超时 (TLE)。

2. 核心思路：动态规划 (DP)

O(N3)O(N^3)O(N3) 的瓶颈在于重复检查。这是一个具有最优子结构的问题，适合使用 DP 解决。

状态定义：

定义 dp[i]dp[i]dp[i] 表示：以字符串第 iii 个字符 结尾 的 最长 合法括号序列的 长度。

状态转移：

我们遍历字符串 iii 从 1→N1 \to N1→N：那就会存在两种情况。

  • Case 1: s[i] 是左括号 ( 或 [

    • 合法的括号序列不可能以左括号结尾。
    • 所以 dp[i]=0dp[i] = 0dp[i]=0。

  • Case 2: s[i] 是右括号 ) 或 ]

    • 这是需要进行状态更新的节点。
    • 那么就存在两种情况，嵌套以及连接。
    • [[]] 嵌套；[][] 连接。
    • 接下来我们严格判断这两种情况即可。

模拟：

s = "()[()]"

( ) [ ( ) ]
– - - i - dp[i-1] - 1 - - i
dp 0 2 0 0 2 …

我们对 i=6i = 6i=6 进行分析，s[6] = ']'， s[i - dp[i-1] - 1] = '['。

发现可以匹配，那么 dp[i] = dp[i-1] + 2。

同时还要判断在这之前还有没有连接情况存在。

也就是 i=2i = 2i=2 这一格，发现能连上，所以最终的答案应该是：

dp[i] = dp[i-1] + 2 + dp[i - dp[i-1] - 2]

  • 复杂度对比：
    • 暴力 $O(N^3) $ TLE
    • DP $O(N) $ AC

3. 关键点与总结

  • 时间复杂度： O(N)O(N)O(N)。我们只需要一次遍历。
  • 空间复杂度： O(N)O(N)O(N)，用于存储 dp 数组。
  • 核心逻辑： 通过 dp[i]dp[i]dp[i] 的定义，方便地判断“嵌套”和“连接”两种情况。
