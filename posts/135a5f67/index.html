07 博弈论 核心指南

核心哲学：博弈论不是在算计人心，而是在推演规则。
它的本质是：如果一个局面能到达的下一步里有一个是对方必输的，那你现在就赢定了。
我们要做的，就是把复杂的规则拆成一个个独立的、可以算清楚的“分值”。

1. 知识图谱与演化

  • 核心抓手: 状态转换、胜负对等、分值化、资源拆解。

  • 演化路径:

    1. 规律找补：最原始的博弈。通过手算几个小规模情况，观察出“奇数必胜”或“偶数必胜”的规律。

    2. 公平博弈 (Nim)：发现游戏可以被拆解。如果一个大游戏由几个小游戏组成，且规则完全一致，我们就可以用异或运算把它们的“分值”合并。

    3. 分值化 (SG 函数)：万物皆可 Nim。通过给每一个局面算一个“分值”（SG 值），把任何奇奇怪怪的博弈规则都转化成简单的数字博弈。

    4. 非对称博弈：进入更高级的领域。当双方的权力不相等（你能走的步我不能走）时，利用超现实数来衡量谁更有“后手优势”。

2. 决策矩阵

  • 题目规则简单且局面唯一（如：每次取 1∼k1 \sim k1∼k 个石子）: 优先尝试找规律（找补）。看当前状态对 k+1k+1k+1 取模的结果。

  • 题目涉及多个独立的堆或独立的游戏分支:

    • 如果规则对双方完全平等，采用 SG 函数。

    • 先分别计算每个分支的 SG 值，最后全部异或起来。如果结果不为 0，先手必胜。

  • 题目涉及“树上取石子”或“删边游戏”: 考虑树上博弈模型，利用递归方式合并子树的 SG 值。

  • 题目涉及“双方可选步不一样”（如：白方只能动白棋）: 考虑不公平博弈。

3. 核心板块索引

3.1 公平博弈

  • [L3] 01 公平博弈: 博弈论的入场券。解决规则对称下的胜负判定。

  • [Note] Nim 游戏: 博弈论的基石。核心是“异或和为 0”代表平衡（必败），不为 0 代表不平衡（必胜）。

  • [Note] SG 函数: 核心是 MEX 运算（求最小未出现的非负整数）。它能把复杂的跳转规则坍缩成一个数字。

3.2 进阶博弈

  • [L3] 02 进阶博弈: 解决更诡异、更贴近真实的策略对抗。

  • [Note] 不公平博弈: 当玩家身份决定了可选操作时，利用权值叠加来判断终局。

  • [Note] 超现实数基础: 衡量博弈局面的“重量”，每一步操作都是在消耗或增加自己的优势权重。

4. 依赖与规划

  • 前置依赖: 必须掌握基础的位运算（异或）、简单的递归思维、以及基本的数学归纳法。

  • 学习路径:

    1. P1 (基础)：熟练掌握 Nim 游戏和基础的 SG 函数推导，能独立手算前几个状态找规律。

    2. P2 (中坚)：学会把复杂的题型（如树上删边、图上移动）规约为 SG 模型。这是 Phase 2.5 冲金的核心。

    3. P3 (顶尖)：攻克不公平博弈和具有无限状态的动态博弈。
