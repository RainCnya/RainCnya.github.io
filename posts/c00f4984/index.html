[Leaf] P1544 三倍经验

P1544 三倍经验 - 洛谷

1. 题面梗概

中译中：在一个数字金字塔中从顶走到低，求路径最大和。特殊规则：你有 kkk 次机会让路径上的某个数字变成其 3 倍。

1≤n≤100,0≤k≤n(n+1)21 \le n \le 100, 0 \le k \le \frac{n(n+1)}{2}1≤n≤100,0≤k≤2n(n+1)​，每个数可能为负数。

2. 逻辑推导

如果没有 kkk 的限制，这就是普通的数字金字塔 DP。

现在的难点在于：每一层除了向 “左下” 和 “右下” 走以外，还有一个 “翻倍” 的决策。

那么我们只需要把 翻倍看作一个技能，把技能的使用次数加入阶段就行。

2.1 状态

定义：f[i][j][k]f[i][j][k]f[i][j][k] 为从 (i,j)(i, j)(i,j) 走到 最底层，恰好使用了 kkk 次技能的最大得分。

原始转移：

f[i][j]=max⁡(f[i−1][j],f[i−1][j−1])+a[i][j]f[i][j] = \max( f[i-1][j], f[i-1][j-1] ) + a[i][j] f[i][j]=max(f[i−1][j],f[i−1][j−1])+a[i][j]

那么加上技能后的转移：

  1. 不使用技能

f[i][j][k]=max(f[i−1][j][k],f[i−1][j−1][k])+a[i][j]f[i][j][k] = max( f[i-1][j][k], f[i-1][j-1][k] ) + a[i][j] f[i][j][k]=max(f[i−1][j][k],f[i−1][j−1][k])+a[i][j]

  1. 使用技能 ( 限制 k≥1k \ge 1k≥1 )

f[i][j][k]=max(f[i−1][j][k−1],f[i−1][j−1][k−1])+3a[i][j]f[i][j][k] = max( f[i-1][j][k-1], f[i-1][j-1][k-1] ) + 3a[i][j] f[i][j][k]=max(f[i−1][j][k−1],f[i−1][j−1][k−1])+3a[i][j]
