P1144 最短路计数

Luogu-P1144 / TZOJ-6538最短路计数

题目简述

难度：提高(3838) | 题解格式：V6.3

给定一个 NNN 个顶点 MMM 条边的无向无权图。求从顶点 111 到其他每个点 iii 的最短路径有多少条。输出答案对 100003100003100003 取模。

  • 数据范围：N≤106,M≤2×106N \le 10^6, M \le 2 \times 10^6N≤106,M≤2×106。

算法笔记可以参考：

  • 算法学习笔记：[图论] - 最短路图 (Shortest Path DAG) - RainCnya

解题思路

1. 初步认知

  • 直观想法：由于是无权图（边权为 1），最短路本质上就是 BFS 的层数。
  • 模型本质：所有最短路径实际上构成了原图的一个子图（最短路 DAG）。我们要做的就是在构建这个 DAG 的过程中同步进行动态规划计数。

2. 主题渗透 (正解)

算法：最短路上的 DP (BFS/Dijkstra) (O(N+M)O(N+M)O(N+M) 或 O(Mlog⁡N)O(M \log N)O(MlogN))

  • 基本概念：利用最短路算法的松弛过程维护计数。
    虽然本题是无权图，BFS 是最优解 (O(N+M)O(N+M)O(N+M))，但 Dijkstra 同样适用且逻辑一致（仅多带一个 log⁡\loglog）。
  • 状态定义：
    dist[v]dist[v]dist[v]：起点到 vvv 的最短距离。
    f[v]f[v]f[v]：起点到 vvv 的最短路径条数。
  • 转移逻辑：
    当通过 uuu 扩展邻居 vvv 时：
    1. 更新 (Update)：若 dist[u]+1<dist[v]dist[u] + 1 < dist[v]dist[u]+1<dist[v]，说明找到了更短的路。
      • dist[v]=dist[u]+1dist[v] = dist[u] + 1dist[v]=dist[u]+1
      • f[v]=f[u]f[v] = f[u]f[v]=f[u] (完全继承 uuu 的方案数，且之前的方案作废)
    2. 累加 (Accumulate)：若 dist[u]+1==dist[v]dist[u] + 1 == dist[v]dist[u]+1==dist[v]，说明找到了另一条长度相同的最短路。
      • f[v]=(f[v]+f[u])%modf[v] = (f[v] + f[u]) \% modf[v]=(f[v]+f[u])%mod

#include<bits/stdc++.h>using namespace std;using PII = pair<int, int>;const int maxn = 1e6 + 50;const int mod = 100003;vector<int> adj[maxn];int dist[maxn];int f[maxn]; // 记录最短路条数int n, m;void dijkstra( ){    // 初始化    memset( dist, 127, sizeof( dist ) );    dist[1] = 0;    f[1] = 1; // 起点到自己有1条路径    // 虽然无权图用 queue (BFS) 更快，但 priority_queue (Dijkstra) 通用性更强    priority_queue< PII, vector<PII>, greater<PII> > pq;    pq.push({ 0, 1 });    while( !pq.empty( ) )    {        auto [d, u] = pq.top( );        pq.pop( );                if( d > dist[u] ) continue;        for( int v : adj[u] )        {            // Case 1: 发现累加机会 (相等)            if( dist[u] + 1 == dist[v] )            {                f[v] = ( f[v] + f[u] ) % mod;            }            // Case 2: 发现更新机会 (更短)            else if( dist[u] + 1 < dist[v] )            {                dist[v] = dist[u] + 1;                f[v] = f[u]; // 覆盖计数                pq.push({ dist[v], v });            }        }    }}int main( ){    ios::sync_with_stdio(0);    cin.tie(0); cout.tie(0);    cin >> n >> m;    for( int i = 1; i <= m; ++ i )    {        int u, v;        cin >> u >> v;        adj[u].push_back( v );        adj[v].push_back( u );    }    dijkstra( );        for( int i = 1; i <= n; ++ i )    {        cout << f[i] << '\n';    }    return 0;}

四、 结语

本题是 最短路计数 的模板题。

核心在于理解最短路的最优子结构性质：

任何一条最短路都是由前驱节点的最短路延伸而来的。

这种“一边搜索一边 DP”的思想在图论中非常常见。
